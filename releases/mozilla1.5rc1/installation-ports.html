Mozilla ポートのインストール方法

このドキュメントには、Mozilla 1.5 Release Candidate 1 に関する、システム必要条件、互換性情報、インストール方法が書かれています。

  • OpenVMS
  • Solaris
  • AIX
  • HP-UX
  • IRIX
  • OS/2

OpenVMS

  • OpenVMS Alpha V7.1 以降
  • DECwindows MOTIF V1.2-4 以降
  • Compaq TCP/IP Services Version 5.0 for OpenVMS 以降、MultiNet V4.3A 以降、または TCPware V5.4-3 以降
  • 65 MB のハードディスク空きスペース

より詳しい情報は、OpenVMS Mozilla - installation guide and release notes をご覧ください。

Solaris

  • Solaris では、Mozilla を実行するために、sunsolve.sun.com から以下のパッチをインストールする必要があります。(Bug 88205)
Solaris Patches/SPARC
2.7
  • 106950-22 - "SunOS 5.7: linker patch"
  • 106327-17 - "32-Bit Shared library patch for C++"
  • 106300-18 - "64-Bit Shared library patch for C++"
  • 108376-42 - "OpenWindows 3.6.1: Xsun Patch"
  • 107650-08 - "OpenWindows 3.6.1 X11R6.4 Xprint Extension Patch"
  • 107636-09 - "SunOS 5.7: X Input & Output Method patch"
または、これらのパッチがすべて含まれた、推奨＆セキュリティパッチ集を ダウンロード / インストール してください。

  • 108652-65 - "X11 6.5.1: Xsun patch"

  • 111711-03 - "32-Bit Shared library patch for C++"

または、これらのパッチがすべて含まれた、推奨＆セキュリティパッチ集を ダウンロード / インストール してください。
  • 65 MB のハードディスク空きスペース
  • 128 MB の RAM
  • 印刷をするには Xprint がインストールされている必要があります。xprint.mozdev.org が、インストールのための便利な方法として GISWxprintglue (または GISWxprintglue) パッケージを提供しています。
  • GTK+ と glib ライブラリ - このライブラリは Sun Workshop/Forte でコンパイルされている必要があります。gcc でコンパイルされた GTK+/glib は、Sun Workshop/Forte でビルドされた Mozilla バイナリと 互換性がありません。Sun Workshop/Forte でコンパイルされた GTK+/glib ライブラリは、Sun Workshop と gcc どちらでコンパイルされたバイナリとも互換性があります。(Bug 136144、141415、186534 他多数)

AIX

  • AIX 4.3.3 以降
  • gtk+、glib、libiconv、gettext が必要です。(互換バージョンは IBM の Web サイト にあります)

HP-UX

  • HP-UX ビルドには それぞれの適切なサイトで提供されている gtk+、glib、libiconv、gettext が必要です。
    • HP-UX 11.20 は HP の Web サイトで提供されている 11i V 1.6 用 GTK (b6848ba) depot をベースにしています。
    • HP-UX 11.00 (と 11.11) は HP の Web サイトで提供されている 11.0/11i v1.0 用 GTK (b6848ba) または HP-UX Porting Center の depot をベースにしています。
    • HP-UX 10.20 は HP-UX Porting Center で提供されている depot をベースにしています。

IRIX

DecorateTransients オプションなしに twm ファミリーのウィンドウマネージャを使っていると、検索、保存、設定などのポップアップウィンドウが機能しません。ウィンドウマネージャの rc ファイルで DecorateTransients オプションをセットしてください。(Bug 78928)

OS/2

今回のリリースでは、Mozilla の実行に C ランタイム DLL のアップデートが必要です。これらの DLL を mozilla.exe と同じディレクトリか PATH のどこかに置いてください。

最低動作環境:

  • Pentium 133 MHz クラスの CPU
  • 64 MB の RAM
  • 70 MB のハードディスク空きスペース
  • 64 MB のスワップ空きスペース
  • OS/2 Warp 4 (Merlin Kernel)
  • Fixpack 15 以降
  • MPTS version 5.3 以降
  • TCP/IP version 4.1 以降
  • INETVER: SOCKETS.SYS=5.3007, AFOS2.SYS=5.3001, AFINET.SYS=5.3006 以降

注意: これらの INETVER レベルの上に MPTS & TCP/IP バージョンを使わないでください。これより古いスタックでも Mozilla は起動、実行すると思われますが、Mozilla が必要とする一部の機能は、古いバージョンの MPTS には正しく実装されていません。このため、クラッシュやデータ消失の可能性があります。

推奨動作環境:

  • Pentium II 266 MHz クラスの CPU
  • 128 MB の RAM
  • 150 MB のハードディスク空きスペース
  • 128 MB のスワップ空きスペース
  • Fixpack 16
  • MPTS version 5.5 または 6.0
  • TCP/IP version 4.3
  • INETVER: SOCKETS.SYS=6.2004, AFOS2.SYS=6.2000, AFINET.SYS=6.2013
    または
  • OS/2 Warp 4.52 Convenience Pack 2 (メンバー制)
    または
  • eCom Station 1.0 以降

注意: MPTS 6 と FP 16 は Software Choice メンバーのみ入手できます。その他のアップグレードは現在無料で入手可能です。

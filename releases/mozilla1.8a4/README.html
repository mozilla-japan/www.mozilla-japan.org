Mozilla 1.8 Alpha4 新着情報

この資料では、Mozilla 1.8 Alpha4 に実装された新機能、重要なバグ修正、そして新たな問題をいくつか紹介します。1.8 Alpha4 に施された修正のより包括的なリストは、 大まかな変更ログ【本家にリンク】 をご覧ください。

新機能と修正

ブラウザ
  • Mozilla 1.8a4 では、ポップアップ遮断機能が大幅に改良されています。（bug 252326）
  • Mozilla で XUL チェックボックスやラジオボタンのラベルに、下線のついたアクセス記録がつくようになりました。この機能により、キーボードを使ったダイアログボックスの利用がより簡単になりました。（bug 68841）
  • Mozilla の履歴ウィンドウにリファラを表示できるようになりました。（bug 128398）
  • ブラウザウィンドウに収まるように大きな画像のサイズ調整をする機能が改善され、クリック位置を中心にして画像サイズを元の大きさに戻すことができるようになりました。（bug 207219）
  • Mozilla 1.8a4 では、HTTP Basic 認証を利用して、すべてのサイトからログアウトできるようになりました。認証セッションからログアウトするには、ツール（Tools） -> パスワードマネージャ（Password Manager） -> ログアウト（Log Out）としてください。（bug 231529）
  • Mozilla は、プロンプトの表示されない NTLM 認証をサポートするようになりました。（bug 231529）
  • 1.8a4 は、入力やテキストエリアで gtk2 のネイティブ・キーボード・バインディングを利用するようになりました。（bug 257405）
  • Mozilla は CUPS を経由してプリンタのリストを照会できるようになりました。（bug 251754）
  • Linux 用 Mozilla では、標準のインストールでスペルチェッカが含まれるようになりました。（bug 253443）
メール
  • 依然として「alpha」状態で、新しい Virtual Folder（仮想フォルダ）では検索結果をフォルダとして保存することができるようになりました。検索は、単一フォルダ内、アカウント全体、そしてフォルダとのそのサブフォルダに対して行うことができます。（bug 11051）
Chatzilla
  • Chatzilla 0.9.65 が Mozilla のためにチェックインされました。（bug 255081）
舞台裏
  • Gecko の DHTML パフォーマンスが、最近の 2 つのアルファ・マイルストーンで大幅に改良されました。
  • Mozilla が、overflow-y と overflow-x プロパティをサポートするようになりました。（bug 72747）
  • CSS3 カーソルのサポートが Gecko に追加されました。（bug 163174）
  • Mozilla 1.8a4 は、JavaScript コンソールを経由して CSS エラー報告を提供するようになりました。（bug 65469）
  • Mozilla は、より確実な IE との互換性を目指して、要素の ID の表示をサポートするようになりました。これは quirks モードのみで利用可能です。（bug 256932）

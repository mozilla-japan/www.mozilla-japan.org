ElectricalFire
Java プラットホームのコンパイラ
ElectricalFire は、Java コード実行を加速させるために JIT (Just In Time) コンパイル技術を用いる Java 仮想機械です。ElectricalFire は最初から、高性能な実行コードを生成しかつアーキテクチャが異なる多くのプロセッサに移植できるようにデザインされました。

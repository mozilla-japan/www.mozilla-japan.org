Mozilla Thunderbird オフライン Extension

これは Mozilla メール クライアントにあったオフライン機能を Mozilla Thunderbird に追加する Extension です。機能としては次を含みます：

  • オフラインとオンラインを切り替える機能
  • フォルダのオフライン時動作を設定する機能（例えばオフライン時のためにメッセージを自動的にあるフォルダにダウンロードする）
  • オンライン状態に戻った時に未送信のメッセージをどのように扱うかの設定

この Extension を手に入れる

Thunderbird 用のこの Offline Extension をダウンロードしてローカルに保存してください。Tools / Options / Extensions を選択し、'Install Extension' をクリックして Thunderbird にオフラインサポートをインストールしてください。インストール後 Thunderbird を再起動してください。

オフライン UI インジケータ

この Extension をインストールして Thunderbird を再起動すると、メール３ペインウインドウの下の部分にオフラインインジケータが表示されているはずです：

このインジケータをクリックしてこのアプリケーション上でのオフライン状態を切り替えることが出来ます

オフライン設定を調整する

Tools / Options / Extensions を選択し Offline Support extension をクリックします。Settings ボタンをクリックします。そうすると下のような Thunderbird のオフライン時の動作を設定するダイアログが表示されるはずです。また、File / Offline / Offline Settings からでもこのダイアログを表示させることが出来ます。

また、個々のフォルダ独自のオフライン動作を設定することも出来ます。IMAP フォルダを右クリックして Properties を選び、そして Offline 設定のタブを選択してください。

オフライン Extension をアンインストールする

Thunderbird は今のところ Extension のアンインストールをサポートしていませんが間もなく実装します。それまではあなたの profile（その Extension を profile ディレクトリにインストールしたのであれば）直下にある chrome ディレクトリを直接取り除いてください。これによってあなたの profile ディレクトリにインストールされたすべての Extension が取り除かれます。

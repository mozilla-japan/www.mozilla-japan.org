オープンソースと Mozilla

– コードの向上のために

Mozilla は オープンソース であり フリーソフト でもあります。 個人、企業を問わず、自由に次のようなことができます。

  • 目的を問わず、プログラムを動かすこと
  • プログラムの働きを調べて、それを自分たちのニーズに適合させること
  • 自由にコピーを再配布すること
  • プログラムを改良し、変更を加えたバージョンを配布すること

Mozilla のすべてのソースコードは Mozilla and Netscape Public Licenses のもとで利用可能であり、それらは Free Software Foundation によって Free Software Licenses として認められたものです。

MPL の精神は、私有製品を含むあなたの アプリケーションや製品 に、Mozilla のコードを自由に利用できるということであり、現在の Mozilla のコードベースにあらゆる修正を加えることができるように提供されています。

フリーソフトを使うことによって、あなたのビジネスが一企業の製品に 縛られる ことはありません。 自分自身の将来を自由にコントロールできるのです。

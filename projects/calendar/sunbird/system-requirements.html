Sunbird 必要システム構成

Windows

オペレーティングシステム (OS)
  • Windows XP (推奨)
  • Windows Server 2003
  • Windows Vista
  • Windows 981
  • Windows 98 SE1
  • Windows ME1
  • Windows NT 4.01
  • Windows 20001

1 Windows 98、 Windows 98 SE、 Windows ME、 Windows NT 4.0 及び Windows 2000 のユーザーは Sunbird をインストールする前に、それらのオペレーティングシステムのための既知の問題を必ずお読み下さい。

最小ハードウェア構成
  • Pentium 233 MHz (推奨: Pentium III 500MHz 以上)
  • 64 MB の RAM (推奨: 128 MB 以上の RAM)
  • 52 MB のハードディスク空き容量

Linux

ソフトウェア要件

    注: Linux ディストリビューションによっては、ディストリビュータからシステム要件の異なる( Sunbird の)パッケージが提供されていることがあります。

  • Linux kernel - 2.2.14 以降 - 以下のライブラリまたはパッケージが必要です。:
    • glibc 2.3.2 以降
    • XFree86-3.3.6 以降
    • gtk+2.0 以降
    • fontconfig ( 別名 xft )
    • libstdc++5
最小ハードウェア構成
  • Intel Pentium II または AMD K6-III+ 233 MHz CPU (推奨: 500MHz 以上)
  • 64 MB の RAM (推奨: 128 MB 以上の RAM )
  • 52 MB のハードディスク空き容量

Mac

オペレーティングシステム(OS)
  • Mac OS X 10.2.x 以降
最小ハードウェア構成
  • Intel x86 または PowerPC G3/G4/G5 プロセッサを搭載した Macintosh コンピュータ
  • 128 MB の RAM (推奨: 256 MB 以上の RAM )
  • 200 MB のハードディスク空き容量

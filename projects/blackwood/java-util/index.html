Javaコードのためのユーティリィティ・クラス(Utility classes for Java Code)

このプロジェクトはJavaプログラマのためのユーティリィティ・クラスのコレクションです。これらのクラスは、java.utilおよびJDKコレクションクラスで見つかったクラスを補足します。このドキュメントは、utilクラスのハイライトを示し、コードをチェックすることについての情報をさらに提供します。

完全なAPIドキュメンテーションは http://www.mozilla.org/projects/blackwood/java-util/api/ で見つけることができます。

ハイライト(Highlights)

    Assert

      Javaで非常に欠ける1つのものは assertion facility です。 このクラスはたくましい、完全な、assertion facilityを 供給します。 Assert クラスを使う最も良い方法はこうです：

      if (Assert.enabled && Assert.assert_it(mytest(), "my test failed"));

      このような使用法は assertions が無効になる場合に、mytest()だけでなくassert メソッドも実行されません。また、assertions が無効になる場合、いくつかのコンパイラーは全ステートメントを削除することができます。

    パラメータチェック(ParameterCheck)

      ParameterCheckはパラメーターチェックに便利な手段を供給します。 ParameterCheckクラス中のクラスメソッドはメソッドパラメーターがある条件を満たすか確認して、条件が満たされない場合は例外を上げるために使用することができます。

      Typical usage:

      ParameterCheck.nonNull(myParameter);

      これは、myParameterがそれがそうである場合IllegalArgumentExceptionを投げて、無効ではないことを確認します。

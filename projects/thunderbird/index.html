Mozilla Thunderbird メールプロジェクト

Mozilla Thunderbird は、Mozilla のメールコンポーネントを再設計したものです。私たちの目標は、XUL ユーザインターフェース言語を使った、クロスプラットフォームで独立して動作するメールアプリケーションを作成することです。私たちが想定しているユーザは、Mozilla Firefox (別の独立して動作するブラウザ) をメインブラウザとして使っており、Mozilla を基礎として、ブラウザと「うまく連動する」メールクライアントを求めている人たちです。さらに、独立して動作するメールクライアントに特化することで、必要のないコンポーネントや chrome を取り除くことで、メールクライアントの全般的なフットプリントおよびパフォーマンスの突破口を開くことができるとも信じています。なおその上に、Mozilla Suite の一部であるよりも、独立して動作するアプリケーションとしての方が、より洗練されたユーザインターフェース (UI) になります。

Mozilla Mail にある機能に加えて、Mozilla Thunderbird にはいくつかの新機能や改良が施され、メール・ニュース体験をより良いものにします。注目点には次のようなものが含まれます:

  • 好きなように ツールバーをカスタマイズする 機能。どのウインドウからでも [表示] > [ツールバー] > [カスタマイズ] を選択できます。
  • UI 機能拡張 を Mozilla Thunderbird に追加することができます。これによって必要とする特定の機能を追加したり強化したりして操作感をカスタマイズすることができます。拡張機能 のサポートです。現在利用できる拡張機能の完全なリストは こちら。
  • 新しいルックアンドフィール。Thunderbird はまた、クライアントの外観を変更できるたくさんの ダウンロード可能なテーマ もサポートしています。
  • メール作成用の コンタクト サイドバー によって、アドレスブックの連絡先を簡単便利にメールに挿入することができます。
  • FAQ、ヒントとコツ、そしてその他有益な情報を含む オンラインヘルプ。
  • 簡素化された環境設定 UI とメニュー。
  • フットプリントとパフォーマンスの改善。

SeaMonkey 1.1 Beta

2006 年 11 月 8 日リリース

このリリースは最新の機能を示していますが完成していない機能も示している可 能性がある来る安定バージョンのプレビューです。 開発者とテスターのみ - エンドユーザにではなく - に提供されます。
エンドユーザのために提供されている最新バージョンを得るには、リ リースページへ行ってください。
  • リリースノート - リリースノートはインストール方法、システム要件、新機能、そしてバグを報告する前に読むべき既知の問題のリストを含んでいます。
  • ソースコード Tarball（bzip、 34 MB）（gzip、 43 MB）
  • この README ファイルは CVS からの SeaMonkey 1.1 Beta の取り出し方を説明しています。

公式ビルド

あなたにとってどの パッケージが最も良いのか分からないのであれば、あなたのオペレーティングシステム（プラットフォーム）についてのセクションで最初にリストされている パッケージを使ってください。
MD5 sum は公式パッケージとソースコード tarball に利用できます。

Windows（32bit）
フ ルインストーラ （13 MB） SeaMonkey コンポーネント全てをダウンロードしインストールオプションを指定します。
ネッ トインストーラ （264 KB） 指定したコンポーネントのみをダウンロードします。
Zip ファイル （11 MB）
Mac OS X
ディ スクイメージ（Universal Binary） （23 MB） 完全な SeaMonkey アプリケーションスイーツを含む自己マウント式ディスクイメージ（Universal Binary は PowerPC と Macintosh コンピュータで動作する Intel プロセッサーの両方で使えます）。
Linux（x86）
フ ルインストーラ （14 MB） SeaMonkey コンポーネント全てをダウンロードしインストールオプションを指定します。
ネッ トインストーラ （89 KB） 指定したコンポーネントのみをダウンロードします。
tar.gz フォーマット （15 MB）
GTK1 フルインストーラ （14 MB） SeaMonkey コンポーネント全てをダウンロードしインストールオプションを指定します。
GTK1 ネットインストーラ （125 KB） 指定したコンポーネントのみをダウンロードします。
GTK1 tar.gz フォーマット （14 MB）

Mozilla 1.8 Alpha6 新着情報

この資料では、Mozilla 1.8 Alpha6 に実装された新機能、重要なバグ修正、そして新たな問題をいくつか紹介します。1.8 Alpha6 に施された修正のより包括的なリストは、大まかな変更ログ (英語) をご覧ください。

新機能と修正

ブラウザ

  • Linux/Windows: キャレットブラウズモード (F7) が有効になっている際に Home キーや End キーを押すと、ページの先頭・末尾ではなく行頭・行末に移動するようになりました。
  • Windows: 「壁紙に設定」機能が管理者権限を必要としなくなりました。
  • Linux: 印刷ダイアログに [ファイルへ印刷] というチェックボックスが付き、ファイルに指示を与えることが可能になりました。
  • ページ情報ウィンドウが改良されました。
  • コンテキストメニューキーを押したときに、コンテキストメニューがより使いやすい位置に表示されるようになりました。
  • Windows: クリップボードへのコピーが Internet Explorer の動作により近くなり、ページのアドレスも同時にコピーされるようになりました。
  • Windows 9x: Alt+Tab のキー操作をしたときに、メニューバーにフォーカスが移らなくなりました。
  • 既定のダウンロード先ディレクトリを指定できるようになりました。これまでは、必ず以前に使用したディレクトリを選択肢として提供していました。

メール

  • 新たに「メッセージを並べ替えたときにスレッドを維持する」オプションが付きました。これを有効にしておくと、カラムヘッダをクリックしてメッセージを並べ替えても、スレッドになっているメッセージのグループが保たれます。これにより、任意の条件によるスレッドの並べ替えがより簡単になりました。
  • [表示] > [表示/隠す] メニューからフォルダペインを非表示にすることが可能になりました。
  • フラグ付きメッセージの検索やビュー作成が可能になりました。
  • アドレス帳サイドバーからメールアドレスをコピーできるようになりました。
  • サイズによるメッセージのフィルタや検索が可能になりました。
  • Outlook 2003 形式の 3 カラムレイアウトをサポートしました。
  • レイアウトを変更するために再起動を行う必要がなくなりました。

IRC クライアント (ChatZilla)

  • 0.9.67 へバージョンアップしました。
  • 新機能: チャンネル参加ダイアログ、新しいプラグイン API のサポート、既定のブラウザにリンクを開く機能、パラメータ付きエイリアス、wii コマンドの追加
  • バグの修正: 設定ダイアログを開いた際のクラッシュ (Mac 版のみ)、URL リンクの途切れ、再接続時の離席状態の保存、whois コマンドのバグ、顔文字アイコンのバグ

バックエンド

  • XML パーサが Expat 1.2 から Expat 1.95.7 へバージョンアップしました。これにより、XML 仕様と、XML 仕様の名前空間への準拠度が向上しました。1.8 Alpha6 以降、これまでのバージョンでは容認されてきた一部の valid でない XML ドキュメントは受け付けなくなります。今回の変更により、valid でない XML ドキュメントを使った一部の拡張機能で問題が起きていますが、解決策は、それらのドキュメントを修正することだけです。以下は、バージョンアップ以降、Mozilla が表示する適格性エラーの一例です。
    • valid でない実体名 (コロンを含む実体名など)
    • 宣言されていない接頭辞
    • 同一のローカル名や、同一の名前空間名に関連付けされた異なる接頭辞に付けられた複数の属性
    • 予約済みの接頭辞や名前空間名の再宣言
  • RTL モード (アラビア語、ヘブライ語) の場合に、border collapse モードのボーダー (フレーム、ルール、グループがテーブル上で指定されている場合にも有効になります。また、Microsoft Excel の HTML エクスポートにも使われています) がより正しく表示されるようになりました。これがどういう意味か、Firefox 1.0 で、バグに添付されたテストケースを比較してみてください。この問題は、Bug 174470 で修正され、Bug 267420 で改良されました。
  • 多くの安定性やセキュリティに関する修正が行われました。
  • テキストエリアの解析が格段に向上しました。Bug 88952 と各種依存バグをご覧ください。
  • ソース表示で見られたいくつかの構文解析エラーが修正されました。
  • パーミッションマネージャの新しいバックエンドが反映されました。
  • 幅が自動決定される絶対配置要素のレイアウトが修正されました (Bug 201897)。

Mozilla Foundation セキュリティアドバイザリ 2013-93

タイトル: 様々なメモリ安全性の問題 (rv:25.0 / rv:24.1 / rv:17.0.10)
重要度: 最高
公開日: 2013/10/29
報告者: Mozilla 開発者
影響を受ける製品: Firefox、Thunderbird、SeaMonkey

修正済みのバージョン: Firefox 25.0
  Firefox ESR 24.1
  Firefox ESR 17.0.10
  Thunderbird 24.1
  Thunderbird ESR 17.0.10
  SeaMonkey 2.22

概要

Firefox やその他の Mozilla ベースの製品に搭載されているブラウザエンジンに含まれていたメモリ安全性に関するいくつかのバグを、Mozilla の開発者が特定、修正しました。これらの問題の一部は、特定状況下でメモリ破壊の形跡が見られ、条件が整えば任意のコード実行に悪用可能と思われるものでした。

Thunderbird と SeaMonkey では JavaScript が初期設定で無効になっているため、一般的にこれらの脆弱性はメールを通じた悪用が不可能です。ただしこれらの製品でも、ブラウザもしくはブラウザに類似した機能を利用した場合、潜在的なリスクが生じます。

参考資料

Firefox ESR 17 と Firefox ESR 24、Firefox 24 に影響するメモリ安全性の問題とクラッシュが、Jesse Ruderman と Christoph Diehl によって報告されました。

  • メモリ安全性の問題 - Firefox ESR 17.0.10、Firefox ESR 24.1、Firefox 25.0 で修正済み (CVE-2013-5590)

Firefox ESR 24 と Firefox 24 に影響するクラッシュが、Vladimir Vukicevic によって報告されました。

  • メモリ安全性の問題 - Firefox ESR 24.1、Firefox 25.0 で修正済み (CVE-2013-5591)

Firefox 24 に影響するメモリ安全性の問題とクラッシュが、Jesse Ruderman、Gary Kwong、Kannan Vijayan によって報告されました。

  • メモリ安全性の問題 - Firefox 25.0 で修正済み (CVE-2013-5592)

Mozilla とその派生製品が使用している NSS ライブラリにおけるクラッシュが Carsten Book によって報告され、Firefox 25、Firefox ESR 24.1、Firefox ESR 17.0.10 で修正されました。

  • Bug 894370 – Assertion failure: inputLen >= spec->mac_size, at c:/work/mozilla/builds/beta/mozilla/security/nss/lib/ssl/ssl3con.c:2057 (CVE-2013-1739)

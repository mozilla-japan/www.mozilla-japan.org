Sunbird のビルド

Sunbird は完全にスタンドアロンなアプリケーションで、クロスプラットフォームな Mozilla Toolkit を用いてビルドされます。 以下に示すことが Sunbird をビルドするための基本的な手引きです。 Mozilla をビルドすることに関してもっと情報が必要ならば Mozilla ビルドページにあたってください。

ソースコードのチェックアウト

Mozilla のソースコードを CVS からチェックアウトします。詳細は Mozilla の CVS の手引き にあります。

Sunbird のビルド（トランクから）

  1. 次の行を .mozconfig に付け加えてください 

    . $topsrcdir/calendar/sunbird/config/mozconfig
  2. Mozilla ビルドページ の手引きにしたがってビルドします。

  3. それが終わったら、次を実行して Sunbird を動かすことができます 

    dist/bin/sunbird

Sunbird 0.3 alpha1 のビルド

Sunbird 0.3 alpha1 をチェックアウトしビルドするには、正しいファイルを手に入れるために以下のステップに従ってください。

そのためには空のディレクトリで次のことをしてください：

  1. CVS の手引き に従ってください
  2. cvs checkout mozilla/client.mk

    とタイプする代わりに

    cvs checkout -r SUNBIRD_0_3a1_RELEASE mozilla/client.mk

    とタイプしてください

Sunbird 0.2 のビルド

Sunbird 0.2 のブランチが切られ、ブランチタグは SUNBIRD_0_2_BRANCH となりました。 もし Sunbird 0.2 リリース版をビルドしたいのならば、そのブランチからコードを取りだしてください。

そのためには空のディレクトリで次のことをしてください：

  1. CVS の手引き に従ってください
  2. cvs checkout mozilla/client.mk

    とタイプする代わりに

    cvs checkout -r SUNBIRD_0_2_BRANCH mozilla/client.mk

    とタイプしてください

  3. ビルドが完了したら xpfe/communicator ディレクトリで "make" を実行してください（バグのためです）

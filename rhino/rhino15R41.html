Rhino 1.5 Release 4.1 変更ログ

1.5R4.1 は 1.5R4 で見つかった問題点のバグフィックス・リリースで、ほとんどが、1.5R3 に退行したことへの対処です。 1.5R4 と比較して、唯一の明らかな API の変更は、org.mozilla.javascript.Context 内にある 2 つの新しいメソッド getApplicationClassLoader(), setApplicationClassLoader(ClasssLoader) です。 それらは、アプリケーションのクラスにアクセスするとき、Rhino が利用するクラスローダーの制御を可能にします。

1.5R3 から 1.5R4 への相違点は、1.5R4 変更ログ を参照してください。

解決済みの Bugzilla レポート

下記は、Bugzilla の Rhino に関するもので、Rhino 1.5 Release 4 の問題解決に向けたものです。

96270 - javascript から java オブジェクトを作成することができない。
193168 - v1.5R4 の Rhino デバッガーは、リロードしてもスクリプトソースを更新しない。
193555 - 1.5R4 で退行 … 関数式が、その名前にアクセスできない。
196017 - 1.5R4 で退行 … いくつかの JDK で、スクリプトはクラスを探し出すことができない。
200551 - js.jar を jre/lib/ext に導入していると、JavaAdapter はクラスをロードできない。

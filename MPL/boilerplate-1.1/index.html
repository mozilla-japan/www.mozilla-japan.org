ライセンス定型文

この表には、新しい Mozilla コードに推奨されている mozilla.org のライセンスである MPL / LGPL / GPL 、または MPL でライセンスされるファイルの先頭に表示される定型文のテキストが、前もってフォーマットされたバージョンという形式で含まれています。ツリーにファイルを追加する際には、必ずこれらのテンプレートを使用してください。将来定型文をアップデートしやすいように、細心の注意を払ってフォーマットされています。

利用方法

ライセンス定型文テキストの使い方は次の通りです：

  • 特別な理由がない限り、すべての新しい Mozilla ファイルには MPL / LGPL / GPL トリプルライセンスを使用しなければなりません。特別な理由がある場合、ファイルをコードベースにチェックインする前に netscape.public.mozilla.license で確認してください。
  • Bugzilla やその他 Web ツールといった、その他のプロジェクトでは MPL のみを使用している可能性があります。この場合は、あなたのプロジェクトが利用しているライセンスをご利用ください。
  • バイナリファイルは、該当ディレクトリにあるその他のテキストファイルと同じライセンスが適用されます。1 つのディレクトリでライセンスが混在している場合は、COPYING というファイルを追加して状況を説明してください。
  • 定型文をダウンロードするにあたっては、ブラウザの "Save Link As..." 機能を使用してください。
  • すべての空欄 【訳注： ______ となっている部分】 には適切なテキストが 記入されて いなければなりません。
  • 定型文テキストにおいて、いかなるセクションも変更・削除してはなりません。
  • ファイルを変更する場合、必要に応じて著作権の年を更新すると共に、貢献者のセクション を必ず更新してください。

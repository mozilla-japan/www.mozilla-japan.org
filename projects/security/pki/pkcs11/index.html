PKCS #11 規格適合テスト

ニュースグループ： netscape.public.mozilla.crypto
テクニカルコンタクト： Bob Relyea
管理者： Wan-Teh Chang

このページでは、現在 Mozilla ツリーで利用可能な PKCS #11 規格適合テストスイートについての情報を纏めます。詳細は下に記載する Mozilla CVS 情報を参照してください。

テストスイートと資料

以下のテストスイートが Mozilla CVS ツリーで利用可能になっています：

  • Netscape PKCS #11 テストスイート：Netscape 製品から PKCS #11 インターフェースを通じて、最も一般的にアクセスされる PKCS #11 操作を実行します。あなたの PKCS #11 実装に対してこのスイートを実行すると、手動によるテストでしか発見できないような非互換性を表面化させるのに役立ちます。

新しいテストスイートを貢献したい場合は、まずあなたのテストの情報と資料を Bob Relyea と Sean Cotter に送ってください。彼らがあなたのソースコードのためのディレクトリを作成し、あなたからの資料へのリンクを追加できるようにします。それから、Mozilla CVS ツリーへアクセスして、割り当てられた領域にあなたのテストスイートをチェックインしてください。

Mozilla CVS 情報

PKCS #11 テストスイートは mozilla/security/nss/test/pkcs11/ で利用できます。Netscape PKCS #11 テストスイートのビルドガイドは、間もなく利用可能になります。

coreconf は mozilla/security/ のルートにチェックインされています。

coreconf/location.mk および coreconf/source.mk は dist を $(COREDEPTH)/dist ではなく、$(COREDEPTH)/../dist に探しに行きます。

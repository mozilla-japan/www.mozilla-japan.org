その他のプロジェクト

このページには、Mozilla ソースに関係があって、私たちが知っているその他のプロジェクトへのリンクを載せていますが、これらは mozilla.org が直接サポートしているものではありません。 私たちは便宜上これらのリンクを含めています。

Komodo IDE
Perl や Python アプリケーションを制作するためのクロスプラットフォームな商用 IDE で、Mozilla のクロスプラットフォーム・ツールキットをベースとしています。

Mozdev.org
Mozdev.org は、Mozilla をベースとしたプロジェクトの Sourceforge のようなものです。 彼らは Mozilla テクノロジーをベースとした新しいプロジェクトを始めたいと思っている誰にでも、無料でホスティングを提供しています。 サイトには、ここに載っていない活動的なプロジェクトが多数あり、そのリストは常に増え続けています。

Mozilla Translator
Mozilla Translator は、Mozilla のユーザインターフェイスを他の言語へ簡単に翻訳できるようにするプログラムです。 これは、ユーザがリンクをクリックすることで簡単にダウンロード／インストールできる自動インストール言語パックを生成するので、自分たちの母国語で Mozilla を使うことができます。

mozillaZine
MozillaZine は、Mozilla プロジェクトの周囲で巡らされている主張やニュース、特に興味深い話題に専念しているサイトです。 このサイトの目標は、つい最近まで十分な方法で参加したり貢献することのできなかったデザイナーや一般のユーザ、グループの中に、Mozilla プロジェクトへの興味をかき立てる手助けをすることです。 ぜひ立ち寄ってみてください！

NewZilla
NewZilla は、Mozilla やその他大きな Mozilla ベースのブラウザ製品をまさに使い始めようとしている人にとってのよくある質問について回答します。 また、Mozilla ベースのブラウザを使い始めようとする Web 開発者のためのリンクを含め、役に立つリンクがたくさんあります。 NewZilla 独自の Mozilla ベース プロジェクトページ をチェックしてみてください。

XUL Planet
XUL や XBL を学ぶためのすばらしいサイトです。 最新のチュートリアルや参考文献もあります。

ZopeMozilla
ZopeMozilla は、Zope アプリケーションサーバの管理にフロントエンドとして Mozilla を使っているプロジェクトです。 XUL と JavaScript をクライアントサイドに、RDF をサーバとのやりとりに使っています。 Zope.org のサイトは、クライアントサイドのアプリケーションコード、サーバコード、プロジェクトのためのメーリングリストをホストしています。

この他に現存するブラウザプロジェクトをご存じですか？ ぜひ教えてください！

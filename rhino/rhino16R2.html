Rhino の変更ログ

Rhino 1.6R2、2005-09-19 にリリース

リリース概要

Rhino 1.6R2 は、Rhino の新しいメンテナンスリリースです。Rhino 1.6Rx は ECMAScript for XML (E4X) を新しくサポートします。 詳しくは、Rhino 1.6R1 の変更ログを参照してください。

このバージョンの Rhino は、公開 API のみを使用しているのなら、 現在 組み込んでいるものと、バイナリー互換のはずです。 何か互換性に関する問題があれば、Bugzilla に報告してください。

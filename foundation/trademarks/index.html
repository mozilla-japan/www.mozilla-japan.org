Mozilla Foundation 商標ポリシー

商標ポリシーに関するドキュメントを複数用意しています。Foundation に連絡を取る前に、FAQ および関連する文書に目を通してください。

  • Mozilla の商標およびロゴ使用に関するポリシー
  • Visual Identity Guidelines - 私たちのロゴを使用する際にはこのポリシーに従ってください
  • 商標使用に関する FAQ
  • ソフトウェア・ディストリビュータのための商標ポリシー
  • 公式ローカライゼーションチームのための商標ポリシー
  • Powered by Mozilla ロゴに関する FAQ
  • ローカライゼーションチームのための Web サイトポリシー

また、私たちが保有する商標に似たドメイン名を使用するにあたっての 契約書 (PDF: mozilla.org) も用意しました。

このたび、私たちは、もともと Mozilla Firefox ブラウザの中で使用するために作成された「フィードアイコン」 (別名「RSS アイコン」) の一般利用を認めることにしました。このアイコンは商標ではありませんが、次のような推奨使用ガイドラインを公表しています。

  • フィードアイコンガイドライン
  • フィードアイコン FAQ

以上のドキュメントを読んで、リクエストや質問がある場合は次のようにしてください。

  • Firefox や Thunderbird の商標利用についての質問や申請: trademarks@mozilla.com
  • 他の製品の商標利用についての質問や申請: trademarks@mozilla.org
  • Bugzilla に商標利用についてのリクエストバグを登録する

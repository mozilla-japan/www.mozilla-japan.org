ポートのインストール方法

このドキュメントには、Mozilla 1.7.6 のシステム必要条件、互換性情報、インストール方法が書かれています。

  • OpenVMS
  • Solaris
  • AIX
  • HP-UX
  • IRIX
  • OS/2

OpenVMS

  • OpenVMS Beta v7.1 以降
  • DECwindows MOTIF v1.2-4 以降
  • Compaq TCP/IP Services Version 5.0 for OpenVMS 以降、MultiNet v4.3A 以降、または TCPware v5.4-3 以降
  • 65 MB のハードディスク空き容量

詳しくは HP OpenVMS Systems - Mozilla インストールガイド・リリースノート をご覧ください。

Solaris

  • Solaris では、Mozilla を実行するために、sunsolve.sun.com から以下のパッチをインストールする必要があります。(Bug 88205)
Solaris Patches/SPARC
2.7
  • 106950-22 - "SunOS 5.7: linker patch"
  • 106327-17 - "32-Bit Shared library patch for C++"
  • 106300-18 - "64-Bit Shared library patch for C++"
  • 108376-42 - "OpenWindows 3.6.1: Xsun Patch"
  • 107650-08 - "OpenWindows 3.6.1 X11R6.4 Xprint Extension Patch"
  • 107636-09 - "SunOS 5.7: X Input & Output Method patch"
  • 112689-02 - "SunOS 5.7: UTF-8 locale ICONV patch"
または、これらのパッチがすべて含まれた推奨・セキュリティパッチ集を ダウンロード、インストール してください。
2.8
  • 109147-09 - "SunOS 5.8: linker patch"
  • 108434-02 - "32-Bit Shared library patch for C++"
  • 108435-02 - "64-Bit Shared library patch for C++"
  • 108773-17 - "SunOS 5.8: IIIM and X Input & Output Method patch"
  • 108652-65 - "X11 6.5.1: Xsun patch"
または、これらのパッチがすべて含まれた推奨・セキュリティパッチ集を ダウンロード、インストール してください。
2.9
  • 112963-05 - "SunOS 5.9: linker patch"
  • 111711-03 - "32-Bit Shared library patch for C++"
  • 111712-03 - "64-Bit Shared library patch for C++"
  • 112661-04 - "SunOS 5.9: IIIM and X Input & Output Method patch"
  • 112785-12 - "X11 6.6.1: Xsun patch"
または、これらのパッチがすべて含まれた推奨・セキュリティパッチ集を ダウンロード、インストール してください。
  • 65 MB のハードディスク空き容量
  • 128 MB の RAM
  • 印刷をするには Xprint がインストールされている必要があります。xprint.mozdev.org が、便利なインストール方法として GISWxprintglue (または GISWxprintglue) パッケージを提供しています。
  • GTK+ と glib ライブラリ - これらのライブラリは Sun Workshop/Forte でコンパイルされている必要があります。gcc でコンパイルされた GTK+/glib は、Sun Workshop/Forte でビルドされた Mozilla バイナリと 互換性がありません。Sun Workshop/Forte でコンパイルされた GTK+/glib ライブラリは、Sun Workshop と gcc どちらでコンパイルされたバイナリとも互換性があります。(Bug 136144、141415、186534 他多数)

AIX

  • AIX 4.3.3 以降
  • gtk+、glib、libiconv、gettext が必要です。(互換バージョンは IBM のサイト にあります)

HP-UX

  • HP-UX ビルドには それぞれの適切なサイトで提供されている gtk+、glib、libiconv、gettext が必要です。
    • HP-UX 11.20 は HP のサイトで提供されている 11i v1.6 用 GTK (b6848ba) のデポをベースにしています。
    • HP-UX 11.00 (と 11.11) は HP のサイトで提供されている 11.0/11i v1.0 用 GTK (b6848ba) または HP-UX Porting Center のデポをベースにしています。
    • HP-UX 10.20 は HP-UX Porting Center で提供されているデポをベースにしています。

IRIX

DecorateTransients オプションなしに twm ファミリーのウィンドウマネージャを使っていると、検索、保存、設定などのポップアップウィンドウが機能しません。ウィンドウマネージャの rc ファイルで DecorateTransients オプションをセットしてください。(Bug 78928)

OS/2

今回のリリースでは、Mozilla の実行に C ランタイム DLL のアップデート (libc05) が必要です。これらの DLL を mozilla.exe と同じディレクトリか LIBPATH のどこかに置いてください。これは mozilla-os2-1.7.6-installer.exe を使ってインストールする場合も同様です。このインストーラは C ランタイム DLL を インストールしません が、実行するには DLL が 必要 です。

最低動作環境

  • Pentium 133 MHz クラスの CPU
  • 64 MB の RAM
  • 70 MB 以上のハードディスク空き容量
  • 64 MB のスワップ空き容量
  • OS/2 Warp 4 (Merlin Kernel)
  • Fixpack 15 以降
  • MPTS バージョン 5.3 以降
  • TCP/IP バージョン 4.1 以降
  • INETVER: SOCKETS.SYS=5.3007, AFOS2.SYS=5.3001, AFINET.SYS=5.3006 以降

注意: これらの INETVER レベルの上に MPTS & TCP/IP バージョンを使わないでください。これより古いスタックでも Mozilla は起動、実行すると思われますが、Mozilla が必要とする一部の機能は、古いバージョンの MPTS には正しく実装されていません。このため、クラッシュやデータ消失の可能性があります。

推奨動作環境

  • Pentium II 266 MHz クラスの CPU
  • 128 MB の RAM
  • 150 MB 以上のハードディスク空き容量
  • 128 MB のスワップ空き容量
  • Fixpack 16
  • MPTS バージョン 5.5 または 6.0
  • TCP/IP バージョン 4.3
  • INETVER: SOCKETS.SYS=6.2004, AFOS2.SYS=6.2000, AFINET.SYS=6.2013
    または
  • OS/2 Warp 4.52 Convenience Pack 2 (メンバー制)
    または
  • eCom Station 1.0 以降

注意: MPTS 6.0 と Fixpack 16 は Software Choice メンバーのみ入手できます。その他のアップグレードは今のところ無償で入手可能です。

カレンダー Mozilla カレンダーは、現在アルファ・テスト中ですが大変便利なもので、 標準準拠のカレンダリング・ソリューションになるでしょう。 Mozilla の 6 人のメンバーがこのアプリケーションのチームを組んでいます。 （Windows と Linux のみ）
インストール | ホーム
スペルチェッカー Mozilla はまだ初期設定でスペルチェッカーを搭載していませんが、 MozDev で開発中のものがあります。
インストール | ホーム
Mail の PGP/GPG 暗号化 Mozilla は、安全な E-mail のために、すでに S/MIME を標準サポートしています。 Enigmail は PGP/GPG サポートのためのアドオンです。（コマンド・ライン PGP/GPG プログラムが必要とされます）
インストール | ホーム
Jabberzilla インスタントメッセージ Jabber はオープンソースのインスタントメッセージクライアントで、 地球上にある他のどんなインスタントメッセンジャー（AOL、MSN、ICQ など）にも話しかけることができます。 Jabberzilla は Mozilla Sidebar の Jabber クライアントです。
インストール | ホーム
MozBlog あなたがネットサーフィンをしているあいだに Mozilla を Blog します。
インストール | ホーム
マウスジェスチャー サポート クリック＆ドラッグのマウスジェスチャーを使って Web をナビゲートします。
インストール | ホーム
MathML エディタ Mozilla は MathML （Mathematics Markup Lauguage） の表示をサポートしています。 （専用 フォント を用いると MathML の表現をより優れたものにすることができます） エディタも開発されています。
インストール | ホーム
BannerBlind 私たちの優秀なポップアップ抑制サポート BannerBlind を追加すれば、ほとんどの Web サイトの広告バナーを隠します。
インストール | ホーム

その他の Mozilla アドオンは、 MozDev や XULPlanet でご覧いただけます。

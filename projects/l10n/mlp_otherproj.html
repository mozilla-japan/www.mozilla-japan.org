内容

Mozilla Organization は必ずしもインターネット・スイート（別名：Seamonkey）である Mozilla を意味する訳ではありません。mozilla.org のオープンコミュニティは、Mozilla を取り巻くその他数多くのコードツリーも管理しています。このページでは、Seamonkey プロジェクトの外側で、誰が、どこで、どのような地域化努力を進めているのかを追跡し、彼らの成果をリストしようと思います。

サポートされているプロジェクトに関して、以下の情報を掲載します：

  • 登録されている地域化プロジェクト
  • 関係者のコンタクト情報
  • そして貢献された成果

現在、私たちは以下の mozilla.org プロジェクトを扱っています：

  • Mozilla Application Suite（プロジェクト Seamonkey）
    ブラウザ、メール・ニュースグループクライアント、HTML ウェブページ・ビジュアルエディタ、IRC チャットクライアント、ウェブ開発ツール（JavaScript デバッガと DOM Inspector）が統合されたインターネット・スイート
  • Mozilla Firefox（旧名：Firebird【訳注：さらにその前は Phoenix】）
  • Mozilla Thunderbird
  • Mozilla Camino

Mozilla Firefox

ローカライゼーションプロジェクトとツリー管理は、Firefox ウェブサイト で行われています。

ここに掲載されている貢献者は全てボランティアで、Mozilla Firefox を自分たちの言語で利用できるように頑張っています。もしあなたの言語が掲載されていないのなら、あるいは登録されている人を支援したいのなら（リストの一番右の欄は提供された最新の地域化リリースを表示しています）、Mozilla Firefox の地域化の方法（そして Mozilla の地域化の方法）を読んで、既に登録されている貢献者に連絡を取り、ニュースグループか MLP スタッフ を経由してご自分を貢献者として登録してください。

Bon Echo Alpha 1 スタートページ

おめでとうございます! あなたは Bon Echo Alpha 1 をダウンロード (またはコンパイル) しました。つまり、あなたは Mozilla 製品をテストするコミュニティの一員に名乗りを上げたわけです。協力に時間はかかりませんし、特別な技術も必要ありません。あなたの協力が Firefox の次期バージョンの向上に役立つのです。

【訳注: このページは、英語版のスタートページを参考のために和訳したもので、実際のスタートページではありません。】

注意: あなたがお使いになっている Bon Echo ビルドは、Firefox の 最終版あるいはプレリリース版ではありません。これはテスト目的に限って公開されているもので、サポートはありません。エンドユーザの方は、Firefox の最新版 をお使いになった方が良いでしょう。

報告ツールを使う
Bon Echo には、正しく表示されない Web サイトを報告するための新しいツールが付属しています。[Help] メニューから [Report Broken Web Site] を選択し、あなたが遭遇している問題に関して、できるだけ多くの情報を提供してください。
Talkback を有効にする
Talkback が表示されたときには、それを活用し、報告を送ってください。Talkback レポートは、どのクラッシュがいちばん深刻で、それがどのくらいの頻度で発生するのかといった非常に貴重なデータを私たちに提供してくれます。Tackback がフィードバックを送信するよう求めたら、あなたはただ [OK] をクリックするだけでいいのです。
品質保証チームへの参加
Mozilla QA (品質保証) には、協力に参加していただける方法をご紹介するページがあります。技術的な知識は必要ありません。品質保証への参加は、Mozilla や Bon Echo をもっとよく知ってプロジェクトに貢献したいと思っている方に最適です。

既知の問題

拡張機能
Firefox の過去のバージョンにインストールされている拡張機能は Bon Echo Alpha 1 と互換性がありません。
ブックマークと履歴
今回のリリースは、バックエンド構造とインポートのテストを目的としています。ユーザインターフェイスに加えられている変更の大半は流動的であり、現時点ではテストを必要としていません。今回のリリースに搭載された新しいブックマークと履歴の保存システムは、過去のバージョンとはまったく別になっており、データが失われた場合は Firefox 1.5 に戻れば古いデータをそのまま利用できます。

その他の既知の問題については リリースノート をご覧ください。

今回の開発マイルストーンでの変更点

フィードバックを必要としている Bon Echo Alpha 1 の新機能は次の通りです。

  • タブブラウズ機能の動作変更
  • ブックマークと履歴の新しいデータ保存レイヤー (SQLlite ベース)
  • 拡張された検索プラグインの書式
  • 拡張機能システムのアップデート - より強固なセキュリティを提供し、ローカライズがさらに容易に
  • svg:textPath を使った SVG のサポート
  • 注目すべきバグの一覧

Firefox と Thunderbird ヘルプ

これは Firebird 【訳注:現在は Firefox】 ヘルプ開発者とエンドユーザが開発について話し合い Firefox ヘルプビューア を高めていくメーリングリストとニュースグループです。

これは技術的サポートのためのリストではありません。 それについて、 あなたは mozilla.org サポートページを見るべきです。 どんなテーマからずれた話し合いもヘルプビューアリストでは無視されるでしょう。

  • メーリングリスト
    • 参加または解 除するには: メーリングリストの設定を変更できるメー リングリスト設定ページを見てください。
    • メッセージを投稿するには: あなたのメッセージをメーリングリストに投稿するには firebirdhelp@mozdev.org へ投稿してください。 リスト参加者のみがメーリングリストへ投稿できます。
    • 以前のメーリングリストメッセージを見るには: リ ストアーカイブページへ行ってください。
  • ニュースグループ
    • news://news.mozdev.org/public.mozdev.firebirdhelp に 設置されている。 アクセスするにはユーザー名 "mozdev" とパスワード "maduser" （引用符なしで）を使ってください。
    • Firefox ヘルプニュースグループに投稿された全てのメッセージは自動的に Firefox メーリングリストに送られる（逆も同様）ことに注意してください。

Seamonkey ヘルプビューア

これは Seamonkey ヘルプビューア開発者が Mozilla ヘルプビューアに関係のある問題について話し合う所です。 これはまた mozilla.org 文書の問題についても話し合われる所です。

  • メーリングリスト
    • 参加または解 除するには:メーリングリストの設定を変更できる メー リングリスト設定ページを見てください。
    • メッセージを投稿するには: あなたのメッセージをメーリングリストに投稿するには mozilla-documentation@mozilla.org へ投稿してください。 リスト参加者のみがメーリングリストへ投稿できます。
    • 以前のメーリングリストメッセージを見るには:リ ストアーカイブページへ行ってください。
  • ニュー スグループ
    • news://news.mozilla.org/netscape.public.mozilla.documentationに 設置されている。
    • Mozilla 文書ニュースグループに投稿された全てのメッセージは自動的に文書メーリングリストに送られる（逆も同様）ことに注意してください。

IRC

もし、あなたが興味を持っているなら、Mozilla ヘルプビューア開発者と生で話し合えますし irc.mozilla.org 上のチャンネル #documentation で興味を持ったパーティを利用できます。 これは Mozilla.org web サイトの問題とヘルプビューアの問題（文書と非文書）について話し合う場所です。

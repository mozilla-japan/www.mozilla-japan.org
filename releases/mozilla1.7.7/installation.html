Mozilla 1.7.7 のインストール方法

過去のバージョンとの非互換性によって引き起こされる問題を避けるため、Mozilla のインストーラは、インストールディレクトリが空でない場合、これを削除して良いかどうかを尋ねます。プロファイルその他の個人用ファイルをインストールディレクトリに置いてある場合は NO を選択してください。そうしないと、これらのファイルはすべて削除されてしまいます。プロファイルが標準の場所に置かれている場合は問題ありません。

プロファイルのバックアップに関するヘルプは プロファイル FAQ をご覧ください。

Linux

システム必要条件

  • 以下のバージョンのライブラリ (またはその互換ライブラリ) が必要です: glibc 2.2.4、XFree86 3.3.x、GTK 1.2.x、Glib 1.2.x、Libstdc++ 2.9.0。Red Hat Linux 7.0、Debian 3.0、and SuSE 7.3 (以降) で動作します。
  • Intel Pentium クラスのプロセッサ (233 MHz 以上を推奨)
  • 64 MB の RAM
  • 26 MB のハードディスク空き容量

インストール方法

注意: Mozilla 1.7.7 をインストールする前に、これまでのバージョンの Mozilla をアンインストールすることをお勧めします。これによって、プロファイルディレクトリに保存されているブックマークや履歴、Cookie などの情報が削除されることはありません。

注意: (Mozilla をインストールした人が) 少なくとも 1 度は Mozilla を起動して自動登録を完了させ、インストールしたディレクトリに compreg.dat ファイルを書き込ませなければいけません。別のユーザ (Mozilla をインストールした人以外のユーザ) が最初にプログラムを起動するとコアダンプが起こります。(Bug 49345)

注意: この手順は bash シェルを使っています。他のシェルを使っている場合は、bash シェルを使うか、お使いのシェルに応じてコマンドを合わせてください。

インストーラを使ってインストールする場合

インストーラをダウンロードして Mozilla をインストールするには、以下の手順に従ってください。

  1. mozilla1.7.7 という名前のディレクトリを作って (mkdir mozilla1.7.7) そのディレクトリに移動 (cd mozilla1.7.7) してください。
  2. インストーラ (mozilla-i686-pc-linux-gnu-1.7.7-installer.tar.gz) をダウンロードしてください。
  3. 次のコマンドでアーカイブを展開してください。
    gunzip -dc <ファイル名> | tar -xvf -
    (これで mozilla1.7.7 のサブディレクトリ mozilla-installer にインストーラが置かれます)
  4. mozilla-installer ディレクトリに移動 (cd mozilla-installer) して ./mozilla-installer コマンドでインストーラを実行してください。
  5. インストールウィザードの手順に従って Mozilla をインストールしてください。
  6. Mozilla を起動するには、Mozilla をインストールしたディレクトリに移動して ./mozilla コマンドを実行してください。

tar.gz アーカイブを使って手作業でインストールする場合

tar.gz アーカイブをダウンロードして Mozilla を手作業でインストールするには、以下の手順に従ってください。

  1. mozilla1.7.7 という名前のディレクトリを作って (mkdir mozilla1.7.7) そのディレクトリに移動 (cd mozilla1.7.7) してください。
  2. tar.gz アーカイブ (mozilla-i686-pc-linux-gnu-1.7.7.tar.gz) を mozilla1.7.7 ディレクトリにダウンロードしてください。
  3. 次のコマンドでアーカイブを展開してください。
    gunzip -dc <ファイル名> | tar -xvf -
    (これで mozilla1.7.7 ディレクトリの下に mozilla サブディレクトリが作られます)
  4. mozilla ディレクトリに移動 (cd mozilla) してください。
  5. ./mozilla コマンドで Mozilla を起動してください。

マルチユーザインストール

Unix で マルチユーザ用に Mozilla をインストールするには、普通にインストールした後、Mozilla ディレクトリに以下のスクリプトを作成してください。スクリプトを実行可能に (chmod u+x <スクリプト名>) してから、その Mozilla ディレクトリに書き込み権限のあるユーザ名 (root など) で実行してください。このスクリプトを su[do] root 権限で実行しないでください。 (Bug 52653)

#!/bin/sh
dist_bin=`dirname $0`
MOZILLA_FIVE_HOME=$dist_bin
LD_LIBRARY_PATH=$dist_bin
export MOZILLA_FIVE_HOME LD_LIBRARY_PATH
$dist_bin/regxpcom
$dist_bin/regchrome
touch $dist_bin/chrome/user-skins.rdf $dist_bin/chrome/user-locales.rdf

これで、インストールされた Mozilla にアクセス権限を持つすべてのユーザが Mozilla を実行できるはずです。

GNOME パネルに Mozilla を追加する

  1. GNOME メインメニューボタンをクリックして Panel メニューを開き、Add to Panel サブメニューから Launcher を選択してください。
  2. パネルの Mozilla アイコンを右クリックして次のコマンドを入力してください。
    <ディレクトリ名>/mozilla
    ディレクトリ名の部分は Mozilla をダウンロードしたディレクトリの名前です。Mozilla が指定するデフォルトディレクトリの場合は /usr/local/mozilla になります。
  3. アイコンに名前をつけ、必要に応じてコメントをつけてください。
  4. アイコンボタンをクリックし、アイコンの場所として次のように入力してください。
    <ディレクトリ名>/icons/mozicon50.xpm
    ディレクトリ名の部分は Mozilla をインストールしたディレクトリの名前です。デフォルトディレクトリの場合は /usr/local/mozilla/icons/mozicon50.xpm になります。

RPM ビルドのインストール

Red Hat RPM ビルドは、デフォルトの Red Hat 7.x/8.x で動くように作られています。これらの RPM ビルドを SuSE で使うには、gdk-pixbuf パッケージをインストールする必要があります。(Bug 85218)

アンインストール方法

  • Mozilla をインストールした mozilla1.7.7 ディレクトリを削除します。
  • プロファイル情報 が保存されている ~/.mozilla ディレクトリを削除します。

Macintosh

システム必要条件

  • Mac OS X 以降 (v10.2 以降を推奨)
  • PowerPC プロセッサ (266 MHz 以上を推奨)
  • 64 MB の RAM
  • 36 MB のハードディスク空き容量

インストール方法

ディスクイメージをダウンロードして Mozilla をインストールするには、以下の手順に従ってください。

  1. お使いのマシンにディスクイメージ (mozilla-mac-1.7.7.dmg.gz) をダウンロードしてください。デフォルトでは、ダウンロードしたファイルはデスクトップに置かれます。
  2. ディスクイメージをダウンロードしたら、それを Stuffit Expander にドラッグして展開します (Stuffit Expander が自動的に起動して、ファイルが展開されます)。ディスクイメージが自動的にマウントされない場合は dmg ファイルをダブルクリックしてマウントしてください。これでもマウントできず、そのファイルがディスクイメージに見えないときは、ファイルメニューから「情報を見る」を選択し、「このアプリケーションで開く」カテゴリで「ディスクコピー」を選択してください。Mac OS X v10.2 以降では、コンテキストメニューから「このアプリケーションで開く」を使ってください。
  3. ディスクイメージをマウントしたら、それを開いて、Mozilla アイコンをハードディスクにドラッグします。
  4. このアイコンはアプリケーションフォルダにコピーすることをお勧めします。
  5. ディスクイメージを取り出します。
  6. 必要に応じて Dock に Mozilla を追加しておけば、いつでも簡単にアクセスできます。また、(Web タブの下にある) インターネットシステム環境設定ペインから、Mozilla をデフォルトブラウザとして選択することもできます (訳注: Mac OS X v10.3 以降では、デフォルトブラウザの変更は Safari の環境設定から行います)。

アンインストール方法

  1. /Users/<ユーザ名>/Library/Mozilla にあるアプリケーションファイルを削除してください (ただし Profiles というサブフォルダは残してください)。
  2. /Users/<ユーザ名>/Library/Preferences から Mozilla Registry というファイルを削除してください。
  3. インストールした場所 (例: /Applications) から Mozilla を削除してください。

Windows

システム必要条件

  • Windows 95/98/98SE/ME/NT 4.0/2000/XP
  • Intel Pentium クラスのプロセッサ (233 MHz 以上を推奨)
  • 64 MB の RAM
  • 26 MB のハードディスク空き容量

一部の ATI ビデオドライバは Mozilla と互換性がなく、クラッシュの原因となる場合があります。クラッシュに遭遇した場合は最新の ATI ドライバに アップグレード してみてください。(Bug 101055)

インストール方法

注意: セットアッププログラムを実行する前に、すべてのプログラムを終了することを強くお勧めします。また、ウイルス検出ソフトが起動している場合は、一時的に無効にしてください。

インストーラを使ってインストールする場合

インストーラをダウンロードして Mozilla をインストールするには、以下の手順に従ってください。

  1. お使いのマシンにインストーラ (mozilla-win32-1.7.7-installer.exe) をダウンロードしてください。
  2. ダウンロードしたファイルのある場所に移動し、Mozilla のプログラムアイコンをダブルクリックしてセットアッププログラムを起動してください。
  3. セットアッププログラムの手順に従って Mozilla をインストールしてください。初回は Mozilla が自動的に起動します。

ZIP ファイルを使って手作業でインストールする場合

ZIP ファイルをダウンロードして Mozilla を手作業でインストールするには、以下の手順に従ってください。

  1. お使いのマシンに ZIP ファイル (mozilla-win32-1.7.7.zip) をダウンロードしてください。
  2. ダウンロードしたファイルのある場所に移動し、ZIP ファイルをダブルクリックしてください。注意: この手順は、あなたが最近のバージョンの WinZip をインストールしていて、その使い方を知っていることを前提としています。そうでなければ、WinZip とこのプログラムに関する情報を www.winzip.com [日本語版] で入手してください。
  3. C:¥Program Files¥Mozilla 1.7.7 などのディレクトリに ZIP ファイルを展開してください。
  4. Mozilla を起動するには、Mozilla を展開したディレクトリに移動し、Mozilla.exe アイコンをダブルクリックしてください。

アンインストール方法

  • インストーラを使った場合は、[設定] > [コントロール パネル] > [プログラムの追加と削除] を開き、Mozilla 1.7.7 を選択、削除してください。
  • ZIP ビルドを使った場合は、Mozilla をインストールした Mozilla 1.7.7 ディレクトリと プロファイルディレクトリ を削除してください。

Mozilla が生成・使用するファイル

Mozilla は、個人用データを プロファイル という場所に保存します。プロファイルには、設定、メールフォルダ、履歴、Cookie、アドレス帳などの重要データが含まれています。以下のリストは、それらのファイルを一覧にまとめたものです。

訳注: OS の種類、インストールしたビルドの違い、インストール時のコンポーネント選択、追加インストール (XPI) の有無などによって、すべてのディレクトリやファイルがあるとは限りません。

次のリストは 標準的な プロファイルの場所を表したものです。

Windows

(ログインを必要としない) Windows 95/98/ME では、標準的なプロファイルディレクトリは
  C:¥Windows¥Application Data¥Mozilla¥Profiles¥<ユーザプロファイル>¥<ランダムな文字列>.slt¥

  C:¥Windows¥Profiles¥<ログイン名>¥Application Data¥Mozilla¥Profiles¥<ユーザプロファイル>¥<ランダムな文字列>.slt¥

Mozilla ユーザー情報交換フォーラム

インターネット掲示板

MozillaZine はユーザー間の情報交換のために様々な掲示板を運営しています。

  • Mozilla Firefox Support [日本語版]
  • Mozilla Thunderbird Support [日本語版]
  • Mozilla 1.x Support [日本語版]
  • Camino

もじら組でもユーザー間の交流や情報交換のためのユーザーフォーラムを設けています。この掲示板では Mozilla に関するあらゆる話題を扱います。上の MozillaZine.jp の掲示板と併せて、是非ご利用ください。

ニュースグループ

Netscape は安全なサーバを利用してユーザー間の情報交換を目的としたニュースグループを運営しています。（Linux ユーザへ：セキュアなニュースグループに接続するには Personal Security Manager（PSM）が必要です。これらのニュースグループに接続できない場合は、PSM を備えた Mozilla を再インストールしてください。）

投稿の前には掲示板のルールをお読みください。

  • 一般的な話題：netscape.mozilla.user.general
  • Windows：netscape.mozilla.user.win32
  • Macintosh：netscape.mozilla.user.mac
  • Unix と Linux：netscape.mozilla.user.unix

役に立ちそうなユーズネット・ニュースグループ：

  • alt.fan.mozilla
  • comp.infosystems.www.browsers.mac
  • comp.infosystems.www.browsers.ms-windows
  • comp.infosystems.www.browsers.x
  • comp.infosystems.www.browsers.misc
リアルタイムチャット

IRC クライアントを使って mozilla.org のチャットサーバ（irc.mozilla.org）上の #mozillazine チャンネルに参加して、他のユーザーとリアルタイムで会話する事ができます。

mozilla.org のチャットサーバでは、#mozillazine-jp という日本語のチャンネルも運営されています。まず、文字コードセットを /charset iso-2022-jp と設定してから入室してください。

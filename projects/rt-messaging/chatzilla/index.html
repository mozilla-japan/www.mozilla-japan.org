Mozilla の IRC、またの名を ChatZilla

訳注：日本語で提供されている IRC チャンネルもあります。詳しくは「もじら組」の mozillaZine IRC のご案内 と、「はじめての Mozilla 1.4」の ChatZilla 機能説明 をご覧ください。

基本

ChatZilla は 完全に JavaScript と XUL で記述された IRC クライアントです。ChatZilla を実行するには Mozilla ベースのブラウザを使用してください。

  • Mozilla ベースのブラウザを 持っていない 場合は、最新の Netscape ブランドのブラウザ をインストールするか、mozillaZine のビルドコメントのページ から最新の mozilla.org ビルドを入手することができます。 【訳注：現在このページは更新されていません。最新の Mozilla は リリース ページから入手してください。】
  • Mozilla ベースのブラウザを 持っている 場合は、今すぐ ChatZilla をインストールできます。hacksrus にある ChatZilla のページ には、あなたのブラウザからすぐにインストールできる形式になっている ChatZilla の最新バージョンが置いてあります。 【訳注：現在、標準構成以上でインストールした Mozilla には、最初から ChatZilla が付属していますので、ここから別途インストールする必要はありません。】
  • ChatZilla でバグを発見したら、未解決バグリスト をチェックして、そこにあなたが発見した問題がなければ ChatZilla に対してバグを登録 してください。

ChatZilla が正しくインストールされると、次のどの方法でもスタートできます。

  • ［タスク］メニューから IRC Chat を選択【訳注：［タスク］メニューがなければ「ウィンドウ」メニューを開いてください】
  • アドレスバーに irc: urlと入力
  • -chat オプションでブラウザを起動
  • -chrome chrome://chatzilla/content オプションでブラウザを起動
  • 【訳注：コンポーネント バーに cZ というアイコンがあれば、それをクリックしてもスタートできます】

貢献者

Robert Ginda（）は ChatZilla の原作者です。

Samuel Sieb（）は、今では 1年以上、プロジェクトに対して常に貢献してきました。彼は多くのバグを登録してパッチを提供し、ChatZilla の大きな力になっています。

James Ross（）は、ChatZilla の設定パネルに貢献し、また、他の多くのバグに対してもパッチを提供しました。

Peter Van der Beken（）は、bslib の代わりに necko を使うため connection-xpcom.js を再実装しました。こうして ChatZilla を C++ コードから解放したのです。彼は ChatZilla の最高の Mac 仲間でもあります。Peter は（jar.mn ファイルが存在する前） ChatZilla 関連のビルド goop を Macintosh 版で最新のものに保ち、自分の Mac で ChatZilla を主要クライアントとして動かしています。

Christine Begle は /stalk と /unstalk コマンドを記述しました。

Josh Gough（）は、ChatZilla が mungus という名前の JavaScript 'bot だったときに、それをハックしています。彼は ChatZilla のために多数のコマンドを実装し、ユーザリストにコンテキストメニューサポートを追加しました。

Dan Matejka は manifest.rdf ファイルが導入された際に ChatZilla makefile を移行しました。

Simon Fraser は、初期の Macintosh ビルド goop を統合し、Mac ユーザが ChatZilla の楽しみを体験できるようにしました。

mozilla.org ソースツリーで ChatZilla をビルドする

Unix

Unix などの autoconf 環境では、 --with-extensions=default,irc を付けて configure して必要な makefile を生成し、それから再ビルドしてください。ツリーが既にビルドされている場合は、configure の後に extensions/irc で gmake を行えば IRC クライアントを追加することができます。

Windows & Mac

ChatZilla はデフォルトで Windows 版と Mac 版がビルドされています。

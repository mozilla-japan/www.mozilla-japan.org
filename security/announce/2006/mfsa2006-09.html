タイトル
イベントハンドラを使ったクロスサイト JavaScript インジェクション
重要度
高
公開日
2006/04/13
報告者
shutdown
影響を受ける製品
Firefox
Thunderbird
Mozilla Suite
修正済みのバージョン
Firefox 1.5
Firefox 1.0.8
Thunderbird 1.5
Thunderbird 1.0.8
Mozilla Suite 1.7.13
SeaMonkey 1.0

概要

他のページに遷移する際に、モーダルな警告ダイアログを使ってイベントハンドラを一時停止させることで、その異なるサイトのページに JavaScript コードを注入して実行させる方法が shutdown 氏によって報告されました。攻撃者はこの脆弱性を利用して、被害者になりすましてそのサイトにログインできるパスワードや Cookie など、遷移後のページに含まれる可能性のある秘密情報を盗み取ることが可能です。

shutdown 氏からは、モーダルダイアログを必要とせず、より目立たない、2 つの引数を取る eval() の形式を用いた別の方法も報告されました。また、最初の修正を回避する 2 つの別の方法が moz_bug_r_a4 氏によって報告されました。このうちひとつは new Script() を使うもので、もうひとつは window.__proto__ を使って eval() 形式の攻撃を拡張したものでした。

注意: Thunderbird は Firefox と同じブラウザエンジンを利用しており、JavaScript が有効になっている場合には影響を受けます。デフォルト設定では無効になっており、メールを表示する際は JavaScript を実行させないよう強く推奨します。

回避策

修正版にアップグレードするまで JavaScript を無効にしてください。

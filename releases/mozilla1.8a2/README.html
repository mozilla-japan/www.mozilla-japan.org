Mozilla 1.8 Alpha2 新着情報

このドキュメントには、Mozilla 1.8 Alpha2 で実装された新機能や重要なバグの修正、その他新たな問題の一部が載っています。1.8 Alpha2 で行われた修正の詳細なリストは「大まかな変更ログ」をご覧ください。

新機能と修正

ブラウザ

  • Mozilla 1.8 Alpha2 では、[Tools] メニューの [Popup Manager] を開いた中にある [Allow Popups From This Site]、またはステータスバーにあるポップアップインジケータのコンテキストメニューアイテムから、一度ブロックされたポップアップを次回からブロックしないよう設定することが可能になりました。

メール

  • Mozilla Mail に、同じフォルダ階層から複数の POP3 アカウントを管理できるようにする「グローバル Inbox」が実装されました。

バックエンド

  • Mozilla 1.8 Alpha2 は、最近の プレスリリース で発表された新しい Netscape Plugin API 拡張を初めてサポートしました。詳しくは "Scripting Plugins in Mozilla" をご覧ください。

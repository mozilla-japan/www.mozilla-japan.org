6.1. IRC (Internet Relay Chat) とは何で、ChatZilla とは何ですか？

IRC は、インターネット・チャット・ネットワークの元祖です。IRC の一般的な概説は、Open Directory の IRC ヘルプ セクションを見てください。

ChatZilla は、完全に XUL と JavaScript で書かれた Mozilla アプリケーションであり、Mozilla テクノロジーで何ができるかを示すデモの一つです。

6.2. 「cZ」ボタンをクリックして、ChatZilla を起動しました。この後どうすれば良いのですか？

ChatZilla は、他のほとんどの IRC プログラムと同様、標準的なコマンド (/server、 /join、 /nick、 など) をサポートしています。ChatZilla ホームページ には、新規ユーザーの為の FAQ リストがあります。

手始めに単純なコマンドをいくつか：

/networks
全ての有効なネットワークをリンクとしてリスト。
/server サーバ名
特定のサーバに接続。別の方法としては、ChatZilla ウィンドウの上に出ている「[INFO] 有効なネットワーク」（[INFO] Available networks are）でリストされているネットワーク名のどれかをクリックしてください。
/nick 新しいニックネーム
新しいニックネームに変更。
/list
ネットワークで開かれている全てのチャンネルをリスト。
/join #チャンネル名
#channel 名を使用してチャンネルに参加。
/me アクションの種類
あなたのニックネームでアクションを実行。
/whois ニックネーム
サーバ上の人物に関する情報。
/help コマンド
コマンドに関するヘルプ。
/help
単独では、全てのコマンドのヘルプを表示。

MozTips にも、広範な ChatZilla の導入ガイドがあります。

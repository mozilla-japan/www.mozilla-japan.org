タイトル
EvalInSandbox の回避 (プロキシ自動設定、Greasemonkey)
重要度
中
公開日
2006/06/01
報告者
moz_bug_r_a4
影響を受ける製品
Firefox
Thunderbird
SeaMonkey
修正済みのバージョン
Firefox 1.5.0.4
Thunderbird 1.5.0.4
SeaMonkey 1.0.2

概要

EvalInSandbox を通じて実行した JavaScript が、サンドボックスの外で作成してから挿入したオブジェクトの valueOf() 関数を呼び出すことで、そのサンドボックスを回避して高度な特権を得られることが、Mozilla 研究家の moz_bug_r_a4 氏によって実証されました。悪質なスクリプトがこの特権を利用して、ユーザのコンピュータやデータを侵害する可能性があります。

Mozilla 製品での EvalInSandbox の主な用途は、ネットワーク管理者によって指定されたプロキシ自動設定スクリプトの実行です。これが一般のホームユーザに利用されていることはまれで、主にリモート設定が必要とされる組織ネットワーク内で利用されているものです。

人気のある Greasemonkey 拡張機能は、訪れた Web ページを好みに合わせて操作できるユーザスクリプトの実行に EvalInSandbox を利用しています。この脆弱性を利用して、悪質なユーザスクリプトが特権を得て、悪質のあるソフトウェアをインストールする可能性があります。ただし、Greasemonkey が設計通りに機能した場合であっても、その悪質なユーザスクリプトを使った攻撃が可能です。ユーザスクリプトをインストールする場合は、必ず信頼できるサイトかどうか確かめてください。

回避策

オプションウィンドウの接続設定で、デフォルトの「インターネットに直接接続する」または「手動でプロキシを設定する」を選択してください。

Greasemonkey を利用している場合は、信頼できるサイトからのみユーザスクリプトをインストールし、valueOf() が含まれていないかどうか確かめてください。または、単純に新しいバージョンに更新するまで Greasemonkey を無効にしてください。

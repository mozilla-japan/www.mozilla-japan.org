Mozilla Firefox プライバシーポリシー

2006 年 10 月

私たちは、パーソナルコンピュータを利用する上で必要なパフォーマンスと保護機能を備えた製品やサービスを提供するとともに、プライバシー保護にも力を入れています。本プライバシーポリシーでは、非営利団体 Mozilla Foundation の完全子会社である Mozilla Corporation (「Mozilla」) が公式な Mozilla Firefox® Web ブラウザ (「Firefox」) でユーザの皆さんの情報をどのように収集し利用しているのかを説明します。本ポリシーは、Mozilla 関連の他の Web サイト、製品、またはサービスには該当しません。

情報の種類

Firefox は多くのインターネット Web ブラウザと同様に、閲覧する Web サイトに特定の情報を送信します。この情報は次の 3 つのカテゴリに分類できます。

  • 個人情報：「個人情報」とは、その他の追加情報なく、特定の人物や団体に直接関連付けることができる情報を指します。例として、名前、電話番号、または電子メールアドレスなどが挙げられます。Mozilla は以下に示す以外、Firefox のエンドユーザの皆さんに対して個人を特定できる情報を提供するようには求めません。
  • 個人が特定されない情報：「個人が特定されない情報」とは、特定の人物や団体に直接関連付けられない情報を指します。例として、コンピュータの設定やお使いの Firefox のバージョンなどが挙げられます。
  • 潜在的な個人情報：「潜在的な個人情報」とは、情報自体は個人が特定されない情報でありながら、他の情報と連携させることでお客様を特定できるようになる可能性をもつ情報を指します。例として、Uniform Resource Locator (「URL」) (Web ページのアドレス) やインターネットプロトコル (「IP」) アドレス (インターネット上のコンピュータのアドレス) などがあり、これらの情報自体は個人が特定されない情報です。しかし、インターネットサービスプロバイダ (「ISP」) の記録と照らし合わせることで個人情報になり得ます。

Firefox が Web サイトに送信する情報

Firefox は多くの Web ブラウザと同じく、閲覧している Web サイトに情報を送信します。送信される情報は、(1) 使用しているブラウザの種類、言語設定、参照元のページといった、普段 Web ブラウザが送信する個人が特定されない情報、そして (2) IP アドレスといった潜在的な個人情報です。これらの情報は閲覧する Web サイトに記録される可能性があります。どういった情報が記録され、その記録された情報がどのように利用されるかは、Web サイトの方針によります。

Web サイトは、個人が特定されない情報および潜在的な個人情報の配布および使用に関して、それぞれ独自にプライバシーポリシーを定めています。こうした情報を Web サイトがどのように利用するのかについて懸念がある場合は、そのサイトのプライバシーポリシーをご確認ください。Mozilla が同社の所有する Web サイトでこれらの情報をどのように利用するのかについては、Mozilla プライバシーポリシー をご覧ください。

インタラクティブな製品機能

  • クラッシュ報告機能：Firefox のクラッシュ報告機能である「Talkback」は、Firefox がクラッシュすると Mozilla に報告を送信します。Mozilla は、報告にある情報を利用して Firefox 内に存在してクラッシュを引き起こした問題を診断し修正します。Talkback は Firefox がクラッシュすると自動的に起動しますが、お客様が明示的に情報の送信を許可しない限り Mozilla に情報は送信されません。Talkback は標準で、さまざまな個人が特定されない情報を Mozilla に送信します。その例として、スタックトレース (クラッシュ時にアクティブだった Firefox コードの詳細情報) や使用しているコンピュータの種類が挙げられます。加えて、お客様の IP アドレス、コンピュータ名、そしてクラッシュ時に実行していたプロセスといった、個人情報および潜在的な個人情報も収集されます。お客様は、これら情報のなかから取捨選択して、送信しないようにもできます。さらに、Firefox がクラッシュした際に閲覧していたサイトの URL、コメント、そして電子メールアドレスを報告に含めるオプションもあります。Mozilla は、Talkback の報告に含まれる個人が特定されない情報および潜在的な個人情報を公開します。これらの情報は http://talkback-public.mozilla.org/ で閲覧できます。お客様のプライバシー保護のために、Mozilla は報告に含まれる個人情報を、同社従業員、契約者、およびお客様の個人情報を Mozilla 関連の目的以外では利用または公表しないという守秘義務契約を結んだ一部の参加者に対してのみ公開します。
  • 問題のある Web サイトを報告する機能：Firefox の問題のある Web サイトを報告する機能では、訪れた Web サイトが正しく表示されなかったり、誤った動作をする場合に、その問題を Mozilla に報告できるようになっています。この機能は、問題のある Web サイトの URL を Mozilla に送信します。加えて、お客様の電子メールアドレスおよび問題の説明を送ることもできます。この機能はまた、さまざまな個人が特定されない情報を Mozilla に送信します。例として、ご利用の Firefox のバージョンや言語設定が挙げられます。Mozilla は、お客様の電子メールアドレス以外、すべての情報を公開します。この機能では、お客様が明示的に情報の送信を許可しない限り Mozilla に情報は送信されません。個人情報や潜在的な個人情報が公開されてしまわないように、Web サイトの URL にお客様の個人情報や潜在的な個人情報を含む Web サイトは報告しないようにするとともに、問題の説明に個人情報を含めないでください。いかなる情報も公開したくない場合は、この機能を使って問題のある Web サイトを報告しないでください。
  • 偽装サイトを報告する機能：Firefox の偽装サイトを報告する機能では、悪意を持ち、正当な Web サイトになりすました、疑わしい「フィッシング詐欺」サイトまたは偽装サイトに遭遇した場合に、偽装が疑われる Web サイトを Mozilla およびフィッシング詐欺サイト警告サービス機能のために同社が契約するサードパーティサービスプロバイダに報告できます。この機能は、不正が疑われる Web サイトに関するお客様のコメント、および一般の Web サイトを訪問する際にブラウザが送信するのと同様の情報を Mozilla および同社のサードパーティプロバイダに送信します。Mozilla のサードパーティプロバイダ各社は、Mozilla と契約を交わしており、個人情報を公開することはありません。これらサードパーティプロバイダは、寄せられた情報を、各プロバイダが提供するフィッシング詐欺サイト警告サービスとの関連においてのみ利用します。加えて、各サードパーティプロバイダは独自のプライバシーポリシーを定めており、Web サイト捏造の疑いを報告するオンラインフォームからリンクされています。サードパーティプロバイダにいかなる情報も提供したくない場合は、この機能を使って Web サイト捏造の疑いを報告しないでください。(併せて下記の「フィッシング詐欺サイト警告サービス」もご覧ください。)
  • 自動更新サービス：Firefox の自動更新機能は、Mozilla から Firefox のアップデート版が公開されていないか定期的に確認します。この機能は、個人が特定されない情報を Mozilla に送信します。例として、お使いの Firefox のバージョン、インストールされているアドオンのリスト、そして言語設定が挙げられます。この機能はまた、個々の Firefox インストールを区別するために、一意の数値を含む「aus」という名前がついた Cookie という形で、潜在的な個人情報を Mozilla に送信します。Mozilla は、お客様に Firefox のアップデート版がある旨をお知らせし、Firefox ユーザの利用パターンを把握するために、この個人が特定されない情報および潜在的な個人情報を利用します。私たちはこの情報を利用して、製品およびサービスの向上につなげ、機能および性能策定の際の方針決定に役立てます。私たちは、いかなる個人情報やお客様が閲覧する Web サイトについてのいかなる情報も収集または追跡しませんし、この機能を経由して寄せられた情報をそのまま公開することはありません。ただし、コミュニティ全体がより良い製品や設計を行えるように、集積データを含むレポートを公開する可能性はあります。Mozilla にこれらの情報を寄せたくない場合は、Firefox の設定画面でこの機能を無効にすることができます。設定項目の変更についての情報は、ヘルプページ をご覧ください。

フィッシング詐欺サイト警告サービス

Firefox のフィッシング詐欺サイト警告サービスでは、訪れようとしている Web サイトが正当な Web サイトに成りすましている疑いのある場合に警告を表示します。Firefox は、さまざまレベルでのフィッシング詐欺サイト警告サービスを提供しており、お客様の設定に応じて Firefox は異なる情報を送信します。

Firefox は標準で、お客様が訪れる Web ページを Web 偽装が疑われるサイトのリスト (「ブラックリスト」) に照会します。このリストは定期的なスケジュールにしたがってお客様のハードドライブにダウンロードされま す(例：1 時間におよそ 2 回。頻度は状況に応じて変化します)。リストに一致すると、Firefox は「偽装サイトの疑いがあります」という警告を表示します。Firefox がサードパーティプロバイダにチェックインして新しいブラックリストをダウンロードするたびに、お客様のシステムにあるブラックリストのバージョン番号をはじめ、ブラウザで Web サイトを訪れる際に普段送信される個人が特定されない情報および潜在的な個人情報が、サードパーティプロバイダに送信されます。お客様のプライバシーを保護するため、この Firefox の標準モードでは訪問した Web ページの URL は転送されません。

別の方法として、閲覧する Web ページをサードパーティサービスが提供するオンラインリストでチェックすることもできます。このオプションを選択すると、Firefox は Web ページの URL を、お客様の IP アドレスおよびその他の個人が特定されない情報とともに、選択したサードパーティサービスプロバイダへと送信します。Firefox は、アクセスしようとしている URL が Web 偽装の疑いがあるとする反応をサードパーティサービスプロバイダから受け取ると、警告を表示します。

最後に、フィッシング詐欺警告メッセージに対して何らかの対応をした場合、その時の反応およびページの URL がそのフィッシング詐欺サイト警告サービスプロバイダに記録されるとともに、お客様のコンピュータに Cookie が保存される可能性があります。ご利用のサービスプロバイダに送信された URL そのものに個人情報が含まれる可能性もあるため、Mozilla のサードパーティサービスプロバイダは Mozilla との間で、サービスの向上および維持以外の目的で個人情報を利用しないとの合意を書面で交わしています。これらサードパーティサービスプロバイダは、独自に該当するプライバシーポリシーをお客様にお知らせする可能性があります。

Firefox の設定画面でフィッシング詐欺サイト警告機能を完全に無効化することもできます。この機能を無効化すると、ここで挙げられた情報はお客様のハードドライブには一切ダウンロードされず、いかなるサードパーティサービスプロバイダへも送信されません。設定項目の変更についての情報は、ヘルプページ をご覧ください。

この段落では、お客様に怒鳴っているわけではありませんのでご了承ください。お客様に必ず読んでいただけるよう、目立つようにしなければならないとのアドバイスを弁護士から受けているのです。フィッシング詐欺サイト警告機能は「現状 (as is)」のまま、お客様へのアドバイスおよびガイドのための参考情報としてのみ提供されます。Mozilla は、フィッシング詐欺サイト警告機能によって悪意ある Web サイトによる詐欺被害からお客様を防ぐという保障はしません。オンライン状態の間、特に電子メールでお客様のもとに送られたリンクをたどる際には、引き続き警戒を怠らないよう強い喚起を促します。Mozilla Firefox エンドユーザソフトウェアライセンス契約 はこの点についてより詳細に論じています。

Cookie

Cookie とは、Web サイトがお客様のコンピュータ上に保存する小さな情報の断片で、その Web サイトにアクセスするたびにサイトはブラウザ経由でその情報にアクセスします。Firefox は、Web サイトからのリクエストに応じてお客様のコンピュータ上に Cookie を保存します。Web サイトは Cookie を、閲覧者、当該 Web サイトの利用状況、そして複数回にわたるリクエストおよび閲覧にわたってその訪問者による Web サイトのアクセス傾向の特定および追跡の一助にします。Cookie には、個人情報や個人情報に言及する情報が含まれる可能性があり、そのため Web サイトは特定個人のオンライン上での行動を追跡できるようになっています。Firefox が特定の Web サイトに Cookie を送信しないようにするには、Web サイトが Cookie を設置したい場合に Firefox がダイアログを表示するよう設定してください。Firefox がいかなる Web サイトにも Cookie を送信しないようにするには、Cookie を無効化してください。設定項目の変更についての情報は、ヘルプページ をご覧ください。

プライバシーポリシーの変更

Mozilla は Firefox プライバシーポリシーを変更することがあります。いかなる変更もすべてこのページに反映されます。Mozilla が本ポリシーに重要な変更を行う際には、www.mozilla.com の Web サイトに告知を掲載します。大幅な変更を行う際にも、「mozilla-announce」メーリングリストといった Mozilla が同社のユーザおよびコミュニティと連絡を取る際と同様のメカニズムを通してアナウンスされます。【訳注: 英語のみのサービスです】 お客様の責任において、本プライバシーポリシーの条項をかならずご理解ください。現行ポリシーに変更がないかどうか、定期的に本ページを確認されることをお勧めします。

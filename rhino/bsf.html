Rhino と BSF

BSF って何？

Bean Scripting Framework (BSF) は、もとは IBM によって開発されたもので、 現在は Apache Software Foundation のオープンソース・プロジェクトとして公開されています。 それは、Java での様々なスクリプト言語を利用するためのフレームワークを提供します。Rhino はサポートされている言語のひとつです。 【訳注: Jakarta BSF 】

このフレームワークは、数多くのオープンソース・プロジェクトに組み込まれていて、 XSL プロセッサーの Xalan や、XML/Java ビルドツールの Ant にも含まれています。 XSL で JavaScript を使い、また、Ant ビルドファイルにスクリプトを記述するための Apache Ant マニュアル【訳注: Ant 日本語マニュアル】 にある、optional Script task (オプションタスクの Script) の記述についての、さらなる情報には Xalan-Java 拡張 を参照してください。

Rhino と BSF を使用する

Apache Jakarta Bean Scripting Framework (BSF) バージョン 2.4.0、が公式リリースされたので、BSF と共に Rhino を容易に使用することができます。 http://jakarta.apache.org/bsf/index.html を参照してください。

ftp.mozilla.org - jp ミラー

このページは JavaScript を使っています。
一部のブラウザでは 文字コードを Auto-Detect にしたままでは 期待通りに動作しません。
ミラーリストが表示されない場合は 文字コードを EUC-JP にして リロードしてみてください。
私たちのファイルアーカイブは、 ftp://ftp.mozilla.org と http://ftp.mozilla.org で利用できますが、同様に、世界中に数多くのミラーサイトがあります。 フィードバックは mirrors@mozilla.org まで。 私たちの FTP サイトは、暗号に関するソースコード や その ソースコード から派生する バイナリー を含んでいて、 それらは アメリカ合衆国を含むいくつかの国では規制されています。 私たちの FTP サイトをミラーリングする前に、 Crypto FAQ を読んでください。
このソースコードは、アメリカ合衆国の 輸出管理規制、およびその他の合衆国法の対象であり、 いくつかの国へは輸出、および再輸出できません （現在は、アフガニスタン（タリバン勢力域）、キューバ、イラン、イラク、リビア、 北朝鮮（朝鮮民主主義人民共和国）、スーダン および シリア）。 また 合衆国からの輸入を禁止されている人や団体 への輸出、および再輸出もできません （輸出管理局のリストに記載された非合法団体、組織、 および 特別指定外国人を含みます）。

各ミラーによって ディレクトリやファイルの登場が オリジナルサイトより 数時間から数日 遅れるかもしれません。また、一部のディレクトリが置かれない ミラーサイトもあるかもしれません。ミラーの宿命ですが、ネットワーク・サーバ負荷分散のため ミラー利用に御協力を。

  JavaScript を使って 新規ウィンドウを開きます。

このドキュメントの URL の後ろに ?[ディレクトリ] をつけて ディレクトリやファイル名を指定したミラーリストを開くこともできます。たとえば mirrors_jp.html?mozilla/releases/mozilla1.4/ といったふうに。

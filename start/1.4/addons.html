Mozilla アドオン

より詳細なアドオンの一覧は、mozdev.org の Extension Room でご覧になれます。開発者の方は Web 開発者アドオン 一覧をご覧ください。

注意: Mozilla 用のプラグイン (Java、Flash など) を手に入れるには プラグイン をご覧ください。

一般的なアドオン

地域化 (ローカリゼーション) Mozilla のユーザインターフェイスを、英語から他の言語に変更できます。
  • 現在利用可能な Mozilla 1.4.1 言語パック
  • 現在利用可能な Mozilla 1.4 言語パック

利用可能な言語の数は増え続けていますので、もしあなたがお使いの言語がリストになければ、また今度チェックしてみてください。

あなたの言語への地域化に貢献する方法についての詳細と情報は Mozilla 地域化プロジェクト をご覧ください。 貢献された地域化資料の 完全なリスト もあります。

スペルチェッカー辞書 各言語のスペルチェッカー辞書は MozillaCafe からダウンロードできます。
ダウンロード
カレンダー Mozilla Calendar は、現在アルファ版のテスト段階ですが、大変便利で、標準準拠のカレンダリング・ソリューションになるでしょう。 Mozilla アプリケーションスイートの 6 番目のメンバーです。(Windows、Linux のみ)
インストール | ホーム
検索エンジン Mycroft は、現存するあらゆる検索エンジンを Mozilla で実用的に利用できるようにするファイル集です。
(訳注: 日本語で利用可能な検索プラグインは もじら組のサイト から利用できます)
インストール | ホーム
メールの PGP/GPG 暗号化 Mozilla は、メールを安全にやりとりするために、S/MIME を標準サポートしています。 Enigmail は PGP/GPG サポートのためのアドオンです。(コマンドラインの PGP/GPG プログラムが必要です)
インストール | ホーム
Jabberzilla インスタントメッセージ Jabber は、オープンソースのインスタントメッセージクライアントで、地球上のあらゆるインスタントメッセンジャー (AOL、MSN、ICQ など) と会話できます。 Jabberzilla は Mozilla Sidebar の Jabber クライアントです。
インストール | ホーム
MozBlog あなたがネットサーフィンをしているあいだに Mozilla で Blog します。
インストール | ホーム
マウスジェスチャーサポート クリック & ドラッグのマウスジェスチャーを使って Web をナビゲートします。
インストール | ホーム
MathML エディタ Mozilla は MathML (Mathematics Markup Lauguage) の表示をサポートしています。 (専用 フォント を使えば、MathML の表現をより優れたものにすることができます) エディタも開発されています。
インストール | ホーム
BannerBlind 私たちの優秀なポップアップ抑制サポート BannerBlind を追加すれば、ほとんどの Web サイトの広告バナーを隠します。
インストール | ホーム
AutoScroll AutoScroll は、マウスの中ボタンの機能を変更し、マウスを上下に動かすだけで Web ページをスクロールできるようにします。
インストール | ホーム
Crash Recovery どのようなプログラムでも必ずクラッシュします。 Recall プロジェクトは、クラッシュしたときに訪れていた Web サイトを記憶して、それらのクラッシュから回復できるように設計されています。
インストール | ホーム
タブブラウザ拡張 Mozilla のタブに必要な機能が付いていませんか? 「タブブラウザ拡張」がその機能を実現します。 タブの並び替えや、ウィンドウの代わりに新しいタブで開く機能などが含まれています。
ホーム

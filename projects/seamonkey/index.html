SeaMonkey® プロジェクト

SeaMonkey プロジェクトは、かつて「Mozilla Application Suite」として知られたアプリケーションをベースに、高品質なリリースを送り出すことを目指すコミュニティによる活動です。Mozilla Corporation の主力製品は、Firefox へ移っていますが、ボランティアから成る私たちのグループでは、ユーザが、人を引きつける、統合インターネットソフトウェアスイーツ — を提供されること ― さらに企業レベルの使用に耐えるだけの安定性を目標に活動しています。

製品： SeaMonkey® Suite

Navigator

タブブラウズ機能 でより快適にネットを閲覧できます。新しいページの度に新しいウィンドウを開く必要はもうありません。この機能により、一回のクリックで一度にいくつものページを開くことができます。さらにタブを使って複数のページをホームページに設定することができます。

ポップアップブロック機能 のおかげで、目障りな広告に煩わされることなく Web を閲覧できます。

画像マネージャ では画像全般、不快な画像を遮断したり、Web サイトの描画を高速化できます。

Find As You Type はページを閲覧する際のもう一つの手段を提供します。タイプをすればリンクからリンクへとジャンプでき、ページ内の単語や語句を探すことができます。

さらに現代のブラウザが備えているべき機能を備えています：進化したセキュリティ設定、パスワード、ダウンロード、そして Cookie マネージャ、テーマ、複数言語・複数プラットフォーム対応、そして最新の Web 標準のサポート。

Mail & Newsgroups

迷惑メールフィルタ はスパマーの手から電子メールの管理を取り戻します。SeaMonkey の学習型迷惑メールフィルタは使うたびに進化し、あなたが受け取る電子メールに最適化されます。

カスタマイズ可能な ラベル機能 と メール表示 機能を使ってメールを管理してください。カラーコードを使って電子メールの優先度を設定できます。表示機能を使って電子メールをより素早く表示・検索できます。

SeaMonkey は 複数アカウント もサポートしており、ひとつのインターフェイスからすべてのメールを管理することができます。

SeaMonkey Messenger には S/MIME、受信通知、アドレス帳、LDAP サポートや電子署名といった企業向け機能も備えています。

Composer

SeaMonkey の HTML エディタ も進化し続けています。動的な画像・テーブルサイズ調整機能、テーブルセルの簡単な挿入・削除、改良された CSS サポート、さらにレイヤーの位置指定もサポートしています。簡単な文書・ウェブサイトプロジェクトであれば、Composer ですべて整います。

ヘルパーアプリケーション

Christian Biesinger <cbiesinger@web.de>

このドキュメントは、Mozilla の「ヘルパーアプリケーション」という概念について、それがどういうもので、何に使われるのか、ということを解説する目的で作成されました。ヘルパーアプリケーションのユーザインターフェイスは [Edit (編集)] メニューの [Preferences (設定)] > [Navigator] > [Helper Applications (ヘルパーアプリケーション)] にあり、また Mozilla が処理できない種類のファイルに出会ったときに表示されます。

ヘルパーアプリケーションはプラグインではありません

これは良くある誤解だと思います。ヘルパーアプリケーションは、ある特定の種類のファイルを Mozilla 本体でもプラグインでも処理できない場合にのみ使われます。このような場合に限って、Mozilla は [Helper Applications] パネルで選ばれたアプリケーションを使ってファイルを開きます。

ヘルパーアプリケーションとは

Mozilla は時々、zip ファイルなど、自分では表示できない URI を読み込もうとすることがあります。そのような場合、Mozilla はヘルパーアプリケーションの設定を参照し、どのように処理すべきかを確認します。このとき、次のようないくつかの可能性があります。Mozilla はファイルを (システム全体の) デフォルトアプリケーションで開くことができます。また、ユーザが指定したアプリケーションで開いたり、あるいはディスクに保存することもできます。

これらはすべて [Helper Applications] パネルで設定することができます。Mozilla はまず、サーバが送ってきた MIME タイプを調べ、そこで設定されたエントリーを見つけようとします。それがうまくいかなかった場合、サーバが送ってきたファイルの拡張子を使ってエントリーを探します。このエントリーの設定があった場合、それが使われます。

エントリーが見つからなかった場合、Mozilla はその処理方法をユーザに尋ねるようデフォルトで設定されています。このときのデフォルトは [Open with default application (デフォルトアプリケーションで開く)] になっています。Mozilla がユーザに尋ねて、ユーザが選択した場合、この選択された設定がその後も使われるよう保存されます。この設定は [Helper Applications] パネルで見ることができます。

拡張子と MIME タイプ

ローカルファイルその他のファイルで、Mozilla がサーバから MIME タイプを取得できない場合、ヘルパーアプリケーションのエントリーを使って、特定の拡張子の付いたファイルに特定の MIME タイプを割り当てることができます。この設定をするには、[Helper Applications] パネルにエントリーを新規作成し、[MIME Type (MIME タイプ)] 欄に設定したい MIME タイプ、[Extension (拡張子)] 欄に (複数の) 拡張子を入力するだけです。残りはデフォルト値のままにしておいて構いません。

Mozilla がそのタイプを内部で処理可能になったことを告げるダイアログが表示されるはずです。ファイルに割り当てられた MIME タイプを変更したいだけなら、([Proceed anyway (構わず先に進む)] をクリックして) このメッセージを無視することができます。

1 種類の MIME タイプに複数のヘルパーアプリケーションを割り当てる

通常、1 種類の MIME タイプには 1 つのアプリケーションしか割り当てることができません。しかし、ファイルが application/octet-stream というタイプの場合、状況が異なってきます。ユーザは、Mozilla がこのタイプの設定を保存せず、MIME タイプによってヘルパーアプリケーションが見つからなかった場合、拡張子からそれを探すという事実を利用することができます。つまり、拡張子 ABC に対して (application/octet-stream 以外のタイプで) ヘルパーアプリケーションのエントリーを作成し、拡張子 XYZ に (こちらも application/octet-stream 以外のタイプで) 別のエントリーを作成すれば良いのです。

これで、Mozilla が application/octet-stream のタイプで拡張子 ABC が付いたファイルに出会った場合、最初のヘルパーアプリケーションが使われ、拡張子が XYZ だった場合は 2 番目のヘルパーアプリケーションが使われます。この設定は、メールの添付ファイルについても同じように機能します。

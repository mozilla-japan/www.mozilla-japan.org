タイトル
addSelectionListener を使った特権の昇格
重要度
最高
公開日
2006/06/01
報告者
moz_bug_r_a4
影響を受ける製品
Firefox
SeaMonkey
修正済みのバージョン
Firefox 1.5.0.4
SeaMonkey 1.0.2

概要

Web コンテンツから、Selection オブジェクトの nsISelectionPrivate インターフェイスにアクセスし、それを利用して SelectionListener を追加することが可能です。このリスナーは、ユーザがページ上で検索を行ったり、テキストをすべて選択した場合に呼び出され、本来は何か問題を引き起こすものではありません。しかし、MFSA 2006-31 の PAC サンドボックスや、MFSA 2006-37 のコンテンツ定義 DOM セッターを回避するのと同じように、任意のコード実行につながる特権コンテキストで通知関数が作成されてしまう問題を攻撃に利用できる方法が moz_bug_r_a4 氏によって発見されました。

回避策

修正版にアップグレードするまで JavaScript を無効にしてください。

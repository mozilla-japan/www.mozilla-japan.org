タイトル
メモリ破壊の形跡があるクラッシュ (rv:1.8.0.2)
重要度
最高
公開日
2006/04/13
報告者
複数
影響を受ける製品
Firefox
Thunderbird
Mozilla Suite
修正済みのバージョン
Firefox 1.5.0.2
Thunderbird 1.5.0.2
SeaMonkey 1.0.1

概要

Firefox 1.5.0.2 リリースの一環として、製品の安定性を向上させるため、いくつかのクラッシュバグを修正しました。中でも DHTML によって引き起こされるクラッシュの発見に努めました。これらのクラッシュの一部に、周到に準備すれば任意のコードを実行可能と思われるメモリ破壊の痕跡が見つかりました。

注意: Thunderbird は Firefox と同じブラウザエンジンを利用しており、JavaScript が有効になっている場合には影響を受けます。デフォルト設定では無効になっており、メールを表示する際は JavaScript を実行させないよう強く推奨します。

回避策

修正版にアップグレードしてください。Thunderbird や SeaMonkey のメールクライアントでは JavaScript を有効にしないでください。

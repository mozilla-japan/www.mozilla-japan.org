プライバシー保護機能を使う

インターネットとは、非常に多くのコンピュータにより構成される公共ネットワークで、全てのコンピュータが情報を共有しています。そのインターネットの上では、無数の接続を通して通信が行き交っており、全ての公共通信網と同様、盗聴・覗き見が可能です。

嬉しいことに、ブラウザにはセキュリティを守ってくれる機能が含まれています。このドキュメントにある情報に加えて、最新のセキュリティについての情報を Netscape の Security Center （訳注：日本語版） で得られます。ウィルス、安全な電子メール、安全なオンラインショッピングや銀行取引、そして子供のための安全な Web サーフィンなどの情報は、 Security Center にあるドキュメント、Understanding Security をご覧下さい。

下の一番最初のリンクをクリックして、インターネット上のプライバシーについての背景情報をご覧下さい。あなたのプライバシーを保護してくれる特定のブラウザ機能の使い方は、使いたい機能についてのトピックをクリックして下さい。

インターネット上のプライバシー

この文書では、インターネット上でのプライバシーについて解説します。またあなたのプライバシーを守るためにできることを紹介します。

プライバシーに関する事項についてのより詳しい解説は、 プライバシーチュートリアル をご覧下さい。

Web サイトは私の個人情報をどのように集めるのですか？

あなたの個人情報を集める方法としては二通りあります：

  • あなたがサイトからページを要求した場合、あなたの代わりにブラウザが一定量の情報をサイトに公開します。

  • もしあなたがオンラインフォームに書き込み、送信した場合は、その情報がサイト側に送信されます。

サイトがあなたの個人情報を保存する方法には二通りあります：

  • ブラウザが Web サイトからページを受け取っている間、サイトはブラウザに対して、あなたのコンピュータのハードディスクにあなたに関する少量の情報を保存するように要請します。ここで保存される情報は " cookie " と呼ばれます。 cookie を保存した （"設置した"） サイトは、あなたが次回そのサイトを訪れたときに情報を読み出せるようにブラウザに要請します。

  • （オンラインフォームに書き込んで） Web サイトに渡したあらゆる情報が、そのサイトのコンピュータに保存されます。

あなたが意図的に提供しない限り、あなたの電子メールアドレス、名前、その他個人情報は Web ページには 分かりません。あなたが主導権を握っているのです。あなたが許可しない限り、誰もあなたの個人情報を得ることはできません。

 

ブラウザはどういう情報を Web サイトに渡しているのですか？

リンクをクリックしたり URL を入力したりしてあなたがページをサイトから要求した場合には、少量の情報がサイト側に渡されます。こうした情報には、あなたの利用環境、あなたのインターネットアドレス （あなたの電子メールアドレスではありません）、そしてあなたがどのページから来たのか、が含まれます。

利用環境

サイトは、あなたのブラウザの種類やオペレーティングシステムといったあなたの利用環境についての情報を受け取ります。この情報によってサイトは、あなたが要求しているページをスクリーン上に最適の状態で表示できるようになります。例をあげると、サイトはあなたが mozilla ベースのブラウザのフランス語版を、 Windows 2000 コンピュータで利用している、という情報を得るのです。

インターネットアドレス

ブラウザはサイト側にインターネットアドレス （インターネットプロトコルまたは IP アドレスとしても知られています） を教えなければなりません。この情報でサイトは、あなたが要求しているページをどこに送ればいいのかが判るのです。あなたの IP アドレスが判からないと、サイトはページを表示できません。

あなたの IP アドレスは一時的なものか、固定 （静的） されたものかのどちらかです。

電話線に繋がれた標準的なモデムでインターネットに接続している場合、インターネットサービスプロバイダ （ISP） があなたがログオンする度に一時的な IP アドレスを割り当てています。あなたは、その IP アドレスをそのインターネットセッションの間だけ利用します。 ISP は沢山の IP アドレスを保有しており、ユーザーに対して無作為にアドレスを割り当てています。 Web サイトはその一時的な IP アドレスがどの ISP から来たものなのかは判りますが、その一時的な IP アドレスからはあなたの個人情報は一切わかりません。

重要： IP アドレスはあなたの電子メールアドレスでは ありません。電子メールアドレスは、ちょうど現実の世界での社会保障番号のように、サイバースペースであなたを特定することができるものです。一時的な IP アドレスは、電話をかける時にたまたま使用した公衆電話の電話番号のようなものと変わりありません。

DSL、ケーブルモデム、または光ケーブル接続を行っている場合、インターネットにつなぐ度に固定された IP アドレスを使っている可能性があります。

IP アドレスが一時的なものであれ、固定されたものであれ、訪れるサイトにその情報を渡したくないこともあるかもしれません。 IP アドレスの流出を防ぐ方法は、 匿名で Web をブラウズする をご覧下さい。

どのページから要求を送っているか

サイトは、あなたがページのリンクをクリックした時にそのサイトのどのページを読んでいたのか、という情報を得ます。これによりサイトは、どのサイトがあなたにそのページを教えたのかが判ります。またこの情報から、あなたがサイト内を動き回る際にも、あなたがそのサイトのどの部分にいたのかが判るようになるのです。

 

cookie とは何ですか？どのように使われるのでしょう？

cookie とはいくつかの Web サイトによって利用される少量の情報です。あなたがそうしたサイトを訪れた時、そのサイトはあなたのブラウザに、一つまたは複数の cookie をあなたのコンピュータのハードディスクに保存するように要請することがあります。

その後あなたがそのサイトに戻ったとき、あなたがブラウザはその cookie をサイトに送り返します。

標準の設定では、このやり取りはあなたの見えないところで行われ、サイトがいつ cookie を設定したのか、いつあなたのブラウザが Web サイトの cookie を送り返しているのかは分かりません。

なぜ cookie が必要なのですか?

あなたが前回訪れた際の情報をサイトが覚えておくように許可することで、あなたにとって便利になる場合もあります。例えば、もしあなたが長い文書を読んでいて 17 ページまで読み進んだとしましょう。あなたが次回に訪れたとき、サイトが 17 ページ目をすぐに表示してくれたら便利ですね。

cookie を拒否しなければならない理由は何ですか？

あるサイトが cookie を保存できる場合、あなたがそのサイトでしたこと全てを cookie に書き込み、更新して、追跡できるのです。つまりあなたのプロファイルを作成することができるのです。

そのサイトがそうして収集した情報で何をするかで、良くも悪くもなります。例えば、もし書籍販売のサイトがあなたが犬に興味を持っている事を知っていて、新しく犬に関する本が発売されたら教えてくれる、というのは良い面でしょう。しかし、もしそのサイトがそうした情報を地域の衛生局に販売する場合は悪い面といえます。そうした情報が正式な許可を得ていないまま飼い主になる可能性のある人を調査する一つの手がかりになり得るからです。

サイトは cookie の情報をどう利用するのですか？

Web サイトは cookie の情報を利用して、あなたに対してサイトの表示を最適化できます。例えば広告主は、 cookie の情報を利用してあなたに関心のある分野や購買情報に的を絞った広告を出せるのです。きちんとしたサイトは、収集した情報をどのように利用するのかを説明した個人情報を扱う際の指針 （プライバシーポリシー） を公開しています。

疑問を感じたら、 cookie が設置される前や （あなたの名前や電子メールアドレスといった） 情報を提出する前に、必ず Web サイトのプライバシーポリシーを確認してください。

外部 cookie に対処する

あるサイトが cookie を保存した場合、将来その cookie はそのサイトのみに返されます。ブラウザは、あるサイトが設置した cookie を他のサイトに渡すようなことはしません。サイトは自身が設置した cookie しか受け取ることができず、 インターネット全般をブラウズしている際の行動に関してではなく、その特定のサイト上でのあなたの行動に関してのみ追跡できます。

しかし、あなたが ABC.com というサイトを訪れている際に、 cookie が ABC.com によってではなく XYZ.com によって設置された場合を想定して下さい。 ABC.com は XYZ.com からの画像を表示することで簡単にそのような設定ができます。tつまり、あなたが ABC.com を訪れた際、ブラウザは画像を取り込みに XYZ.com まで回り道をして、その際に XYZ.com は cookie を設置できるのです。

XYZ.com が cookie 設置用の画像を多くのサイトに置いている場合は、そうしたサイトでのあなたの行動を全て含んだ cookie を作成することができるのです。 XYZ.com からの画像を掲載するサイトが増えれば増えるほど、より多くの情報を含んだあなたのプロファイルを作成することができるのです。

あなた自身が訪れていると考えているサイトによってではなく、他のサイトによって設置される cookie を 外部 cookie と呼びます。もし外部 cookie について心配であれば、外部 cookie は設置せず、普通の cookie のみを設置できるように許可をすることもできます。

あなたのブラウザが受け入れる cookie のタイプの設定は、 Cookie マネージャでできます。

 

私の個人情報を私の許可なく使われないようにすればどうすればいいのでしょう？

あなたの個人情報を守るための最も良い方法は、個人情報を他人に提供する際に注意深くなることです。インターネットは公共のネットワークです。もし誰かがあなたの名前、電話番号、住所、およびその他の情報を得た場合は、その情報がそれ以外の人にも共有される可能性があるのです。

Web 上であなたの名前、住所、電話番号を提供するということは、電話帳のリストに載るのと同じことです。実際、電話帳に掲載された場合、あなたの名前、住所、電話番号はオンライン登録簿や他の World Wide Web 上のデータベースにも掲載されるでしょう （People Finder や Yellow Page を覗いてみて下さい）。

Web サイトがあなたに関しての情報を求めてきた場合、先に進む前にそのサイトのプライバシーポリシーを確認してください。以下に Web サイトのプライバシーポリシーに関して確認する項目を挙げました：

  • そのサイトはどういう種類の個人情報を集めているのか？
  • そのサイトは集めた情報をどのように利用するつもりなのか？
  • そのサイトは本来の目的とは違う目的で、外部とその情報を共有するつもりなのか？
  • 私は情報の管理や更新のために、そのサイトが関して集めた一部または全てのデータにアクセスすることができるのか？
  • そのサイトはどのようにして保存している情報を得ようとする侵入者を防いでいるか？
  • 質問や問題が発生した場合、 Web サイトにどうコンタクトすればよいのか？

 

匿名で Web をブラウズする

あなたがあるサイトのページを見たいと要求した際、そのページの送信先が分かるように、ブラウザはそのサイトにあなたの インターネットアドレス （IP アドレス） を教えなければなりません。あなたの IP アドレスは一時的なものか、固定 （静的） なものかのどちらかです。

あなたの IP アドレスが一時的なものであれ固定的なものであれ、訪れようとしているサイトに対してその情報を渡したくない場合もあるでしょう。しかし、もしあなたのブラウザがこの情報を渡さないと、サイト側は要求されたページをどこに向けて送信すればいいのかが分からないのです。そのため、この情報だけはブラウザが送信するのを止められないものなのです。

もし本当に IP アドレスをサイト側に渡したくない場合は、信頼できる仲介サイトを利用しなければなりません。そうした仲介サイトに行き、あなたが欲しいページのサイト名を指定します。その仲介サイトは、独自の IP アドレスを使用してあなたの代わりにページを要求し、ページを受け取るとそのページをあなたに向けて転送します。そのページを提供したサイトからはあなたの IP アドレスを決してみることはできません。

このようなサービスを提供するサイトがいくつか存在します。使いなれた検索ページを使って、"anonymous" や "surfing" といったキーワードでページを探してみてください。

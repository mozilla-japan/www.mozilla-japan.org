SeaMonkey 1.0.6 インストール方法

以前のバージョンとの非互換 問題を避けるために、SeaMonkey インストーラはインストールディレクトリが空でない場合、削除するよう指示します。 インストールディレクトリにプロファイルまたはその他、個人的なファイルを置いている場合、NO と答えてください。 そうでなければファイルは削除されるでしょう。デフォルトの場所にプロファイルを置いているユーザは影響を受けません。

プロファイルをバックアップするヘルプはプロファイル FAQ を見てください。

Linux

シ ステム要件

  • 次のライブラリバージョン（または互換）は GTK1 ビルドで求められます：glibc 2.2.5, XFree86 3.3.x, GTK 1.2.x, Glib 1.2.x. Red Hat Linux 7.0, Debian 3.0、そして SuSE 7.3 （またはそれ以降）のインストールは動作するはずです。 推奨されたビルド（GTK2）は glibc 2.3.2, XFree86 4.3.x, GTK 2.x, glib 2.x, libstdc++-3.3 を求めます。
  • Intel pentium クラスプロセッサー（233 MHz またはより速いのを推奨）
  • 64 MB の RAM
  • 100MB のハードディスク空き容量（キャッシュサイズを減らせば、はっきりとスペースが空きます）

イ ンストール方法

注意： SeaMonkey 1.0.6 をインストールする前に、SeaMonkey の以前のバージョンをアンインストールすることをお勧めします。 これはブックマーク、履歴、cookie そしてプロファイルディレクトリに保存されているその他の情報を削除しません。

注意： ファイルをインストールディレクトリに書き込むには少なくとも一回は SeaMonkey を実行しなければなりません。 異なったユーザ（製品をインストールした人以外）が最初にプログラムを実行する場合、コアダンプが発生するでしょう。 (Bug 49345)

注意： これらの説明は bash シェルで使えます。bash を使っていないならば、bash シェルを使うかコマンドをそれに応じて調整してください。

SeaMonkey インストーラによるインストール

ダウンロードする SeaMonkey インストーラによる SeaMonkey インストールはこれらのステップに従ってください：

  1. seamonkey1.0.6 という名前のディレクトリを作って（mkdir seamonkey1.0.6）、そのディレクトリに移ってください（cd seamonkey1.0.6）。
  2. インストーラファイルをダウンロードするリンクをクリックしてください（seamonkey-1.0.6.en-US.linux-i686.installer.tar.gz）。
  3. SeaMonkey ディレクトリに移って（cd seamonkey1.0.6）、コマンドに従ってアーカイブを解凍してください：
    gunzip -dc sea*.tar.gz | tar -xvf -
    （seamonkey-installer という名前の SeaMonkey のサブディレクトリに置いてあります）
  4. seamonkey-installer ディレクトリに移って（cd seamonkey-installer）、 ./seamonkey-installer コマンドでインストーラを実行してください。
  5. SeaMonkey のインストールウィザードの説明に従ってください。
  6. SeaMonkey を実行するには、インストールしたディレクトリに移って ./seamonkey コマンドを実行してください。

tar.gz アーカイブによる手動インストール

ダウンロードする .tar.gz アーカイブファイルによる SeaMonkey のインストールをするには

  1. "seamonkey1.0.6" という名前のディレクトリを作って(mkdir seamonkey1.0.6)、そのディレクトリに移ってください (cd seamonkey1.0.6)。
  2. 非インストーラのseamonkey*.tar.gz ファイルを seamonkey1.0.6 ディレクトリの中にダウンロードするリンクをクリックしてください。
  3. seamonkey ディレクトリに移って（cd seamonkey1.0.6）、コマンドに従ってファイルを解凍してください。
    gunzip -dc sea*.tar.gz | tar -xvf - （これはあなたの seamonkey1.0.6 ディレクトリの下に "seamonkey" サブディレクトリを作ります。）
  4. seamonkey ディレクトリに移ってください（cd seamonkey）。
  5. ./seamonkey 実行スクリプトで seamonkey を実行してください。

マルチユーザ Linux インストール

Unix 上のマルチユーザに SeaMonkey をインストールするには、通常のインストールをし、その後、SeaMonkey ディレクトリに次のようなスクリプトを 作り、それを実行できるようにし（chmod u+x <scriptname>）、 mozilla ディレクトリに書き込み権限のあるユーザで実行します（例えば root）。 このプログラムを su[do] root として実行してはいけません。 (Bug 52653) 

#!/bin/sh
dist_bin=`dirname $0`
MOZILLA_FIVE_HOME=$dist_bin
LD_LIBRARY_PATH=$dist_bin
export MOZILLA_FIVE_HOME LD_LIBRARY_PATH
$dist_bin/regxpcom
$dist_bin/regchrome
touch $dist_bin/chrome/user-skins.rdf $dist_bin/chrome/user-locales.rdf

そ の時、それにアクセスする許可を持っているどんなユーザも SeaMonkey のインストールを実行できるようにすべきです。

SeaMonkey を GNOME パネルに追加する

  1. GNOME メインメニューボタンをクリックし、パネルメニューを開き、その後、パネルに追加サブメニューを開きランチャーを選びます。
  2. パネル上の SeaMonkey アイコンを右クリックし次のコマンドを入力します：

    ディレクトリ名/seamonkey

    ディレクトリ名はあなたの SeaMonkey をダウンロードしたディレクトリの名前です。 例えば SeaMonkey の提案するデフォルトのディレクトリは /usr/local/seamonkey です。
  3. アイコン名を入力し、望むならコメントを入力してください。
  4. アイコンボタンをクリックし、アイコンの場所を次のように入力してください：

    ディ レクトリ名/icons/mozicon50.xpm

    ディレクトリ名はあなたが SeaMonkey をインストールしたディレクトリです。デフォルトディレクトリは /usr/local/seamonkey/icons/mozicon50.xpm です。

RPM ビルドをインストールする

これはまだ SeaMonkey にはあてはまらないかもしれません。

Red Hat RPM ビルドはデフォルト Red Hat 7.x と Red Hat 8.x インストールで動作するよう作られています。 これらの RPM を SuSE 上で使うには gdk-pixbuf パッケージのインストールが求められます。 (Bug 85218)

アンインストール

  • SeaMonkey をインストールした seamonkey1.0.6 ディレクトリを削除してください。
  • プロファイル情報が 含まれている~/.mozilla ディレクトリを削除してください。

Macintosh

システム要件

  • Mac OS X 10.2 またはそれ以降
  • PowerPC プロセッサー (266 MHz またはより速いのを推奨)
  • 64 MB の RAM （256 MB 推奨）
  • 100MB のハードディスク空き容量（キャッシュサイズを減らせば、はっきりとスペースが空きます）

インストール 方法 (Mac OS X)

ダウンロードする SeaMonkey ディスクイメージをインストールするにはこれらのステップに従ってください：

    あなたのマシンに SeaMonkey をダウンロードするための seamonkey-1.0.6.en-US.mac.dmg リンクをクリックしてください。 デフォルトで、そのファイルはデスクトップにダウンロードされます。
  1. .dmg ファイルをダブルクリックしハードディスクに SeaMonkey アイコンをドラッグしてください。
  2. 私たちはそれをアプリケーションフォルダにコピーすることを勧めます。
  3. 今、ディスクイメージをイジェクトしてください。
  4. あなたが好むなら、いつでも容易にアクセスできるように dock にドラッグできます。

ア ンインストール方法

  1. /Users/& lt;あなたのユーザ名>/Library/Mozilla のアプリケーションファイルを削除してください （ただし Profiles サブフォルダは残してください）。
  2. /Users/yourusername/Library/Preferences から "Mozilla Registry"　ファイルを 削除してください。
  3. インストールした場所から SeaMonkey を削除してください（言い換えれば /Applications)。

Windows

シ ステム要件

  • Windows 95*, 98, 98SE, Windows Me, Windows NT 3.51, Windows NT 4.0, Windows 2000, Windows XP または Windows 2003
       * DCOM for Windows 95 を用いた場合
  • Intel pentium クラスプロセッサー （233 MHz またはより速いのを推奨）
  • 64 MB の RAM
  • 100MB のハードディスク空き容量（キャッシュサイズを減らせば、はっきりとスペースが空きます）

特定の ATI ビデオドライバーは SeaMonkey と互換性がなくクラッシュするかもしれません。クラッシュを経験したのであれば 最新の ATI ドライバーにアッ プグレードしてみてください。 (Bug 101055)

インス トール方法

注意： セットアッププログラムを実行する前に全てのプログラムを終了することを強くお勧めします。また一時的にウィルス検出ソフトを動作しないようにすべきで す。

SeaMonkey インストーラによるインストール

ダウンロードする SeaMonkey インストーラで SeaMonkey をインストールするにはこれらのステップに従ってください：

  1. あなたのマシンにファイルをダウンロードするには seamonkey-1.0.6.en-US.win32.installer.exe リンクをクリックしてください。
  2. ダウンロードしたファイルに行き、セットアッププログラムを始めるには、あなたのマシン上の SeaMonkey プログラムをダブルクリックしてください。
  3. セットアッププログラムの画面上の説明に従ってください。プログラムは最初、自動的に始まります。

.zip ファイルによる手動インストール

ダウンロードする .zip ファイルと手動インストールによる SeaMonkey のインストールをするには、これらのステップに従ってください：

  1. あなたのマシンに .zip ファイルをダウンロードするには seamonkey-1.0.6.en-US.win32.zip リンクをクリックしてください。
  2. ダウンロードしたファイルに行き、圧縮ファイルをダブルクリックしてください。 注意：このステップは WinZip の最近のバージョンをもうインストールしていて使い方を知っていると想定しています。 そうでないなら、 www.winzip.com. で WinZip とプログラムについての情報を得ることができます。
  3. .zip ファイルから C:\Program Files\SeaMonkey1.0.6 のようなディレクトリに解凍してください。
  4. SeaMonkey を始めるには、SeaMonkey を解凍したディレクトリに行き SeaMonkey.exe アイコンをダブルクリックしてください。

ア ンインストール方法

  • SeaMonkey インストーラを使ったなら、単に設定/コントロールパネル/プログラムの追加と削除に行き SeaMonkey 1.0.6 を選択してください。
  • zip ビルドを使ったなら単に SeaMonkey をインストールした SeaMonkey1.0.6 ディレクトリを削除してプロファイルディレクトリを 削除してください。

プロファイルの場所

以下のリストは代表的なプロファイルの場所です：

Windows

Windows 95、Windows 98、または Windows ME （ログインなしで）上で、プロファイルディレクトリは主に
    C:\Windows\Application Data\Mozilla\Profiles\[プロファイル名]\[ランダムな 記号列].slt\

Windows 95、Windows 98 またはログインありの Windows ME 上で、プロファイルディレクトリは主に
    C:\Windows\Profiles\[ロ グイン名]\Application Data\Mozilla\Profiles\[プロファイル名]\[ラ ンダムな記号列].slt\

Windows 2000 または Windows XP 上で、プロファイルディレクトリは主に
    C:\Documents and Settings\[ログイン名]\Application Data\Mozilla\Profiles\[プロファイル名]\[ランダムな記号列].slt\

注意：Windows 2000 または Windows XP で、Document and Settings ディレクトリは二重に隠されています。アクセスするには、Windows エクスプローラでシステムディレクトリ・ファイルの表示と隠しディレクトリ・ファイルを 有効にする必要があります。

Mac OS X

プロファイルの場所は主に：
    ~/Library/Mozilla/Profiles/[プロファイル名]/[ランダムな記号列].slt/

Linux

プロファイルの場所は主に：
    ~/.mozilla/[プロファイル名]/[ランダムな記号列].slt/

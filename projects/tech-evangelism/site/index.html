Mozilla サイト・エバンジェリズム

Mozilla、Mozilla Firefox、Camino、Netscape 7 などをお使いの場合

あなたがもし、お気に入りの Gecko ブラウザをサポートしていないような問題を抱えたサイトを見つけたためにこのページを訪れた場合は、問題のあるサイトを見つけたら のページをご覧になり、自分自身でサイトに連絡を取ることで貢献してください。

Web 開発や品質保証の経験をお持ちの場合

既存のエバンジェリズム・バグの選別に参加したり、有名サイトに接触を試みて、Mozilla その他 Gecko をベースとしたブラウザ、さらには標準に準拠したブラウザをサポートする必要があるという情報を伝えることで、Web 上で Mozilla のサポートレベルを向上させる協力ができます。

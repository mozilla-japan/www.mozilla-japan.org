Mozilla 1.0 へのガイド


イントロダクション

Mozilla 1.0 は Mozilla ソフトウェアの初めてのメジャーバージョン リリースです。クロスプラットフォームのツールキットと同様に、最新のインターネットスタンダードに基づいた、完全に一人前になったブラウザスイート Mozilla 1.0 は、開発コミュニティーにターゲットを向け、インターネットベースのアプリケーション制作を可能にしています。これはオープンソース環境で開発され、インターネット上の何千人ものプログラマーたちのクリエイティブパワーを結集し、彼らの最高の機能強化を一体化して作られました。

幅広いアプリケーションに利用することのできるコンポーネントのセットは、すべてオープンソース・無料で、Mozilla 1.0 のエンドユーザアプリケーションで使われてテストされたもので、それらの提供によって Mozilla 1.0 は開発者がクロスプラットフォーム、主にネットワーク分野のためのアプリケーション制作を可能にしました。

Mozilla 1.0 はまた、そのアーキテクチャが Web コンテンツ制作に使われているのと同じ技術に基礎をおいた制作を可能にすることから、複合アプリケーションの記述ができる開発者の範囲を広げます。

これらのページは、利用者、評論家、また OEM にとっても Mozilla 1.0 への入門ガイドとしてデザインされていて、FAQ ページ と並んで、できる限り凝縮した形で Mozilla 1.0 についての核となる情報を提供するようになっています。

注意

Mozilla 1.0 と過去のリリース（Netscape 6.x / 7.0 PR1 のような、いくつかのディストリビューションも含む）との違いのため、それらとのプロファイルの共有は避けるべきです。このバージョンは Mozilla 1.0 とアドレス指定されていて、Mozilla 1.0 をベースとしたブラウザとのプロファイルの共有は、ユーザに影響することはありません。

以前の Mozilla や Mozilla ベースのインストレーションからのプロファイルを持っている場合、Mozilla を最初に起動したときに、どのプロファイルを使うか訪ねられるでしょう。この時点で新しいプロファイルを作成すると、新しいインストレーションのためのデータを分けて保存し、互換性のないプロファイルによる潜在的な問題を回避できます。プロファイルに関する詳しい情報は FAQ でご覧いただけます。

Mozilla を Windows にインストールしたときは、特に遅いプロセッサのコンピュータを使っている場合を除いて、高速起動を有効にするといいでしょう。高速起動は、Mozilla に一部のコンポーネントをブートタイムにプレロードさせることで、より速く起動できます。また、システムトレイにある Mozilla アイコンを右クリックすることで、どのコンポーネントを起動させるか選ぶこともできます。

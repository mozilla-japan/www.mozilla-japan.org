ショートカットの使い方

このドキュメントでは、次のような形式でショートカットを表記します。

例:

コマンドWindowsMacLinux
コピーCtrl+CCmd+CCtrl+C

コマンドを実行するには、書かれているボタンを同時に押してください。たとえば、ウィンドウ内でコピーをするには、Ctrl と C を同時に押してください。

ヒント: プラス記号 (+) は、その左右に書かれている 2 つのキーを同時に押すという意味です。さらにプラス記号がある場合 (例: Ctrl+Shift+C) には、書かれているすべてのキーを同時に押すということです。

略語:

  • Ctrl = Control キー
  • Cmd = Macintosh キーボードの Command キー

注意: 一部のキーボードショートカットは、カーソルの位置 (フォーカス) によって異なる働きをします。たとえば、Windows では、Web ページを見ているときに Home を押すと Mozilla はそのページのトップに移動しますが、カーソルがテキストフィールドにある場合は、カーソルがテキストフィールドの先頭に移動します。

(訳注: 同じプラットフォームでも、バージョンや設定によっては、ここに書かれた動作をしないものもあります。プラットフォーム側で同じキーボードショートカットが使われていて、それらが優先される場合もあります。また、Mozilla バージョンアップによって追加・変更される場合もあります。実際に使ってみて、どのように機能するか試してみてください)

(訳注: Mozilla Firebird / Mozilla Thunderbird のキーボードショートカットは若干異なります)

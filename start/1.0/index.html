mozilla.org の 使命 は、ソフトウェア開発者が Web アプリケーションの制作に利用できるオープンソースのコードを作ることです。 私たちは、あなたのような方々がテストに協力していただけるように、Mozilla をダウンロードして使えるようにしています。 テスターになるのは難しいことではありません。 ただ毎日の仕事のために Mozilla を使い、私たちが尋ねたときにアップグレードし、プロンプトが表示されたらクラッシュの報告を送ってくださるだけでいいのです。 そうしていただくことが、私たちが Mozilla の安定性を評価し、向上させる手助けになるのです。

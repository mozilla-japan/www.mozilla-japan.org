ソフトウェアアップデート: 1.0 RC2/最終版に向けたアップデート

これは、Firefox 1.0 RC2 のテストに協力しようと考えている方々のためだけに書かれたドキュメントです。 Firefox の様々なバージョンからアップグレードする際に、アップデート機能が正しく動作するかどうかテストを行う方法を解説します。

Firefox Preview Release をお使いの方

Firefox Preview Release をお使いで、Firefox 1.0 RC2 にアップグレードしたい方は、次の手順に従ってください。

  1. Firefox を起動して、ロケーションバー に about:config と入力して Enter キーを押します。
  2. パネルが表示されたら、Filter ボックスに app.update.url と入力します。
  3. 表示された行をダブルクリックします。
  4. ダイアログボックスが表示されますので、テキストボックス内の値を https://addons.mozilla.org/update/firefox/en-US-1.0-rc.rdf に変更して OK をクリックします。
  5. Tools メニューから Options を開き、Advanced パネルを開きます。Software Update セクションが見えるまでスクロールし、Check Now をクリックします。
  6. Update Wizard が表示されますので、アップデートを確認します。確認が終わったら、利用可能なアップデートが表示されます。"Firefox 1.0 RC2" オプションを選択して Next をクリックします。
  7. ソフトウェアのダウンロードとインストールを許可し、ブラウザを再起動します。これで Firefox 1.0 RC2 が使えるはずです。

Firefox 1.0 RC1/RC2 をお使いの方

Firefox 1.0 RC1/RC2 をお使いで、Firefox 最終版に (リリースされた時点で) アップグレードしたい方は、次の手順に従ってください。

  1. Firefox を起動して、ロケーションバー に about:config と入力して Enter キーを押します。
  2. パネルが表示されたら、Filter ボックスに app.update.url と入力します。
  3. 表示された行をダブルクリックします。
  4. ダイアログボックスが表示されますので、テキストボックス内の値を https://addons.mozilla.org/update/firefox/en-US-1.0-rc.rdf に変更して OK をクリックします。
  5. Filter ボックスに app.version と入力します。
  6. 表示された行をダブルクリックします。
  7. ダイアログボックスが表示されますので、テキストボックス内の値を 1.0 から 0.10 に変更して OK をクリックします。これは、アップデートシステムに対して、あなたが以前のバージョンの Firefox を使っていると思い込ませるためのものです。
  8. Tools メニューから Options を開き、Advanced パネルを開きます。Software Update セクションが見えるまでスクロールし、Check Now をクリックします。
  9. Update Wizard が表示されますので、アップデートを確認します。確認が終わったら、利用可能なアップデートが表示されます。"Firefox 1.0 RC1/Firefox 1.0" オプションを選択して Next をクリックします。
  10. ソフトウェアのダウンロードとインストールを許可し、ブラウザを再起動します。これで Firefox 1.0 RC2/最終版が使えるはずです。

アップグレード手順のインストール段階まで進んだら、ブラウザを終了しないでください!

アップグレードの途中で何か問題に遭遇したら、FTP からフルインストーラをダウンロードして、ブラウザを再インストールすれば復旧できます。問題を発見したら Bugzilla に報告してください (プロダクト: Firefox、コンポーネント: Software Update)

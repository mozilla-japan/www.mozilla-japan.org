Mozilla が基盤にしていて、Mozilla のユーザ・インタフェースが使っているもっとも重要な技術への短いリストです。

World Wide Web Consortium(W3C) 標準：
XML 拡張性のあるマークアップ言語（eXtensible Markup Language）
RDF リソース記述フレームワーク（Resource Description Framework）
XSL 拡張性のあるスタイルシート言語（eXtensible Stylesheet Language）
CSS カスケーディング・スタイル・シート（Cascading Style Sheets）
DOM ドキュメント・オブジェクト・モデル（Document Object Model）
mozilla.org 技術：
XUL XML ユーザ・インタフェース言語（XML User Interface Language）
XPI クロス・プラットフォーム・インストーラ（Cross Platform Installer）
XBL 拡張性のある結びつけ言語（eXtensible Binding Language）
Netscape Java Script
Unicode コンソーシアム
ECMA スクリプト, ECMA specification 262

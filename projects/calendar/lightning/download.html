Lightning 0.9

Lightning は Sunbird カレンダーを人気の電子メールクライアント Mozilla Thunderbird に実装します。

ダウンロードの手順

  1. お使いのオペレーティングシステムに合ったリンク先に行き、addons.mozilla.org ウェブサイトの手順に従ってください。
  2. Thunderbird を開き、[ツール] メニューから [アドオン] を選択します。
  3. [インストール] ボタンをクリックして、ダウンロードしたファイルを選択し、[OK] をクリックします。
  4. インストールが終了したら Thunderbird を再起動をしてください。
  5. ソフトウェアのご使用前には必ず Lightning リリースノート をお読みください。

Lightning をインストールしてから初めて Thunderbird を起動する際には、必ず設定パネルまたはオプションダイアログを開いて正しい時間帯を手動で設定してください。そしてもう一度 Thunderbird を再起動してください。こうしないと作成したイベントが誤った時間帯に設定されてしまいます。

その他のシステム

Lightning ナイトリービルド (開発ビルド)

Lightning のナイトリービルドが開発者・テスタ向けに用意されています。

  • Mozilla 1.8 branch ビルド (Thunderbird 2.0 用)
    • Windows
    • Linux (i686)
    • Mac OS X (PowerPC)
  • Trunk ビルド (Thunderbird 3.0 用)
    • Windows
    • Linux (i686)
    • Mac OS X (PowerPC)

開発者向けナイトリービルドを利用するということはそれに伴うリスクもあるということを念頭においてください。こうしたビルドを業務用データでは使 用しないでください。データロスや恐ろしい事故が起こりかねません。これは脅しではありませんよ。

フィードアイコンガイドライン

Mozilla Firefox は、次のような新しい「フィードアイコン」 (別名「RSS アイコン」) を以前のバージョンで採用し、使用してきました。これは RSS や Atom などのオープンな Web シンジケーション フォーマットと関連して使用されるものです。

Mozilla Foundation は、Mozilla Foundation 以外の方がこれらのアイコンを自由にご使用いただけるようにしました。このドキュメントは、アイコンの使用に関する推奨ガイドラインを提供しており、以下の皆様に向けてのサービスとして Mozilla Foundation によって発行および管理されています。

  • 独自の製品およびサービスと関連してフィードアイコンを使用したいと考えている個人および組織のコミュニティ
  • これらの製品およびサービスを使用し、オープン Web シンジケーションフォーマットの使用を示す際にフィードアイコンの標準的な意味に依存する一般の方々

フィードアイコンの使用について

オープン Web シンジケーションフォーマットを使用した Web フィード と関連して一般的に使用する場合、商標のライセンス契約または同等の法的契約を締結する必要はなく、フィードアイコンを自由に使用できます。これらのガイドラインには法的拘束力がないことにご留意ください。

ただし、以下の場合はその限りではありません。

  • Web フィードの購読または操作を目的とした、ソフトウェアまたはハードウェアを作成および配布する。
  • 同等の目的のためにオンラインサービスを提供する。または
  • 関連製品またはサービス (コンサルティングまたはシステムインテグレーションサービスなど) を提供または供給する。

ならびにこのような製品またはサービスに関連してこのアイコンを使用したい場合には、あなた (またはあなたが認定した代理人) がこれらのガイドラインに従うことを確約する旨を公式に述べることをお願いしています。

以下のセクションでは、推奨ガイドラインを提供します。詳細は 添付の FAQ をご覧ください。

使用ガイドライン

フィードアイコンは、特に広範に使用されている RSS 1.0、RSS 2.0、ならびに Atom 1.0 の各フォーマットをはじめとするオープンフォーマットの Web シンジケーションを介して提供される情報の存在を示すために使用されなければなりません。

「Web シンジケーション」とは、通常は HTTP プロトコルを使用して転送される XML ベースの文書フォーマットを使用した、Web サイトまたはその他のソースのコンテンツ項目 (ニュースまたはブログへの投稿またはその概要) をその他のサイトまたはアプリケーションが使用できるようにするプロセスを意味します。

「オープンフォーマット」とは、以下のフォーマットを意味します。

  • 発行された仕様によって定義されている。
  • 使用に関する法的制限、特にフリー・オープンソースソフトウェアによるそのフォーマットの実装を妨げる制限がない。
  • オープンなプロセスによって開発・メンテナンスされている。

フィードアイコンは Web シンジケーションを介しても使用可能となる情報 (ブログの投稿、ニュース記事など) を含む Web ページで、Web フィードの URL にリンクするアイコンを付けて使用するのが正規の使用方法です。

フィードアイコンは、オープン Web シンジケーションフォーマットの情報の発見、検索、購読、作成、または操作に関わるその他の状況で使用することができます。たとえば、オープン Web シンジケーションフォーマットと関連したアプリケーションまたはオンラインサービスの特定の機能を起動するメニューアイコンまたはツールバーアイコンとしてアイコンを使用することができます。　

フィードアイコン (またはそれに類似したアイコン) は、以下の状況で使用してはなりません。

  • オープン Web シンジケーションフォーマットで使用可能となっていない情報に関連して使用しないでください。
  • アプリケーションアイコン、Web サイトのロゴ (ファビコン を含む) として、またはそれらの一部に組み込んで使用しないでください。またはフィードアイコンが特定のアプリケーションまたは Web サイトに排他的または主として関連していることを第三者から見ても暗示するその他のいかなる方法で使用しないでください。
  • 特定の個人、組織、製品、またはサービスに関連した商標またはサービスマークとして、またはそれらの一部に組み込んで使用しないでください。またはフィードアイコンが特定の個人、組織、製品、またはサービスに排他的または主として関連していることを第三者から見ても暗示するその他のいかなる方法で使用しないでください。

これらのガイドラインは、ブログの投稿、ニュース、雑誌の記事、書籍、テレビ番組、映画、またはオープン Web シンジケーションフォーマットまたはこれらのフォーマットをサポートするアプリケーションや Web サイトについて検討するその他の著作物に関連してアイコンを使用することを含み、フィードアイコンの公正な使用を妨げることを決して意図していません。

視覚的なガイドライン

フィードアイコンは、無地の明るいまたは暗い背景上にアイコン全体を表示しなければなりません (中間の色調または極めて濃い色の背景は避けてください)。フィードアイコンの一部分のみを表示したり、アイコンの上に他の要素を重ねて配置してはなりません。

フィードアイコンは、対象とするユーザが読みやすく、また関連したテキスト要素のサイズと適合するサイズで表示しなければなりません。

フィードアイコンは、色を再現することが不可能な状況下では、白黒で表示することができます。フィードアイコンは、背景色および前景色とのコントラストが適切である場合に限り、アイコンの色を反転して使用することができます (たとえば、オレンジの代わりに白、白の代わりにオレンジを使用)。

ユーザインターフェースの要素として使用する場合 (たとえばツールバーボタンとして)、フィードアイコンは関連するユーザインターフェースの要素および適用されるユーザインターフェースの規格 (基礎をなすオペレーティングシステム、ウィンドウシステムなど) と調和して表示しなければなりません。

フィードアイコンは、上記のガイドラインに準拠しない修正された形態、またはアイコンを視覚的に傷つけるような方法で表示してはなりません。特に、アイコンは以下のように表示してはなりません。

  • 向きを変えて使用しないでください (つまり、通常の向きと相対して回転したり、反転させないでください)。
  • アイコンの「丸みのある四角形」の要素を円、三角、またはオリジナルの背景で使用した形状以外のいかなるその他の形状で置き換えて使用しないでください。
  • アイコンの「丸みのある四角形」の要素の角を丸みではなく鋭角にして使用しないでください。

サイズと色に関わる上記のガイドラインは、視覚に障害がある人が使用するために設計した支援技術によってフィードアイコンが表示される方法を制限する意図はありません (このような技術としては、画面のコンテンツを拡大して表示する、画面の色、コントラスト、明るさを変更するソフトウェアや Web サイトの代替スタイルシートなどが挙げられます)。

これらのガイドラインは、ガイドラインの全体的な精神に準拠するフィードアイコンのその他の使用を妨げることは意図していません。このような事例としては、必要に応じて (サイトのテーマと適合させるなどのために) アイコンの色を変更する場合、またはスタイルおよび性質が適合しているその他のアイコンと組み合わせてアイコンを使用する場合が挙げられます (たとえば、提供されているフィードの種類を識別する)。ただし、ユーザに一貫したエクスペリエンスを提供するために、フィードリーダーおよび関連製品およびサービスの提供者がアイコンの代替の表示の使用を最小限に抑えることを推奨します。

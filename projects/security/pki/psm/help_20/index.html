PSM 2.0 ヘルプ: 状況と進行中の作業

ニュースグループ: netscape.public.mozilla.crypto
作成者: Sean Cotter

状況

関連する情報

状況

このページからリンクされているファイルは Netscape6 かそれ以降のバージョンで表示されるようになるでしょう。

進行中の作業

ヘルプを見直すのに一番良いのは、(訳注: ブラウザを操作している)状況の中で、現在のビルドを使って見直すことです。この内容は、論点の参照のため、現在行っている作業のスナップショットを下に提供します。

左の三角形 (訳注: 原文 twisties ) が Mozilla ヘルプビューアのナビゲーションフレームを開くとき、以下の見出しが現れます。対応するセクションを見るには、下のリンクをクリックしてください。Mozilla ヘルプビューアでのフォーマットとは違います。

ダイアログボックスヘルプセクションにある、ほとんどの見出しは、UI のヘルプボタンに対応します。対象となる文字列は PSM 2.0 UI 要素の対象とテキスト に一覧してあります。

  • プライバシー保護機能を使う
    • インターネット上のプライバシー (背景と一般的なガイドライン)
      • Web サイトは私の個人情報をどのように集めるのですか？
      • ブラウザはどういう情報を Web サイトに渡しているのですか？
      • cookie とは何ですか？どのように使われるのでしょう？
      • 私の個人情報が許可なく使われないようにするにはどうしたらいいのでしょう？
      • 匿名でブラウズする
    • Cookie マネージャを使う (手続き型ヘルプ)
    • パスワードマネージャを使う (手続き型ヘルプ)
    • フォームマネージャを使う (手続き型ヘルプ)
    • 保存されている機密情報を暗号化する (手続き型ヘルプ)

  • 証明書を使う (手続き型ヘルプ)
    • 自分の証明書を手に入れる
    • Web ページのセキュリティをチェックする
    • 証明書を管理する
    • スマートカードと他のセキュリティデバイスを管理する
    • SSL の警告と設定を管理する
    • 検証の設定を管理する

  • ダイアログボックスヘルプ

    • 設定 (状況依存型ヘルプ)
      • プライバシーとセキュリティ
        • Web パスワード
        • マスターパスワード
        • SSL
        • 証明書
        • 検証

    • SSL のページ情報 (状況依存型ヘルプ)
      • Web サイトを特定できません。検証されていません／接続は暗号化されていません
      • Web サイトを特定しました。検証されています／接続は暗号化されています
      • Web サイトを特定しましたが、条件つきで検証されています／接続は暗号化されています
      • Web サイトを特定しました。検証されています／接続は暗号化されていません
      • Web サイトを特定しましたが、条件つきで検証されています／接続は暗号化されていません

    • 証明書マネージャ (状況依存型ヘルプ)
      • あなたの証明書
        • 証明書をバックアップするパスワードを選択する
        • あなたの証明書を削除する
      • Web サイトの証明書
        • Web サイトの証明書の設定を編集する
        • Web サイトの証明書を削除する
      • 認証局の証明書
        • 認証局の証明書の設定を編集する
        • 認証局の証明書を削除する
      • セキュリティデバイスマネージャ

    • 証明書の情報と決定 (状況依存型ヘルプ)
      • 証明書の詳細
      • セキュリティデバイスを選択する
      • 証明書のバックアップ
      • 身元確認の要求
      • アクセスできない身元証明
      • 新しい認証局
      • Web サイトの証明書
        • 新しい Web サイトの証明書
        • 期限切れになった Web サイトの証明書
        • Web サイトの証明書はまだ検証されていません
        • 予期しない証明書名

その他の情報

  • より古いバーションの Calendar を試すには、 http://ftp.mozilla.org/pub/mozilla.org/calendar/xpi/ へどうぞ。
  • SVG が利用可能な Mozilla ビルドならどれでも、標準で Calendar が利用できるようになっています。
  • コマンドラインから Calendar を利用するには、 "-calendar" 引数をつけて起動してください。
  • 注意！：サードパーティー製のテーマをインストールすると、 Calendar の実行時に問題が発生するかもしれません。
  • Mozilla Application Suite 1.5 及び 1.6 の 2003/09/26 より後のビルドでは Calendar に対応しなくなりましたが、 Eric Belhaire のブログにある小さなアドオンを組み込むことで、それらのビルドを以前と同じように Calendar に対応させることができるようになります。
    【訳注： Mozilla Application Suite 1.5 及び 1.6 に対応した Calendar はこちらのページにあります。】
  • Bjarne D Mathiesen のページでも、サードパーティー製の Mac 向けビルドが公開されています。

インストール方法

Mozilla Calendar を Mozilla または Mozilla Firefox にインストールする

あなたが使用している Mozilla または Mozilla Firefox のバージョンに適合した「インストール」のリンクをクリックしてください。 XPI インストーラが起動し、自動的にインストールされます。

Mozilla Calendar を Mozilla Thunderbird にインストールする

Thunderbird にこの Calendar 拡張機能をインストールする場合、以下のステップを踏んでください：

  1. 普段使用しているブラウザで、あなたが使用している Mozilla Thunderbird のバージョンに適合した「インストール」のリンクを右クリックして「リンク先を名前を付けて保存」を選び、適当なフォルダにインストール用 XPI ファイルをダウンロードしてください。
  2. Mozilla Thunderbird を起動し、メニューバーから[ツール] -> [拡張機能]メニューを選択してください。
    【訳注： Thunderbird 1.0 日本語版 の場合。それ以外のバージョンをご使用の方は、適宜読み替えてください。】
  3. [拡張機能]パネルが開きますので、[インストール]ボタンをクリックしてください。
  4. ファイルの選択画面が開きますので、先ほどダウンロードされた XPI ファイルを選択してください。ファイルが読み込まれ、自動的にインストールが行われます。
  5. 【訳注： その後、 Thunderbird を再起動すれば、 Calendar のインストールが完了します。】

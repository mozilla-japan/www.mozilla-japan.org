SeaMonkey 1.1.4 インストール方法

以前のバージョンとの非互換問題を避けるために、そのインストールディレクトリが空でなければ、SeaMonkey インストーラは、それを削除するよう促すでしょう。インストールディレクトリに、プロファイルや他の個人的なファイルを置いている場合は、いいえと答えてください。そうでなければ、ファイルは削除されるでしょう。プロファイルをデフォルトの場所に置いているユーザは影響ありません。

プロファイルをバックアップするヘルプについては プロファイル FAQ を見てください。

このドキュメントは、システム要件、互換性情報、そして SeaMonkey 1.1.4 のインストール方法を述べています。

Linux

システム要件

  • 以下のライブラリバージョン（または互換性のある）は GTK 1 ビルドで必要とされます：glibc 2.2.5, XFree86 3.3.x, GTK 1.2.x, Glib 1.2.x. Red Hat Linux 7.0, Debian 3.0, そして SuSE 7.3 （またはそれ以降）のインストールは動作するはずです。推奨されるビルド (GTK2) は glibc 2.3.2, XFree86 4.3.x, GTK 2.x, glib 2.x, libstdc++-3.3 を必要とします。
  • Intel pentium クラスプロセッサー（233 MHz またはより速いのが推奨される）
  • 64 MB の RAM
  • 100 MB ハードディスク空きスペース（キャッシュサイズを減らすと、必要なスペースが著しく少なくて済みます）

インストール方法

注意： SeaMonkey 1.1.4 をインストールする前に、SeaMonkey の以前のバージョンをアンインストールすることをお勧めします。インストーラは、ブックマーク、履歴、クッキー、そしてプロファイルディレクトリに保管されているその他の情報といったものを削除しないでしょう。

注意： 自動登録と compreg.dat ファイルにインストールディレクトリを書き込むのを完了するために、少なくとも 1 回は SeaMonkey を実行しなければなりません。違うユーザ（製品をインストールした人以外）が最初にプログラムを実行すると、コアダンプが発生するでしょう。(Bug 49345)

注意： これらの方法は bash シェルを利用します。bash を使っていないならば、bash シェルを使うか、状況に応じてコマンドを調整してください。

SeaMonkey インストーラによるインストール

SeaMonkey インストーラをダウンロードして SeaMonkey をインストールするには、これらのステップに従ってください：

  1. seamonkey1.1.4 という名のディレクトリを作り (mkdir seamonkey1.1.4)、そのディレクトリに移動してください (cd seamonkey1.1.4) 。
  2. インストールファイルをダウンロードするためのリンク（seamonkey-1.1.4.en-US.linux-i686.installer.tar.gz と名付けられている）をクリックしてください。
  3. SeaMonkey ディレクトリに移動し (cd seamonkey1.1.4) 、以下のコマンドでアーカイブを解凍してください：
    gunzip -dc sea*.tar.gz | tar -xvf -
    （これはインストーラを seamonkey-installer という名の SeaMonkey のサブディレクトリに置きます。）
  4. seamonkey-installer ディレクトリへ移動し (cd seamonkey-installer) 、./seamonkey-installer コマンドでインストーラを実行してください。
  5. SeaMonkey をインストールするためのインストールウィザードの説明に従ってください。
  6. SeaMonkey を起動するには、インストールしたディレクトリへ移動して ./seamonkey コマンドで実行してください。

tar.gz アーカイブによる手動インストール

.tar.gz アーカイブファイルをダウンロードして SeaMonkey をインストールするには：

  1. seamonkey1.1.4 という名のディレクトリを作り (mkdir seamonkey1.1.4)、そのディレクトリに移動してください (cd seamonkey1.1.4) 。
  2. 非インストーラ seamonkey*.tar.gz ファイルを seamonkey 1.1.4 ディレクトリの中にダウンロードするためのリンクをクリックしてください。
  3. SeaMonkey ディレクトリへ移動し (cd seamonkey1.1.4) 、以下のコマンドで、そのファイルを解凍してください：
    gunzip -dc sea*.tar.gz | tar -xvf - （これは seamonkey 1.1.4 ディレクトリの下に "seamonkey" サブディレクトリを生成します。）
  4. seamonkey ディレクトリへ移動してください (cd seamonkey)。
  5. ./seamonkey 実行スクリプトで SeaMonkey を実行してください。

マルチユーザ Linux インストール

Unix 上のマルチユーザのために SeaMonkey をインストールするには、通常通りインストールし、その後、SeaMonkey ディレクトリに、以下のスクリプトを作り、実行できるようにし (chmod u+x <スクリプト名>)、mozilla ディレクトリに対するユーザ権限を持ったユーザになってそれを実行してください（例えば root）。このプログラムを su[do] root で実行してはいけません。 (Bug 52653)

#!/bin/sh
dist_bin=`dirname $0`
MOZILLA_FIVE_HOME=$dist_bin
LD_LIBRARY_PATH=$dist_bin
export MOZILLA_FIVE_HOME LD_LIBRARY_PATH
$dist_bin/regxpcom
$dist_bin/regchrome
touch $dist_bin/chrome/user-skins.rdf $dist_bin/chrome/user-locales.rdf

その後、あなたは SeaMonkey にアクセスできる権限を持ったユーザとして、そのインストールを実行できるはずです。

SeaMonkey を GNOME パネルへ追加する

  1. GNOME メインメニューボタンをクリックし、パネルメニューを開き、その後、「パネルに追加」サブメニューを開き、ランチャーを選択してください。
  2. パネル上の SeaMonkey アイコンを右クリックし、以下のコマンドを入力してください：

    directory_name/seamonkey

    directory_name はあなたが SeaMonkey をダウンロードしたディレクトリの名前です。例えば、SeaMonkey が勧めるデフォルトのディレクトリは /usr/local/seamonkey です。
  3. アイコン名を入力し、あなたが望むならコメントを入力してください。
  4. アイコンボタンをクリックし、アイコンの場所として以下を入力してください：

    directory_name/icons/mozicon50.xpm

    directory_name はあなたが SeaMonkey をインストールしたディレクトリです。例えば、デフォルトのディレクトリは /usr/local/seamonkey/icons/mozicon50.xpm です。

RPM ビルドをインストールする

これはまだ SeaMonkey に適用できません。

Red Hat RPM ビルドは、デフォルトの Red Hat 7.x と Red Hat 8.x インストールで動作するよう作られています。これらの RPM を SuSE で使うには、gdk-pixbuf パッケージのインストールが必要です。(Bug 85218)

アンインストール方法

  • SeaMonkey をインストールした seamonkey1.1.4 ディレクトリを削除してください。
  • プロファイル情報 を含んでいる ~/.mozilla ディレクトリを削除してください。

Macintosh

システム要件

  • Mac OS X 10.2 またはそれ以降
  • PowerPC プロセッサー（266 MHz またはより速いのが推奨されている）
  • 64 MB RAM （256 MB が推奨されている）
  • 100 MB ハードディスク空きスペース（キャッシュサイズを減らすと、必要なスペースが著しく少なくて済みます）

インストール方法 (Mac OS X)

SeaMonkey ディスクイメージをダウンロードして SeaMonkey をインストールするには、これらのステップに従ってください：

  1. あなたのマシンにそれをダウンロードするための seamonkey-1.1.4.en-US.mac.dmg リンクをクリックしてください。デフォルトで、そのファイルはあなたのデスクトップへダウンロードされます。
  2. .dmg をダブルクリックし、SeaMonkey アイコンをハードディスクへドラッグしてください。
  3. それをアプリケーションフォルダーへコピーすることを、私たちはお奨めします。
  4. それから、ディスクイメージを取り出してください。
  5. お好みで、いつでも簡単にアクセスできるように、SeaMonkey をあなたの dock へドラッグできます。

アンインストール方法

  1. /Users/<あなたのユーザ名>/Library/Mozilla のアプリケーションファイルを削除してください（ただし Profiles をサブフォルダに残しておいてください）。
  2. /Users/あなたのユーザ名/Library/Preferences から "Mozilla Registry" ファイルを削除してください。
  3. インストールした場所から SeaMonkey を削除してください。（すなわち /Applications）

Windows

システム要件

  • Windows 95*, 98, 98SE, Windows Me, Windows NT 3.51, Windows NT 4.0, Windows 2000, Windows XP または Windows 2003
       *DCOM for Windows 95 を用いた場合
  • Intel pentium クラスプロセッサー（233 MHz またはより速いのが推奨される）
  • 64 MB RAM
  • 100 MB ハードディスク空きスペース（キャッシュサイズを減らすと、必要なスペースが著しく少なくて済みます）

特定の ATI ビデオドライバは SeaMonkey と互換性がなく、クラッシュを引き起こすかもしれません。クラッシュを経験しているならば、最新の ATI ドライバへ アップグレード してみてください。(Bug 101055)

インストール方法

注意： セットアッププログラムを実行する前に、全てのプログラムを終了することを強くお奨めします。また、ウィルス検知ソフトウェアを一時的に無効にすべきです。

SeaMonkey インストーラによるインストール

SeaMonkey インストーラをダウンロードして SeaMonkey をインストールするには、これらのステップに従ってください：

  1. あなたのマシンにファイルをダウンロードするには seamonkey-1.1.4.en-US.win32.installer.exe リンクをクリックしてください。
  2. セットアッププログラムを始めるには、ダウンロードしたファイルの場所へ移動し、あなたのマシン上の SeaMonkey プログラムアイコンをダブルクリックしてください。
  3. セットアッププログラムでは、画面上の説明に従ってください。プログラムは最初、自動的に起動します。

.zip ファイルによる手動インストール

.zip ファイルをダウンロードし手動でインストールすることで SeaMonkey をインストールするには、これらのステップに従ってください：

  1. あなたのマシンにファイルをダウンロードするには seamonkey-1.1.4.en-US.win32.zip リンクをクリックしてください。
  2. ファイルをダウンロードした場所へ移動し、圧縮ファイルをダブルクリックしてください。注意：このステップは、あなたが既に WinZip の最近のバージョンをインストールしており、その使い方を知っていることを前提としています。そうでなければ、www.winzip.com で WinZip と、そのプログラムに関する情報を入手できます。
  3. C:\Program Files\SeaMonkey1.1.4 といったフォルダへ .zip ファイルを解凍してください。
  4. SeaMonkey を起動するには、SeaMonkey を解凍したフォルダへ移動し、SeaMonkey.exe アイコンをダブルクリックしてください。

アンインストール方法

  • SeaMonkey インストーラが使われていたならば、単純に、設定/コントロールパネル/プログラムの追加と削除へ行き、SeaMonkey 1.1.4 を選択してください。
  • zip ビルドが使われていたならば、単純に、あなたが SeaMonkey をインストールした SeaMonkey 1.1.4 フォルダを削除し、プロファイルフォルダ を削除してください。

プロファイルの場所

以下のリストは代表的なプロファイルの場所です：

Windows

Windows 95, Windows 98, または Windows ME （ログインなし）上での、プロファイルフォルダは一般的に
    C:\Windows\Application Data\Mozilla\Profiles\[プロファイル名]\[ランダムな文字列].slt\

Windows 95, Windows 98 またはログインする Windows ME 上での、プロファイルフォルダは一般的に
    C:\Windows\Profiles\[ログイン名]\Application Data\Mozilla\Profiles\[プロファイル名]\[ランダムな文字列].slt\

Windows 2000 または Windows XP 上での、プロファイルフォルダは一般的に
    C:\Documents and Settings\[ログイン名]\Application Data\Mozilla\Profiles\[プロファイル名]\[ランダムな文字列].slt\

注意：Windows 2000 または Windows XP 上では、Document and Settings フォルダが二重に隠されています。そのフォルダにアクセスするには、すべてのファイルとフォルダを表示するように Windows Explorer に許可してください。

Mac OS X

プロファイルの場所は一般的に
    ~/Library/Mozilla/Profiles/[プロファイル名]/[ランダムな文字列].slt/

Linux

プロファイルの場所は一般的に
    ~/.mozilla/[プロファイル名]/[ランダムな文字列].slt/

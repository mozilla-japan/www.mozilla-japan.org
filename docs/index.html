    • XML バインディング言語 - Marcell Ortutay 著

    • XUL Planet
      • XUL チュートリアル
      • XUL 1.0 ガイド - 既存の XUL コードを XUL 1.0 と互換性をもつように更新するために、知っておくべき様々なことが書いてあります
    • XUL プログラマーのための参考書 (圧縮 HTML 版 と フランス語版 もあります)
  • XPInstall - クロスプラットフォームのインストール、パッケージング、ソフトウェアの更新を行うための技術
  • NSPR - クロスプラットフォームのオペレーティングシステム機構

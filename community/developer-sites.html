Mozilla 開発者サイト

Mozilla は単なるブラウザの名前だけではありません。Mozilla は開発プラットフォームでもあるのです。mozilla.org 以外の場でも、数多くの開発者が mozilla.org テクノロジを使ったソフトウェアを作成しています。以下に Mozilla 開発に関連するリストを挙げます。

XULPlanet

XUL、XPCOM、その他のテクノロジーを使ったアプリケーションのビルドに関するする情報満載のサイト。サイトには完全なチュートリアルと参照資料があります。

OEOne

Mozilla テクノロジーを取り込んだ Linux デスクトップ環境の開発者。

Nvu

Nvu は、Mozilla Composer に基づいたオープンソースな Web 作成アプリケーションです。このアプリケーションは Daniel Glazman の Disruptive Innovations によって開発されています。

Mozilla ActiveX Project

Mozilla を ActiveX として利用可能にするための活発なプロジェクト。自分のプリケーションに最新のインターネット標準をサポートするブラウザコントロールを組み込みたいと考えているソフトウェア開発者は、要チェック。

Beonex

Mozilla をベースにしたブラウザスイートの開発者。同社はカスタム開発、コンサルティングそしてサポートを請け負います。

Xm

Xm は、Mozilla に接続する XPCOM コンポーネントの事です。これにより、電気的、論理的、そして物理的なマシーンをシミュレートする事ができるようになります。

Zvon.org Mozilla Tutorial

XUL サンプルを多数集めたサイト。

MozillaTranslator

数多くの Mozilla ソフトウェアローカライズ担当者にとって欠かす事のできない MozillaTranslator のホーム。さらに、ローカライズ担当者やローカライズ版を探しているユーザのためのヘルプサイトでもあります。

LiveSidebar.com

Mozilla に対応したサイドバーアドオンの開発についての情報サイト。計算機やメモ帳など多数のサンプルも揃えています。サイトにはサイドバーディレクトリも備えています。

Hall of Fame [xml]

私たちの殿堂ページで mozilla.org テクノロジーを使った数多くの開発者を見る事ができます。

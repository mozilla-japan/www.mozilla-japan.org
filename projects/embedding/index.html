Mozilla を組み込む

目標：

組み込みブラウザと HTML エディタ SDK を提供すること


参考文献

とにかく始めに FAQ を読んで下さい。組み込みのための現在の API セットは、こちらにあります。

一般的な質問には embedding newsgroup をご利用ください。 ［netscape.public.mozilla.embedding］

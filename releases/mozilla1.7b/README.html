Mozilla 1.7 Beta 新着情報

このドキュメントには、Mozilla 1.7 Beta で実装された新機能や重要なバグの修正、その他新たな問題の一部が載っています。1.7 Beta で行われた修正の詳細なリストは「大まかな変更ログ」をご覧ください。このリストには、1.7 Alpha 以降の 新機能 と 修正 がすべて載っているわけではありません。

新機能と修正

ブラウザ

  • Mozilla 1.7 Beta の公式インストーラビルド (Mac OS 版ディスクイメージを含む) には、Talkback クラッシュ報告ユーティリティ が付属しています。Mozilla 1.7 をこれまでで最も安定したリリースにするため、クラッシュレポートを送信して開発にご協力ください。
  • ブラウザのコンテキストメニューを表示させないようにする、JavaScript を使った Web ページの仕掛けをブロックできるオプションが追加されました。
  • Mozilla と関連付けられたファイル用の新しいアイコンセットが用意されました。
  • パスワードマネージャに、保存したパスワードを表示する「パスワードを表示」モードが実装されました。マスターパスワードを使っている場合は、モードを切り替える際にそのパスワードを入力する必要があります。
  • コンテキストメニューから「壁紙に設定」機能を使ったときに、確認ダイアログが表示されるようになりました。
  • Linux 版 (GTK2 ビルド) の OS テーマサポートが向上しました。
  • Cookie 関連のダイアログが改良され、より使いやすくなりました。
  • Mac OS 版では特に、日付処理が改良されました。

メール

  • Address Book Palm Sync に多くの改良が加えられました。
  • IMAP の IDLE コマンドをサポートしました。
  • SMTP と POP3 用の "MSN Authentication" とクロスプラットフォームな Secure Password Authentication をサポートしました。
  • 返信元メッセージの文字コードを使わずに「メッセージに返信する際は常にデフォルトの文字コード使用する」設定が追加されました。
  • メッセージのダウンロードと表示、保存処理のパフォーマンスが向上しました。

ChatZilla (IRC クライアント)

  • キーボードショートカット (Ctrl++/Ctrl+-) または View (表示) メニューを使って、文字サイズを自由に変えられるようになりました。
  • 日付処理が改良され、地域設定に合った日付と時刻の形式を使えるようになりました。

バックエンド

  • 今回のリリースで、Mozilla 1.7 のサイズとパフォーマンスが劇的に向上しました。Mozilla 1.7 Beta は、Mozilla 1.6 と比べて、起動時間が 7%、ウィンドウが開くまでの時間が 8%、ページ読み込み時間が 9% も速くなりました。また、バイナリサイズが 5% 小さくなりました。
  • 長年にわたる懸案事項であった、段組みと正確なレイヤー配置のサポートなど、テーブル要素の CSS バックグラウンドに関するバグが修正されました (標準モードのみ)。
  • GSSAPI を利用した Kerberos HTTP 認証をサポートしました。これは、Linux や Mac OS X など Unix ベースのプラットフォームに恩恵をもたらします。
  • gnome-vfs ライブラリを使った smb:// URL をサポートしました。この機能は Linux 版 (GTK2+XFT ビルド) でのみ有効になっています。
  • multipart/x-mixed-replace と XMLHttpRequest を使った XML ドキュメントのサーバプッシュをサポートしました。
  • Java アプレットのコードベースが異なるドメインにあった場合でも LiveConnect が機能するようになりました。
  • Mac OS 版で、幅が非常に大きい (4095px 以上の) 画像が正しく表示されるようになりました。
  • CSS3 の opacity (透過) プロパティをサポートしました。

新たな問題

ここには、前回のマイルストーン以降 既知の問題 のページに追加された事項が載っていますが、バグ自体は前からあった可能性もあります。

一般

  • プロファイルが複数ある場合に代替テーマ (Modern を含む) を使うと、ユーザインターフェイスの一部が、その代替テーマではなく、Classic または「ネイティブの」テーマで表示されてしまうことがあります。このバグ (Bug 237407) は、userChrome.css でテーマをカスタマイズしている場合にも影響を及ぼす場合があります。
  • Mozilla 1.7 Beta では Web サービスが機能しません。これは Bug 237845 です。
  • Mac OS 版では、IPv6 サポートがデフォルトで無効になっています。これは、IPv6 による DNS 検索を無効化する既知の方法が Mac OS X にないためで、DNS サーバとホームルータのどちらか、あるいは両方が同時にダウンした場合、接続速度が極端に低下する恐れがあります。Bug 68796 と Bug 231607 をご覧ください。Mac OS X v10.3 では、(about:config から) network.dns.disableIPv6 の値を false に設定すれば、IPv6 を再有効化できます。

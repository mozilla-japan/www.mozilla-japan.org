ユーザ向けニュースグループ

Netscape がセキュアサーバ上でいくつかのユーザ向けセルフサポート・ニュースグループをホストしています。(Linux ユーザ: セキュア・ニュースグループにアクセスするには Personal Security Manager (PSM) が必要です。これらのニュースグループにアクセスできない場合は PSM 付き Mozilla を再インストールしてください)

  • 一般的な問題 - netscape.mozilla.user.general
  • Windows - netscape.mozilla.user.win32
  • Macintosh - netscape.mozilla.user.mac
  • Unix/Linux - netscape.mozilla.user.unix

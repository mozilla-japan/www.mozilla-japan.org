ウェブサイトマークアップの使用方針

www.mozilla.org またはその他 mozilla.org ウェブサイトにある HTML および CSS ファイルをあなたのウェブサイトに使用することができます。ただし、視覚デザインやスタイル （色、グラディエント、フォントなど） は使用しないでください。言い換えると、あなたは HTML や CSS コードを使用することができますが、私たちのページにあなたが作成したロゴを足しただけ、というようには見えないようにしてください。

mozilla.org は、あらゆる HTML/CSS に適用され得る著作権を行使して、あなたが HTML/CSS をコピーしたり再利用したりすることを禁止する方針はとりません。しかし、商標、商号、サービスマーク、あるいはトレードドレス 【訳注：製品やサービスに特有のデザイン】 を使用する権利をあなたに与えるものではありません。私たちの商標ポリシーについての情報は、Mozilla 商標およびロゴ使用に関するポリシー をご覧ください。

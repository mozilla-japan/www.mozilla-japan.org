Mozilla 1.0 サポート

Mozilla は開発プロジェクトで、テスト用のビルドやテクノロジーの試用版を提供しているだけです。 mozilla.org は、残念ながらエンドユーザ向けのテクニカル・サポートを提供する資源を持っていません。

あなたが「洗練された」エンドユーザ・サポートを求めるなら、ページの下の方に載っている配布元のうちのどれかを利用するべきでしょう。 それらの多くがエンドユーザ・サポートを提供することができます。

Mozilla や Mozilla をベースとしたブラウザに関する多くの文書や、よくある質問が載っている Web サイトもいくつかあります。

  • Mozilla 1.0 リリースノート
  • Mozilla 1.0 FAQ
  • MozTips
  • Hogarth による Mozilla 参考資料
  • The Merry Amateur による Mozilla FAQ - Linux ユーザ向け
  • Gunnar による 非公式 Mozilla ヘルプサイト
  • Holger Metzger による Netscape 6/7 Tips - 英語版／ドイツ語版
  • Netscape 非公式 FAQ サイト （訳注 : 日本語版 もあります）
  • NewZilla
  • Pratik Solanki による 隠し Mozilla はしがき

それから、Mozilla のビルドを試している人のために、（Netscape のサーバ上で）いくつかのエンドユーザ・セルフサポート・ニュースグループが提供されています。

  • 一般的な問題 - netscape.mozilla.user.general
  • Windows - netscape.mozilla.user.win32
  • Macintosh - netscape.mozilla.user.mac
  • Unix / Linux - netscape.mozilla.user.unix

Mozilla 開発者のニュースグループは Mozilla ソフトウェア開発者専用で、エンドユーザの質問は上のニュースグループに転送されるということを、どうぞ忘れないでください。 開発者にサポートに関する質問についてメールを送らないでください。

以下の組織は Mozilla ベースのブラウザ製品を作っています。

Beonex
Beonex は、エンドユーザ - 個人と企業のどちらも対象とした、Windows / Linux 向けの Mozilla のオープンソースの配布元です。
Debian
Debian は、Linux カーネルを使った、フリーのオペレーティングシステムです。 3950 以上のパッケージ - あなたのマシンに簡単にインストールできるように適した形式でバンドルされている、プレコンパイルされたソフトが出ています。 それらのアプリケーションのひとつが Mozilla です。
IBM Web Browser
IBM Web Browser は、OS/2 向けの Mozilla のポートで、スペルチェッカーと Flash プラグインが加えられています。
Netscape
Netscape は Windows / Linux / Mac 向け Mozilla の市販配布元です。 これはエンドユーザをねらいとしており、AIM クライアントとスペルチェッカーが含まれています。
OEone HomeBase
HomeBase は、Mozilla ブラウザと Linux オペレーティングシステムをベースとした、完全なオペレーティング環境です。 たいていのコンピュータ初心者が使えるほどシンプルである一方、コンピュータ・ハッカーが彼らの求めるスペックにコンフィギュレーションすることも可能です。
Red Hat Linux
Red Hat Linux は Linux オペレーティングシステムの配布元で、Mozilla を含む様々なアプリケーションがパッケージされています。
Sun Netscape 6
Sun は Netscape から最新のインターネットブラウジング・メール・インスタントメッセージソフトの Solaris ポートを提供しています。
Ximian GNOME
Ximian GNOME は Linux / UNIX ユーザ向けの完全な使いやすいデスクトップ環境です。 それには Mozilla ブラウザが含まれていて、Red Hat、Mandrake、SuSE、Debian、TurboLinux その他を含む、広範囲の配布元やプラットホームをサポートしています。
Chimera / Galeon / K-Meleon
Chimera （Mac OS X）、 Galeon （Linux）、 K-Meleon （Windows） はすべて、Mozilla テクノロジーのまわりに作られた、ブラウザ専門のプロジェクトです。

これらのプロジェクトについての質問は、mozilla.org ではなく、関連する組織に向けられるべきです。

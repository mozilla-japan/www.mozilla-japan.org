Mozilla 英語コミュニティサイト

mozilla.org の外側では、Mozilla のことを扱った数多くの個人／コミュニティサイトがあります。例えば：

MozillaZine［日本語版］

独立サイトで現存する最古のサイト。Mozilla に関するニュースや意見を扱っています。Chris Nelson による運営。

Mozilla Firefox Help［日本語版］

Mozilla Firefox に関する FAQ、ヒントやコツ、テーマ、拡張機能そして幅広い資料。David Tenser による運営。

Mozilla Thunderbird Help［日本語版］

Mozilla Thunderbird に関する FAQ、ヒントやコツ、テーマ、拡張機能そして幅広い資料。David Tenser による運営。

非公式 Mozilla ヘルプページ

Gunnar の Mozilla についてのサイト。設定やセキュリティ、その他の昨日について解説した FAQ や一般的なトラブルシューティングガイド、チュートリアルや How-To。

PluginDoc［日本語版］

へルパアプリケーション（例：Flash や Java）やダウンロードマネージャのインストールや使い方について扱ったヘルプサイト。

MozillaNews

ニュース、論説、フォーラム、BonsaiWatch、そしてその他の Mozilla 関連のコンテントを扱ったもう一つのコミュニティ。

Netscape DevEdge［日本語版］

Netscape 7 および Mozilla のための Web 開発サイト。

O'Reilly Network: Mozilla DevCenter

Mozilla 開発者や Web 開発者に向けた質の高い記事のコレクション。

  • About.com Mozilla Resource Center - About.com の Mozilla に関するリソースページ。Mozilla を使うにあたっての数多くの記事を集め、特に Mozilla のメールクライアントを使う上でのヒントが多い。
  • MozManual - 印刷版の Mozilla ガイドを書くプロジェクト。この文書は Academic Free License でライセンスされています。
  • テーマ - Mozilla テーマ集めたディレクトリ。
  • ExtensionRoom - Mozilla 用拡張機能（またはアドオン）を集めたディレクトリ。
  • gemal.dk - Henrik Gemal のホームページ。Mozilla まつわる情報や blog。
  • ZillaVilla - A House for Mozilla - プロファイルやローミングを扱った Mozilla サイト。Henrik Gemal による運営。
  • UFAQ - the Netscape Unofficial FAQ［日本語版］ - 恐らく現存する 最も 古いコミュニティサイト。Mozilla や Netscape についてよくある質問に対する答え。
  • Techaholic.net Mozilla Help and Tips - もう一つのヒントとコツを扱ったページ。50 を超える良くある質問に答えている。
  • Mozilla Tips - 100 を超える Mozilla に関するヒント。このサイトは人生を楽にしてくれます。Robert Linwood と Jeffrey Linwood による運営。
  • Netscape 7 Tips and Tricks（Holger Metzger） - Mozilla の子孫である Netscape 6/7 を扱った最新サイト。サイト内の幅広いヒントとコツを扱ったページには、Mozilla ユーザにとっても便利な情報が数多く掲載されています。このサイトは Netscape 7 Compact を運営しています。
  • Netscape Solutions（Ramona） - Netscape 7 についての豊富なヒントやコツ。これらの情報の多くは Mozilla にも当てはまります。
  • MozTips - mozillaTips.com と間違えないようにしてください。Jayesh のサイトには wiki、プロファイルに関する資料、ヒントが掲載されています。コンテンツの中には過去の MozTips サイトから移行中のものがあります。
  • Introduction to Thunderbird［日本語版］ - Kay Frode による初心者向けの連載記事。
  • Introduction to Firebird/Firefox［日本語版］ - Kay Frode による初心者向けの連載記事。
  • Chris Ilias's Netscape and Mozilla Links - Mozilla Champion にある SChris Ilias のサイト。
  • Ed Mullen's Mozilla page - Ed Mullen の Mozilla Champion にある Mozilla サイト
  • Irwin's Information on Profiles - Mozilla、Thunderbird、そして Firefox のプロファイルを処理する際の情報。Mozilla Champion にある Irwin Greenwald のサイトから。
  • MozillaZine Weblogs - コミュニティメンバーの blogs。
  • Blogzilla - Mozilla についてのコミュニティ Web ログ（blog）。
  • The Burning Edge - Mozilla Firefox のナイトリービルドで行われる開発についての blog。
  • On the Verge -個人の blog。 Internet Explorer から Mozilla Firebird（現在は Firefox） への移行や Outlook Express から Mozilla Thunderbird への移行についての情報を含む。
  • mpt - Matthew Thomas の blog。
  • European Mozilla - Mozilla Developers Meeting in Europe の運営者
  • Mozilla UserGuide Project - Mozilla についてのユーザーマニュアルを作成するプロジェクト。この作業は Creative Commons License でライセンスされています。
  • Scott Baker's Mozilla Starter Guide - Mozilla のクールな機能を扱った短い紹介記事。
  • Dr Unclear's Netscape 7 Page - Mozilla 1.4 をベースにした Netscape 7.1 についての情報
  • MozCafe

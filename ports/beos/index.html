Bezilla: Mozilla for BeOS/ZETA

プロジェクトの目的

Bezilla プロジェクトは BeOS/ZETA プラットフォームに Mozilla や Firefox を実現させるための取り組みです。このプロジェクトの最も重要な目的は、Haiku や ZETA に限らず、 BeOS 系オペレーティングシステムとその後継 OS に、フリーでしかもオープン Web スタンダードに準じたブラウザーを提供することです。

スクリーンショット

リンク

  • BeZilla Blog: Bezilla プロジェクトの開発者が主に開発報告やテストに使っているブログです。
  • Mozilla Firefox: Bezillaチームが定期的にリリースする Firefox for BeOS/ZETA の安定ビルドのダウンロードページです。
  • Mozilla SeaMonkey: Mozilla Web Suite/Browser の bebits.com でのダウンロードページです。
  • Mozilla Firefox - Bleeding Edge: 主にバグ探しのためにリリースされる Firefox for BeOS/ZETA テストビルドのダウンロードページです。
  • HaikuFox - Firefox Theme: BeOS の雰囲気を持った Firefox 用テーマです。

    • チームメンバー

    現在、Bezilla チームは下記のメンバーで構成されています。

  • Sergei Dolgov (Fyysik): 開発者、widget、gfx、i18n やその他。現在、Mozilla SeaMonkey のビルドも担当。
  • tqh: 開発者、BeOS 用 NSPR、XPCOM または Widget での低レベルコード。Firefox のビルドも担当。
  • tb100: 色々なパッチを提供。主に gfx 用。
  • Nielx: 色々なパッチの提供、BeFox のバグトラッキング。
  • Doug Shelton (tigerdog): テストやビルド作成、HaikuFox テーマの開発。
  • Jorge Mare (Koki): テストとこれらのページのメインテナンス

    • Bezillaプロジェクトへの参加

    本プロジェクトを手伝ってくださる方を歓迎します。プログラムを書いたり、テストをしたり、ドキュメントを作成したり、または他のメンバーを応援したりするだけでも、Bezilla プロジェクトに参加する意欲　（と時間） があれば、bezilla@zeta-zone.net 宛てにメールください。日本語でもOK！

    バグ報告

    バグが見つかったら、http://bugzilla.mozilla.orgにてバグ報告してください。その際、BeOS 関連バグであることを指定してください。バグ報告は、bugzilla のガイドラインに従って報告してください。

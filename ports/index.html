Mozilla の移植作業

数々のグループが新たなプラットフォームやツールキットに対応した Mozilla の移植を計画しています。このページでは、現在知られている全ての Mozilla の移植のリストと、そうした活動のもっと詳しい情報の見つけ方について掲載します。Tinderbox Ports のページでは、様々なプラットフォームに対応したビルドをチェックできるようになっています。もしあなたが自動ビルド用のマシーンを持っていたら、Tinberbox のページに掲載することもできます。

その他の Mozilla に関するプロジェクト同様、もし皆さんが何らかの移植プロジェクトを真剣に進めており、プロジェクトに関してのウェブページや メーリングリスト をここ、mozilla.org 上に掲載したいとお考えでしたら、ご連絡ください。

次のポートプロジェクトに関するページは、現在 mozilla.org 内に掲載されています。これらのページは、 mozilla.org のスタッフによってではなく、外部の人たちによって管理されていることにご注意ください。

  • QtMozilla：TrollTech Qt Toolkit への移植プロジェクトの管理。
  • Fizzilla：Cocoa (Yellow Box) ではなく、 Carbon を利用した Mac OS X 用の移植。
  • OS/2: IBM の OS/2 用の移植。
  • BeOS： BeOS や、その後継 OS である Haiku や ZETA への移植。

mozilla.org サイト外のリンク

  • Irix： Irix オペレーティングシステム用移植。
  • OpenVMS： OpenVMDS オペレーティングシステム用移植。
  • Troll Tech は、UNIX フロントエンドを彼らの製品、Qt toolkit（ Motif の代替ツールキット）用に移植しました。
  • Jazilla とは、オープンソース Java をペースにした開発プロジェクトで、Pure Java でウェブブラウザを作成することを目標としています。コードやデザインのほとんどは Mozilla をベースにしており、Netscape Navigator のオープンソース版というべきものです。
  • Java フロントについてもっと興味深いものは、Steve Wynne のページで、Mozilla にフリーの Java 実装を組み込む作業について研究しています（インターネットアーカイブからは遅れています）。
  • mozAmiga：新しい Amiga DE （Digital Environment） への移植。
  • Unix Porting Project - Firefox は、Acorn RISC OS 用の移植プロジェクト。

この他の移植プロジェクトを知っていますか？　教えて下さい！

Mozilla Foundation セキュリティアドバイザリ 2014-64

タイトル: 高品質画像拡大時の Skia ライブラリにおけるクラッシュ
重要度: 高
公開日: 2014/07/22
報告者: John
影響を受ける製品: Firefox、Thunderbird

修正済みのバージョン: Firefox 31
  Thunderbird 31
  Firefox ESR 24.7
  Thunderbird 24.7

概要

高品質画像の拡大時に、拡大操作にかかる時間が長すぎた場合に生じる Skia ライブラリにおけるクラッシュが、Mozilla コミュニティメンバーの John 氏によって報告されました。これは、画像データが拡大操作で使用中にもかかわらず破棄されてしまうことによって生じる問題でした。このクラッシュは一部のシステムでは潜在的に悪用可能なものでした。

Thunderbird では JavaScript が初期設定で無効になっているため、一般的にこれらの脆弱性はメールを通じた悪用が不可能です。ただしこれらの製品でも、ブラウザもしくはブラウザに類似した機能を利用した場合、潜在的なリスクが生じます。

Firefox 1.5/2.0 CCK (クライアント・カスタマイズ・キット) ウィザード

バージョン 1.1 リリースノート

CCK ウィザードのバージョンVersion 1.1 がリリースされました。

addons.mozilla.org で入手できます。

https://addons.mozilla.org/firefox/2553/

このバージョンで Firefox 2.0 がサポートされました。Firefox 2.0 のための更新点は次の通りです。

  • ブックマークへの区切り線挿入のサポート。
  • OpenSearch ファイル読み込みのサポート。

バグの報告は こちら へ。

FAQ のページもあります。質問や問題点があれば、私にお知らせください。FAQ に追加できます。

以前のバージョンにおける変更点については 過去のリリースノート をご参照ください

商標に関する重要な情報

商標についての考慮が必要になるのは、あなたが組織や会社の外部に配布しようとしている場合だけである、 という点に注意してください。

派生物の商標に関する注意点は、それが CCK で作られたかどうかには関係しません。 CCK は、フリーパスを与えるものではなく、また派生物に許されるものと許されないものを 左右するものでもありません。

派生物に許可されているものといないものについては、 こちらのウェブサイト を参照してください。

ディストリビュータ向け

Firefox 1.5/2.0 CCK は、Firefox をカスタマイズする拡張機能を作成することができます。 これにより、以下のカスタマイズが可能になります。

  • ユーザエージェント文字列に識別子を追加する
  • デフォルトのホームページと起動時のページを変更する
  • タイトルバーのテキストを変更する
  • 読み込み中アイコンを変更する
  • 読み込み中アイコンのリンク先とツールチップを変更する
  • ウェブページへリンクするヘルプメニュー項目を追加する
  • ポップアップと XPInstall がデフォルトで許可されるサイトを規定する
  • ブラウザプラグインをプリインストールする
  • 検索エンジンをプリインストールする
  • 個人用ツールバーにフォルダ、ブックマーク、ライブブックマークを追加する
  • ブックマークフォルダにフォルダ、ブックマーク、ライブブックマークを追加する
  • デフォルトの設定を定める
  • 設定をロックする
  • レジストリキーを追加する (Windows のみ)
  • 証明書を追加する (下の注意を参照)
  • デフォルトのプロキシ設定を定める
  • 他の XPI をパッケージに同梱する
  • about:config を無効にする

重要: CCK が動作するには、パスの通ったところに ZIP.EXE がなければなりません。または、MAF 拡張機能をインストールする方法もあります。

注意: 証明書を Firefox へ追加した場合、そのブラウザを Firefox という名前のまま、会社や施設の外で公開することはできません。この機能は、Firefox を会社や施設のユーザーに配布する、という目的のためのものです。

CCK がインストールされると、ツールメニューに "CCK Wizard" というメニュー項目が追加されます。この項目から起動できるウィザードに従うことによって、Firefox 1.5/2.0 をカスタマイズするための XPI ファイルが作成できます。

CCK を Windows のインストーラの一部にする場合、「Firefox 1.5 CCK の Windows インストーラパッケージへの追加」を参照してください。

CCK を Firefox 2 のインストーラの一部にする場合、Firefox Release Repackager を参照してください。現在の所 Mac 限定ですが、他のプラットフォームでも作業中です。

CCK をアップデートするためのアップデートサイトを作りたい場合、この文書 を参照してください。

開発者向け

今のところ、CCK は拡張機能として extensions/cck/browser ディレクトリに置かれています。CCK をトランクにビルドする場合、.mozconfig ファイルに以下の行を追加してください。 

ac_add_options --enable-extensions=default,cck

これにより CCK ウィザードを開発ビルドで使用することが可能になります。CCK ウィザードは Mozilla 1.8 Branch の一部ではないことに気をつけてください - トランクでビルドしなくてはならないのです。

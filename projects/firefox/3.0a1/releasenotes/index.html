Gran Paradiso Alpha 1 は、次世代 Mozilla レイアウトエンジン Gecko 1.9 上にビルドされる Firefox の次期メジャーバージョンに向けた初期開発者向けマイルストーンです。Gran Paradiso Alpha 1 は テスト目的にのみ限って 提供され、Web アプリケーション開発者やテストコミュニティを対象にしています。現在 Mozilla Firefox をお使いの方は、Gran Paradiso Alpha 1 を 利用しないでください。

このリリースノートには、Gran Paradiso Alpha 1 に関する 新機能、ダウンロード、インストール方法、既知の問題、よくある質問 が載っています。Bugzilla にバグを報告する前に、このリリースノートと バグの登録方法 をよくお読みください。

こちらの フィードバック欄 から皆さんのフィードバックをお寄せください。

本開発マイルストーンにおける変更点

Gecko 1.9 Alpha 1 に複数の新機能が導入され、Gran Paradiso Alpha 1 で検証できるようになりました：

  • Cairo が標準のグラフィックライブラリとして採用され、画像およびテキストの描画全体に影響を与えています。
  • Mac OS X 用ビルドでは Cocoa ウィジェット が採用されています。
  • スレッドモデル がアップデートされています。
  • DOM イベントの切り離し方法が変更されています 。（参照：bug 234455）
  • <object> 要素のロード方法が変更されています 。（参照：bug 1156）
  • Web ページのペイント方法 が変更されています。
  • 新たな SVG 要素やフィルタが追加され、SVG 仕様との適合性が向上しています。

Gecko 1.9 Alpha 1 における複数の変更により、Gran Paradiso Alpha 1 の Web やプラットフォームとの互換性に影響が出ています：

  • Windows 95、Windows 98 および Windows ME はサポートされません。
  • OS X 10.2 はサポートされません。OS X 10.3.9 以降を推奨します。
  • ドキュメント間の DOM ノードを移動するには、DOM の仕様 にあるように importNode または adoptNode を呼び出す必要があります。

 

ダウンロードとインストール

必要システム構成

インストールの前に、お使いのコンピュータが 必要システム構成 を満たしているか確認してください。

Gran Paradiso Alpha 1 のダウンロード

Mozilla.org からは、さまざまな言語で Windows、Linux、および Mac OS X に対応した Gran Paradiso を提供しています。Gran Paradiso の最新版は、FTP サイト をご確認ください。mozilla.org によって提供されていない、他のシステム・言語向けのビルドについては、このページの最後にある「提供されたビルド」の項目をご覧ください。

Gran Paradiso のインストール

Gran Paradiso をダウンロードしたら、次の指示に従ってください：

Windows

Gran Paradiso Setup Alpha 1.exe インストーラをダブルクリックして、インストールします。

Mac OS X

Gran Paradiso Alpha 1.dmg ディスクイメージをダブルクリックして、イメージを展開・マウントします。ブラウザによってはこの作業を自動的に処理するものもあります。GranParadiso ディスクイメージをダブルクリックして Finder に開き、Gran Paradiso のアプリケーションアイコンをハードディスクにドラッグしてください。ディスクイメージにあるアイコンをダブルクリックしないでください！ アプリケーションを実行する前に、必ず Gran Paradiso アプリケーションをディスクイメージの外にドラッグしてハードディスクにコピーしてください。アプリケーションを Dock に表示したい場合は、アイコンを Dock にドラッグしてください。

Linux/GTK2

tarball を解凍して ./firefox を実行します：

tar -xzvf granparadiso-alpha1.tar.gz
cd firefox
./firefox

その他のプラットフォーム

圧縮されたアーカイブを解凍して、firefox を実行します。

Gran Paradiso のアンインストール

Gran Paradiso をアンインストールするには、次の指示に従ってください：

Windows

スタート メニューから コントロールパネル を選択します。コントロールパネルが表示されたら、プグラムの追加と削除 をダブルクリックします。一覧から 「Gran Paradiso (3.0a1)」 を探して、削除 ボタンをクリックしてアンインストールします。

Mac OS X

GranParadiso アプリケーションをゴミ箱にドラッグします。

その他

firefox フォルダを削除します。

以上のガイドではプロファイルデータはそのまま残ります。将来 Firefox をインストールした際に再び使えるようにするためです。プロファイルフォルダを削除したい場合は、プロファイルフォルダ のセクションで示されているフォルダを削除してください。プロファイルフォルダを削除すると、ダウンロードした拡張機能やテーマ、ブックマーク、保存されたパスワード、設定その他の情報が削除され、再び Firefox を使用する際にはまっさらなプロファイルで初めから設定し直す必要があります。

 

その他の情報

プロファイルフォルダ

Gran Paradiso は次の場所のいずれかにユーザデータを保存します：

Windows 2000、XP Documents and Settings\<ユーザ名>\Application Data\Mozilla\Firefox
Windows NT WINNT\Profiles\<ユーザ名>\Application Data\Mozilla\Firefox
Mac OS X ~/Library/Application Support/Firefox
Linux および Unix システム ~/.mozilla/firefox

アドオン

Firefox 環境下でインストールされた拡張機能には、互換性が無く、Gran Paradiso で利用するためにアップデートが必要となるものもあります。なにか問題が見つかった場合は、その拡張機能の管理者にご連絡ください。Gran Paradiso をインストールすると、すべてのアドオンは、Gran Paradiso が次のいずれかの判断をするまで利用できなくなります： a） Gran Paradiso リリースと互換性がある、b） 互換性のある新しいバージョンが利用できる。

既知の問題

ran Paradiso Alpha 1 で確認されている問題の一部を掲載します。新しいバグを報告する前に、かならずお読みになるとともに、新しいバグが見つかっていないか確認してください。

すべてのシステム

  • Cairo グラフィックシステムは、過去の Gecko と比較すると、テキストや画像の描画において大幅な変更となります。そのため、非ラテン語系のスクリプトやフォントで誤った描画をすることがあります。
  • フィッシング詐欺サイト警告機能の通知がコンテントエリアに隠れてしまいます。（参照：bug 341950）
  • オンライン・オフラインの切り替え時に Gecko が WHATWG DOM イベントを送るようになりました。（参照：bug 336359）
  • nsCaret の挙動を変更することで、数多くのバグが修正 されました。

Windows

  • アップスケールされた画像の描画にエラーが発生し、画像の輪郭がぼやけることがあります。

Mac OS X

  • ユーザが設定したデフォルトフォントが適用されないことがあります。
  • 下線 や 打ち消し線 といったフォント装飾が適切に表示されない可能性があります。
  • Complex Script （例： Indic） の描画を正常におこなえないことがあります。

Linux および Unix システム

  • 中国語、日本語、または韓国語で書かれたページでブラウザがハングする可能性があります。（参照：bug 357637）
  • Gecko の先のバージョンと比較して、Complex Script （例： Indic） の処理に時間がかかる可能性があります。
  • Xorg 7.0 以降を実行していないと、互換性問題が発生する可能性があります。
  • 複雑なページを印刷する際に問題が発生する可能性があります。

トラブルシューティング

  • 設計が悪い、あるいは互換性のないアドオンを使うと、頻繁にクラッシュしたり、ページの読み込みが遅くなるなど、ブラウザに問題を引き起こす可能性があります。ブラウザの一部機能が使えない、ブラウザが起動しない、ウィンドウの表示がおかしい、パフォーマンスが低下した、などの問題に遭遇した場合、アドオンが原因となっている可能性があります。そのようなときは、ブラウザをセーフモードで再起動してください。Windows では、スタートメニュー内に作成された "Safe Mode" ショートカットを使うか、firefox.exe -safe-mode で起動してください。Linux では ./firefox -safe-mode で起動してください。Mac OS X では、ターミナルを開いて以下のコマンドを実行してください。

    cd /Applications/GranParadiso.app/Contents/MacOS/
    ./firefox-bin -safe-mode

    セーフモードで起動すると、トラブルを引き起こしているアドオンを無効化できます。その後ふたたび再起動してください。

ユーザプロファイルにインストールした拡張機能 （Web ページからインストールしたものなど） をアンインストールし、コマンドラインオプション -install-global-extension を使ってすべてのユーザプロファイルにインストールしたい場合は、コマンドラインオプションを付けてインストールする前に、一度ブラウザを再起動し、その拡張機能の痕跡が残るプロファイルの拡張機能データソースを一掃する必要があります。そうしないと、拡張機能リスト内のエントリーがおかしくなり、目的の拡張機能をグローバルにインストールできなくなる場合があります。

ブックマーク、ダウンロード、ウィンドウの配置、ツールバー、履歴、その他の設定に関して不可解な問題に遭遇した場合は、バグを報告する前に、新しいプロファイルを作成して問題が再現されるかどうかを確認してください。新しいプロファイルを作成するには、-P コマンドライン引数をつけて Firefox を起動し、［新しいプロファイルの作成］ ボタンをクリックしてください。設定ファイル （ブックマーク、保存されているパスワードなど） をひとつひとつ移行し、その度に問題が発生するかを確認してください。特定のプロファイルデータのファイルが問題の原因だとわかったら、バグとして報告して、そのファイルを添付してください。

 

よくある質問

  1. どのような協力ができますか？

    Firefox をこれまでになく良いものにするために、フィードバックをできるだけたくさん集められるよう、開発者の皆さんやテストコミュニティからの支援を必要としています。こちらの Gran Paradiso ページ をご覧ください。

  2. なぜ私が送ったメールに返事をくれないのですか？

    フォーラム をご利用ください。Gran Paradiso チームは常時このフォーラムをチェックしています。私たちは毎日多くのメールを受け取っているので、あなたが送ったメールは見過ごされる可能性もあります。

  3. アドオンはどこで手に入りますか？

    拡張機能 と テーマ は Mozilla Add-ons からダウンロードできます。

  4. 誰が Gran Paradiso の開発をしているのですか？

    非常に多くの人が関わっています。［ヘルプ］ メニューの ［Mozilla Firefox について］ を開いて「クレジット」があります。ここに Gran Paradiso に貢献した人たちの一覧があります。

  5. Gran Paradiso のソースコードはどこにありますか？

    Gran Paradiso のソースコードは tarball として ダウンロード できます。最新の開発コードは cvs を通じて入手可能です。Firefox 固有のソースは "mozilla/browser" と "mozilla/toolkit"、"mozilla/chrome" にあります。詳しくは ビルド方法 に従ってください。

  6. メールクライアントはどこにあるのですか？

    Gran Paradiso は、お使いのシステムでどのアプリケーションがデフォルトのメールクライアントになっていても、それと併せて使うことができます。が、Firefox ブラウザにぴったりな、Mozilla の次世代メールクライアントとして Mozilla Thunderbird をおすすめします。

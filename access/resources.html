リンクと情報

ここに、オープンソースアクセシビリティにもウェブアクセシビリティにも役立つ、参考リンクをまとめました。

  • Bobby - ウェブサイトアクセシビリティチェッカー
  • WAI - Web アクセシビリティ・イニシアチブ
  • WAI interest group - ウェブアクセシビリティの標準のための議論リスト。
  • Trace Center - ウィスコンシン大学のトレースセンタ ( 障害者のための技術の研究 ) 。 LARS (Linux Accessibility Resource Site) のホーム。
  • IBM Accessibility Center - IBM は、アクセシビリティソリューション研究の長い歴史があります。 Understanding Accessibility と同様に Web Accessibility Checklist も必ず見てください。
  • Dive Into Accessibility - もうひとつのすばらしい、アクセシブルなウェブサイトオーサリングにおける、理解しやすい情報。
  • ICAES - International Coalition of Access Engineers and Specialists。 アクセシブル技術の研究、開発および配備のためのグループ。いくつかの標準組織の上に、ボードメンバがいます。
  • IBM's Java accessibility checklist - Java のアクセシビリティ作成のガイドライン。
  • Gnome Accessibility - Gnome アクセシビリティプロジェクト。
  • KDE accessibility - Linux 上で同じく重要なツールキット。
  • Flash accessibility - Flash MX と ActionScript のアクセシビリティ。
  • Apple Accessibility - OS X における進歩。
  • Information and resources on Section 508 - ソフトウェアアクセシビリティを要求する米国政府購入用の法的な政策。

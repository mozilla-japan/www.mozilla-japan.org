Mozilla Firefox - ブランド名に関する良くある質問

なぜ名称を変えたのですか?

重複を防ぐためです。"Firebird" という一般的な名称は、すでに別のオープンソースプロジェクトでも使われています。私たちが Firebird の名称を使っても、彼らの商標を侵害することになるとは思いませんが、同じオープンソースの開発者たちが抱く懸念に対して対応を見せたかったのです。

なぜこれほど時間が掛かったのですか?

「この名称は恒久的なものですか?」をご覧ください。

そもそもなぜ Firebird という名称を選んだのですか?

私たちは Firebird の名称を選択するにあたって、弁護士の助けを借りて、この名称が他でどのように使われているかを評価しました。その結果、それらを使っている企業やプロジェクトが、私たちと同じ Web ブラウザの分野に属するとは認められませんでした。すでに他の企業やプロジェクトで使われていない優れた名称を思い付くのは大変難しいことです。そのため、名称を選び、それが使えるかどうかを確かめることは、多くの難しい判断を必要とする複雑なプロセスなのです。

なぜ Firefox なのですか?

Firebird に似ているからです。覚えやすくて響きも良く、ユニークな名称です。私たちはこの名称を気に入っています。また、少しでも Web ブラウザと関連がある他のプロジェクトや企業で、同じ名称を使っているところはひとつも見当たらなかったからです。

Firefox とは何ですか?

"Firefox" は レッサーパンダ の別名です。

この名称は恒久的なものですか?

そう願っています。この一年で私たちは、名称の選択について (望んでいた以上に) 多くのことを学びました。この名称を選ぶにあたっては、慎重に調査を進め、今後絶対に問題が起こらないことを確認しました。私たちは、この新しい商標を米国特許商標局に登録する手続きを開始しました。

コミュニティとどのような相談があったのですか?

プロジェクトのリーダーたちが関わりました。また、MozillaZine で行われたコミュニティによる提案も検討しました。今回の決定には、十数人のプロジェクトスタッフと、その他関係のある貢献者たちのグループが参加しました。

ロゴやアイコンはどうなりますか?

今回の決定は他の Mozilla プロジェクトにどう影響しますか?

この課題に取り組むために、私たちはこの先数ヶ月にわたって、より詳細なブランド戦略を立てるつもりです。ブランディングドキュメントもそれに沿ってアップデートされます。

なぜ＊＊＊と呼ばないのですか?

すでに名称は選ばれてしまいました。万が一私たちが見過ごしてしまった Firefox という Web ブラウザが他に存在しない限り、この名称を変えることはありません。私たちは他の提案を求めてはいません。何百という選択肢が徹底的に検討された結果、Firefox が新名称になったのです。

Firefox? それとも Mozilla Firefox?

正式名称は Mozilla Firefox です。略して Firefox と呼ぶこともあります。

今回の決定で人々が混乱するのでは?

確かにその通りです。しかし、世界プロレス連合 (訳注: WWF = 世界自然保護基金との裁判に敗訴し、WWF から WWE に名称を変更) が実現したように、私たちにも同じことが可能です。それに、半年も経てば、他に名前があったことなど忘れてしまうでしょう。

それでも私は新名称が嫌いです。ばかげています。

私たち編集者は、これが質問にあたるのかどうかを判断しかねています。もちろん、誰もが新名称を好きになれるとは思えません。特に最初はそうです。しかし私たちは、ほとんどの人がすぐに慣れるだろうと信じています。新名称というのは、最初はひどく聞こえてしまうものなのです。もしあなたがこの新名称に不満を持っているなら、私たちのブラウザの最新リリースに加えられた多くの改良を実際に使ってみてください。少しは気分が良くなると思います。結局のところ、いちばん重要なことは、それがどう呼ばれるかではなく、どう機能するかということではないでしょうか。

- Steven Garrity、Gervase Markham、Ben Goodger、Bart Decrem 他

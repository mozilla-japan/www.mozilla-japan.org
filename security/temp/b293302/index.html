JavaScript を無効化する方法

  • Windows
  • Mac OS X
  • Linux
  • Bug 293302 - Firefox 1.0.3 Critical Vulnerability
  • セキュリティセンターに戻る

Windows

  1. [ツール] メニューから [オプション] を選択します。

  2. 左側にある [Web 機能] アイコンを選択します。
  3. [JavaScript を有効にする] のチェックを外します。

  4. [OK] を選択してオプションウィンドウを閉じます。

Mac OS X

  1. [Firefox] メニューから [環境設定] を選択します。

  2. 左側にある [Web 機能] アイコンを選択します。
  3. [JavaScript を有効にする] のチェックを外します。

  4. [OK] を選択して環境設定パネルを閉じます。

Linux

  1. [編集] メニューから [設定] を選択します。

  2. 左側にある [Web 機能] アイコンを選択します。
  3. [JavaScript を有効にする] のチェックを外します。

  4. [OK] を選択して設定ウィンドウを閉じます。

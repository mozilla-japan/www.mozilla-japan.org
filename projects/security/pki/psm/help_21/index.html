PSM 2.1 ヘルプ ＆ プライバシーヘルプ: 状況と進行中の作業

ニュースグループ: netscape.public.mozilla.crypto
作成者: Sean Cotter

状況
進行中の作業
関連する情報

このページからのリンクは Netscape 6/Mozilla 0.9 それ以降のビルドで見られます。

状況

プライバシー & セキュリティヘルプファイルの整頓や改定は現在更新中で、 Mozilla 0.9.2/Netscape 6.1 と共に公開しています。

最新のチェックインされる変更は以下のものが含まれます：

  • パスワードを忘れた場合はどうしますか？：新しいマスターパスワードを再設定するために書き換えました。
  • 身元証明のための証明書を管理する：証明書やスマートカード/外部セキュリティデバイスのバックアップを取れないという事実を踏まえての変更。
  • CRL を管理する：次回アップデート分野についての情報を含める拡張。
  • プロファイルを管理する：新しいセクション。
  • プライバシー & セキュリティ設定 - Cookie： 新しいセクション。
  • プライバシー & セキュリティ設定 - フォーム： 新しいセクション。
  • プライバシー & セキュリティ設定 - マスターパスワード - マスターパスワードを変更する：変更された UI を反映した変更。
  • プライバシー & セキュリティ設定 - マスターパスワード - マスターパスワードを再設定する：新しい UI のための新しいセクション。
  • プライバシー & セキュリティ設定 - SSL：再検討された TLS 情報（プロキシバグはフィックスされました）そして SSL の警告（オプション名が変更されました）。
  • あなたの証明書： 証明書やスマートカード/外部セキュリティデバイスのバックアップを取れないという事実を踏まえての変更。［証明書の削除］セクションでの変更を UI やヘルプボタンターゲットに反映させるには。
  • Web サイト証明書を設定する：変更を UI に反映するためには。
  • 認証局の証明書を設定する： 変更を UI に反映するためには。
  • 新しい Web サイト証明書：変更を UI に反映するためには。
  • ブラウザが CRL にアクセスしない：サーバーの証明書向けのクライアントの利用可能な CRL が現時点で設定されている時間よりも前に更新時期が設定されている場合に起こる警告のための新しいセクション。
  • 用語集：セキュリティデバイス、ソフトウェアセキュリティデバイスに施された変更の定義。
  • 全編を通して "クリックしてリストを拡張する" から "ダブルクリックしてリストを拡張する" への変更。
  • ヘッドラインや目次との整合性を含めた、様々なフォーマット、テキストへの変更。

進行中の作業

ヘルプを見直すのに一番良いのは、（訳注: ブラウザを操作している）状況の中で、現在のビルドを使って見直すことです。この内容は、論点の参照のため、現在行っている作業のスナップショットを下に提供します。

左の三角形が Mozilla ヘルプビューアのナビゲーションフレームを開くとき、プライバシーとセキュリティーに関する以下の見出しが現れます。対応するセクションを見るには、下のリンクをクリックしてください。Mozilla ヘルプビューアでのフォーマットとは違います。

ダイアログボックスヘルプセクションにある見出しのほとんどは、UI のヘルプボタンに対応します。

  • プライバシー保護機能を使う
    • インターネット上のプライバシー （背景と一般的なガイドライン）
      • Web サイトは私の個人情報をどのように集めるのですか？
      • ブラウザはどういう情報を Web サイトに渡しているのですか？
      • cookie とは何ですか？どのように使われるのでしょう？
      • 私の個人情報が許可なく使われないようにするにはどうしたらいいのでしょう？
      • 匿名でブラウズする
    • Cookie マネージャを使う （手続き型ヘルプ）
    • パスワードマネージャを使う （手続き型ヘルプ）
    • フォームマネージャを使う （手続き型ヘルプ）
    • 保存されている機密情報を暗号化する （手続き型ヘルプ）

  • 証明書を使う （手続き型ヘルプ）
    • 自分の証明書を手に入れる
    • Web ページのセキュリティをチェックする
    • 証明書を管理する
    • スマートカードと他のセキュリティデバイスを管理する
    • SSL の警告と設定を管理する
    • 検証の設定を管理する

  • プロファイルを管理する
    • 新しいプロファイルを作成する
    • プロファイル名を変更・削除する

  • ダイアログボックスヘルプ

    • 設定 （状況依存型ヘルプ）
      • プライバシーとセキュリティ
        • プライバシーとセキュリティ設定 - Cookie
        • プライバシーとセキュリティ設定 - フォーム
        • プライバシーとセキュリティ設定 - Web パスワード
        • プライバシーとセキュリティ設定 - マスターパスワード
        • プライバシーとセキュリティ設定 - SSL
        • プライバシーとセキュリティ設定 - 証明書
        • プライバシーとセキュリティ設定 - 検証

    • SSL のページ情報 （状況依存型ヘルプ）
      • Web サイトを特定できません。検証されていません／接続は暗号化されていません
      • Web サイトを特定しました。検証されています／接続は暗号化されています
      • Web サイトを特定しましたが、条件つきで検証されています／接続は暗号化されています
      • Web サイトを特定しました。検証されています／接続は暗号化されていません
      • Web サイトを特定しましたが、条件つきで検証されています／接続は暗号化されていません

    • 証明書マネージャ （状況依存型ヘルプ）
      • あなたの証明書
        • 証明書をバックアップするパスワードを選択する
        • あなたの証明書を削除する
      • Web サイトの証明書
        • Web サイトの証明書の設定を編集する
        • Web サイトの証明書を削除する
      • 認証局の証明書
        • 認証局の証明書の設定を編集する
        • 認証局の証明書を削除する
      • セキュリティデバイスマネージャ

    • 証明書の情報と決定 （状況依存型ヘルプ）
      • 証明書の詳細
      • セキュリティデバイスを選択する
      • 証明書のバックアップ
      • 身元確認の要求
      • アクセスできない身元証明
      • 新しい認証局
      • ブラウザが CRL にアクセスしない。
      • Web サイトの証明書
        • 新しい Web サイトの証明書
        • 期限切れになった Web サイトの証明書
        • Web サイトの証明書はまだ検証されていません
        • 予期しない証明書名

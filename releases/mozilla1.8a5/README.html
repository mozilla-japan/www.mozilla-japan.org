Mozilla 1.8 Alpha5 新着情報

この資料では、Mozilla 1.8 Alpha5 に実装された新機能、重要なバグ修正、そして新たな問題をいくつか紹介します。1.8 Alpha5 に施された修正のより包括的なリストは、大まかな変更ログ 【本家にリンク】 をご覧ください。

新機能と修正

バックエンド

  • CSS3 コラムの試験的サポートが有効になりました (Bug 251162)。
  • 信頼できないドキュメントがユーザのハードドライブから画像を読み込めないようにしました (Bug 69070)。
  • 「名前を付けて保存」のコードが HEAD を使わないように変更され、多数の問題がまとめて修正されました (Bug 160454)。
  • ウィンドウをリサイズしても背面のタブの表示が崩れなくなりました (Bug 227361)。
  • ソース表示ウィンドウでページ上の一部の文字 (間違った引用句など) が表示されない既知の問題がすべて修正されました (Bug 57724 の依存)。
  • XTF フレームワークが有効になりました (Bug 269859)。
  • (最初の文字の) ドロップキャップの表示が改良されました (Bug 21616)。
  • 再描画の間に、一部の場所でコンテンツが点滅していた多くの問題が修正されました (Bug 238493)。
  • window.print() コマンドを使って Web ページ上から印刷を行う際に、プリンタの設定が反映されるようになりました。

Mozilla Thunderbird 0.1

リリースノート

Mozilla の Thunderbird メールクライアントとは Mozilla メールコンポーネントを再デザインしたものです。私たちの目的は、XUL ユーザインターフェース言語を使用した、単独で動作するクロスプラットフォームなメールアプリケーションを作成することです。Mozilla Thunderbird プロジェクトについての更なる情報はこちらで御覧になれます。私たちは Mozilla Seamonkey メール コンポーネント 用に既に開発されていた機能性の多くを活用しています。　

これは最初のマイルストーン リリースです。さらに多くの作業が必要なことは認識しています。しかし同時に、私たちは  Mozilla Thunderbird のこの進展にとても興奮していて、コミュニティのみなさんとこの気持ちを分かち合いたいと思っています。現在のこのマイルストーンはMozilla 1.5b トランクで進行中の一部分を基礎にしています。

このドキュメントは Mozilla Thunderbird 0.1 リリースの 新機能、ダ ウンロードとインストールの仕方、そして 既 知の問題 をカバーしています。 Bugzilla にバグを報告する前にこのリリースノートに目を通しておいてください。

私たちは Mozilla Thunderbird に関するみなさんのフィードバックをお待ちしています。MozillaZine にホストしてもらっている Mozilla Thunderbird forums に参加してみてください。

新機能

これは最初のマイルストーンリリースなのですべてが新しいです！Mozilla Thunderbird の作業の完全な記録はこちらで見ることができます。ここではいくつか主な機能を紹介します：
  • カスタマイズ可能なツールバーとメール３ペインウインドウ

    望みどおりに ツー ルバーをカスタマイズする ことができます。これを行うにはどのウインドウ上でも View / Toolbars / Customize を選択してください。Mozilla Thunderbird ではまた、新しく縦の３ペイン設定 (Tools / Options / General) もサポートしていて、あなたのメールの見方にさらに多くの選択肢を提供します。

  • Extension（機能拡張）

    Mozilla Thunderbird には UI Extension を追加することができ、これによって特定の機能に関する操作感をカスタマイズしたり、あなたが必要とする機能を強化したりすることができます。 Extension によって、オフライン メール サポート のようなみなさんのニーズにあった機能を付け加えることができます。現在使用することの出来る Extension の完全なリストはこちら（日本語版） を参照してください。

  • コンタクトマネージャー

    メール作成用のコ ンタクト サイドバーによって、アドレスブックの連絡先を簡単便利にメールに挿入することができます。

  • ジャンクメールの検出

    Mozilla Mail と同じ方法で自動的にジャンクメールを検出するのに加えて、Thunderbird では、プライバシーをより保護し、ジャンクとして識別されたメールを安心して閲覧するため に、こうしたメールの HTML 部分を削除します

  • 新しいデフォルトのテーマ

    Mozilla Thunderbird 0.1 には、Arvid Axelsson よる素晴らしい Qute テーマを元にした、生き生きして斬新で、そして魅力的なテーマが備わっています。これは Mozilla Firebird に使用されている物と同じテーマであり、Thunderbird に似たようなルックアンドフィールをもたらします。 Thunderbird はまた、クライアントの外観を変更できるダウンロード可能なテーマ（日本語版）もサ ポートしており、テーマの数は増加し続けています。

  • 能率的なユーザインターフェースと簡素化されたオプション UI
  • 組み込みのスペルチェッカー

ダウンロード、インストールとアンインストール

古い Mozilla Thunderbird ディレクトリは、そこにあるファイルをただ上書きするのではなく、必ず削除してください。 そうしないと結局問題を抱えるはめになります。Mozilla Thunderbird に関するバグ報告を行なう場合、まず最初にクリーンインストールを行なってからにしてください。

ボランティアの方達が Windows、Linux (GTK2)、そして Mac OS X 用の Mozilla Thunderbird バイナリーを提供してくれました。私たちはなるべく早く mozilla.org でのこれらのバイナリーの提供と、加えて Mozilla Thunderbird の GTK2 以外の Linux バージョンの生成を行いたいと考えています。

Windows: http://ftp.mozilla.org/pub/thunderbird/releases/0.1/thunderbird-0.1-win32.zip

Linux（要 GTK2 と XFT）: http://ftp.mozilla.org/pub/thunderbird/releases/0.1/thunderbird-0.1-i686-pc-linux-gtk2-gnu.tar.bz2

Mac OS X: http://ftp.mozilla.org/pub/thunderbird/releases/0.1/thunderbird-0.1-macosx.dmg.gz

提供されたビルド（これらは正式なビルドではなく、おそらく mozilla.org のビルドとは違った設定がなされています。）

現在のところ無し：
Thunderbird が成熟していくにつれて、将来のマイルストーンに渡ってこのリストが大きくなるものと私たちは期待しています。

ローカライズされたビルド。MLP のボランティアたちによって提供されたもの：

  • 簡体字中国語版 (中国)：Linux GTK2 tar.gz, Win32 zip.
  • 繁体字中国語版 (台湾)： Win32 full installer.
  • ドイツ語版： Linux GTK2 tar.bz2, Mac OS X stuffed dmg, Win32 zip.
  • フランス語版： Win32 zip.

このバイナリーをダウンロードしたら、Winzip か gzip を使って空のディレクトリに展開し、thunderbird.exe を起動してください。Thunderbird は専用のプロファイルを作成し、Mozilla seamonkey のプロファイルとは干渉しません。

Mozilla Thunderbird をアンインストールするにはただ単にその Mozilla Thunderbird のディレクトリを削除してください。こうしておくと、将来 Mozilla Thunderbird をインストールして使用する際に利用できるプロファイルを残しておくことができます。もし Mozilla Thunderbird のプロファイル（日 本語版）も取 り除きたいのであれば、Windows の場合は Application Data ディレクトリから Thunderbird フォルダーを削除し、Linux の場合はホームディレクトリから .thunderbird ディレクトリを取り除いてください。

既知の問題

以下のリストは、Mozilla Thunderbird 0.1 での既知の問題を取り上げています。新しいバグを報告する前に、このリストを読んで、定期的に確認してください。（このリリースでの新しいバグが見つか り 次第このリストを更新していく予定です）これが正真正銘の最初のマイルストーンリリースなので、目前に課題が山積になっていることはよく知っています！同 時に、 Thunderbird にわくわくしていますし、進化の過程が今のレベルまで来たことを皆さんと分かち合いたいと思います。

  • デフォルトのテーマがまだ完全には実装されていません。メール３ペインとメール作成ウインドウのためのツールバーアイコンを追 加しただけです。まだ Mozilla クラシックのテーマアイコンをたくさん使っています。オプションダイアログのためのアイコンも持っていません。Arvid がデザインしてくれており、これから先改良がされていく予定です。
  • Linux のオペレーティングシステムとの統合がまだ済んでいません。メールメッセージ内の http url をクリックしてもデフォルトの Linux ブラウザを立ち上げることができません。この問題は次のリリースで解決したいと考えています。この問題が Linux ユーザーにとってどれだけ重要な問題かは理解しています。(^^)
  • 英語以外の辞書を追加するには、イン ストールの方法を読んでください。
  • メール作成ウインドウでアドレスを削除したり入力したりしようとすると、アドレスフィールドが固まってアドレスを入力できなくなる可能性があ ります。マウスを使っ てメッセージ本文の部分にフォーカスを移動し、それからまたアドレスフィールドをクリックすれば、この問題を回避することができます。
  • Tools / Options / Extensions では、Extension のインストールボタンが途中で切れてしまい、全部表示されない場合があります。
  • Tools / Options / Fonts では、フォントサイズのドロップダウンメニューが、実際のフォントサイズではなく、「...」と表示されます。それでもサイズを選択し、正しいサイズで表 示することができます。
  • 受信／送信メッセージのための言語設定は、Tools / Options / Fonts で行うことができます。
  • 送信済みメッセージからのアドレスは、自動的にアドレス帳へと取り込まれます。収集アドレス帳として使われているアドレス帳を変更するための UI は、現在ありません。回避策を 参照してください。
  • プロキシを通した HTTP および SSL 設定はできません。原因究明に関する助力を求 めます。
  • メッセージの削除やゴミ箱を空にする操作で、メッセージがあった場所の read コラムヘッダーに「点」が付いた空白のスレッドペインが表示されたままになることがあります。他のフォルダをクリックして該当のフォルダに戻れば、取り 除くことができます。
  • 学 習機能つきジャンクメールフィルタを有効にするには、アカウントごとの設定（Tools / Junk Mail Controls / Adaptive Filters）で行ってください。ただし、UI だけを見ると全体設定のように見えてしまうので注意してください。複数のアカウントでフィルターを有効にするには、Settings タブをクリックして、アカウントメニューリストから設定したいアカウントを選択して、Adaptive Filter パネルに戻り、自動ジャンクメール検知機能の有効／無効を切り替えてください。
  • SMIME 設定は Tools / Account Settings / Security を通じて設定することができます。このパネルは全体が表示されない場合があるとの報告があります。現時点でこの問題を回避するには、このダイアログのサイ ズを変更してください。
  • メール作成画面のコンタクトサイドバーは、メール作成ウインドウにアドレスを追加するように マウスをクリックしても正常に動作しないことがあります。
  • テスタの中には、Thunderbird はメール３ペインのウインドウサイズを記憶できない、またはツールバーのカスタマイズ設定を記憶できない、という現象を報告してきた人がいます。これが 発生した場合、thunderbird¥defaults¥profile¥localstore.rdf をプロファイル（日本語版） 内に コピーしてください。
  • 作業中は、メールの３ペイン内でのプログレスバーは表示されません。常に隠れている状態のままです。
  • 現時点では Mozilla プロファイルを Thunderbird に移行するようにはなっていません。Thunderbird を試してみたい場合は、新しいプロファイルを使用してメールアカウントの情報を入力してください。Mozilla プロファイルを手動で（日本語版）移 行することもできますが、私たちとしては新しいクリーンなプロファイルの使用をすすめます。

ここでは取り上げていない問題、FAQ、ヒントやコツ、さらに一般的な Mozilla Thunderbird ヘルプについては、Mozilla Thunderbird Help（日本語版）と MozillaZine によってホストされている Mozilla Thunderbird フォーラム を参照してください。

Mozilla を評価している政府およびその他団体のための情報

  1. 第 508 条に対する Mozilla の対応度

    第 508 条は、合衆国連邦機関がどのソフトウェアアプリケーションを使用できるかを判断する目安となるアクセシビリティ要件の一式です。

  2. Mozilla の W3C User Agent Guidelines 評価レポート

    W3C の WAI は一方、UAAG と呼ばれる重要なアクセシビリティ標準を推進しています。これらのガイドラインは、第 508 条のはるか先をいっており、Mozilla はこれらの要件を満たすまでにまだまだ多くの作業をしなければなりません。

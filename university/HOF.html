Mozilla 殿堂

このページの XML 版 もあります。

これらは、Mozilla プラットフォームや Mozilla ブラウザ、ソースコード、あるいは、アプリケーションその他の製品開発に Mozilla の開発ツールを利用しているベンダーの一部です。これらのベンダーは以下の 3 種類の一般的なカテゴリに分類されます。このカテゴリの中でも、Mozilla 関連製品に最近アップデートをおこなったベンダーが、最初にリストされています。

  • Mozilla アプリケーションスイート
  • Mozilla/Gecko 組み込みアプリケーション
  • Bugzilla やその他の開発ツール
MozNGW
MozNGW は、Novell GroupWise サーバコンポーネント用の Mozilla ベースのクライアントです。
Epiphany
Epiphany は、GNOME デスクトップ用 Web ブラウザです。シンプルかつ標準に準拠したブラウザを目指しています。
FireCast
WireSpring Technologies による FireCast は、対話型キオスク (訳注: 街頭、駅、空港、店舗などに設置された、タッチパネル操作の情報端末) のネットワークを展開、遠隔管理するための、Linux ベースのプラットフォームです。Linux ベースのキオスク運営システムである FireCast OS は、クライアントへのコンテンツ表示から GUL 管理ツールの提供まで、あらゆることに組み込みの Mozilla ブラウザを利用しています。
Silicon Graphics
Silicon Graphics は、自社の Irix Operating System のリリースで Mozilla をビルド、ディストリビュートしています。
Conectiva
Conectiva は、ラテンアメリカを対象とした GNU/Linux ディストリビューションです。このディストリビューションは Mozilla と Netscape 7 をバンドルしています。
UnitedLinux
UnitedLinux は、エンタープライズ市場を対象とし、Mozilla を含んだ GNU/Linux ディストリビューションです。以下の創立企業 4 社のリソースを結集しています: Conectiva、The SCO Group、SuSE Linux AG、TurboLinux, Inc.
Netscape 7
Netscape 7.1 [日本語版] は、Mozilla 1.4 をベースとした、最新の Netscape による商用ディストリビューションで、Windows、Mac OS、Linux 版がリリースされています。ユーザインターフェイスは、開発者ではなくエンドユーザ向けに最適化されており、AIM やスペルチェッカーなどの追加機能が含まれています。
PTC
PTC は、賞を取った Web 利用が可能な CADCAM 製品 PRO/Engineer Wildfire に Gecko を利用しています。
Borland
Mozilla は、Borland の Kylix2 IDE 内部の HTML プレビュー機能を提供しています。
mozdev.org
Mozdev は、拡張機能、アドオン、Mozilla アプリケーションを提供している 100 以上のプロジェクトをホストしています。
Xabyl
Xabyl は、XBL ファイルを視覚的に編集するためのエディタです。XUL を使って実装されており、Mozilla にインストールできます。
ManyOne Networks
ManyOne Networks は、「最も遅いインターネット接続環境でも、インターネットを探索するための、より気取らない、直感的、教育的な楽しい方法」となる、Mozilla ベースの ManyOne Universal Browser をリリースしました。
topstyle
Topstyle は、Topstyle Pro HTML/CSS/XHTML エディタに、レンダリングエンジンとして Gecko を採用しています。
AOL
AOL for Mac OS X は Gecko ベースの Web ブラウザを統合しています。
GEtax
スイスのジュネーブ税務署、GEtax (米国内国税歳入局の地方版に相当) は、納税申告 Web アプリケーションへのアクセスと、収入申告フォームへ記入を支援するために、地方の納税者全員に CD を郵送しています。40 万人に出荷されている CD には、Windows、Linux、Mac OS X 向けのフランス語版 Mozilla 1.2.1 が含まれています。
HP/Compaq
HP-UX と IPF プラットフォーム向けの初めての Mozilla が ダウンロード可能に なっています。この他にも、HP は、Tru64 UNIX 向けに Mozilla Web Client を、OpenVMS 向けに Mozilla を提供しており、自社のプリンタ支援製品では Netscape 7.0/Mozilla 1.0.1 の組み込み版を利用しています。
Camino
Camino は、Mac OS X プラットフォームにおける最高のブラウザを供給することを目的としたプロジェクトです。Camino は、Cocoa フロントエンドと結び付ける UNIX バックエンドから構成されている Fizzilla の異種、Cocoazilla を利用しています。
OEOne
OEOne は、使いやすく、多くのアプリケーションが用意されている、「Linux と Mozilla ブラウザのパワーを結集した」オペレーティング環境を制作しています。
Nokia
Nokia は、インターネットとデジタル放送を結合した、家庭向けのパワフルな娯楽情報番組センターである Media Terminal の一部に Mozilla を利用しています。
IBM Web Browser
IBM Web Browser は、スペルチェッカーと Flash プラグインを追加した、OS/2 向けの Mozilla ポートです。IBM は、AIX 向けの Mozilla ブラウザもリリースしています。
また、INotes Web ベースメール製品 にも Mozilla ブラウザを利用しています。
Red Hat Linux
Red Hat Linux は、Mozilla を含む、様々なアプリケーションをパッケージにした、Linux OS のディストリビューションです。
Compuserve
Compuserve は、Compuserve クライアントの バージョン 7.0 に Gecko を組み込んでいます。4 月のリリースを取り上げたニュースが ZDNet [日本語訳] と news.com にあります。
Sun Netscape 7
Sun Netscape 7 は、Netscape からリリースされている、最新のインターネットブラウズ、メール、インスタントメッセージソフトの Solaris ポートです。
Komodo
Komodo は、ActiveState による、スクリプト言語のためのクロスプラットフォームな開発環境です。
Linspire
Linspire (以前は LindowsOS と呼ばれていました) は、Walmart などの大規模小売店で PC に搭載されて販売されている、使いやすい、Windows に似た OS で、Mozilla ブラウザを売り物にしています。
(訳注: 株式会社ライブドアから発売されている LindowsOS 日本語版 では、一般ユーザーのアカウントで「ウェブブラウザ」を起動すると、和ジラプロジェクト のページがデフォルトのホームページになっているブラウザが起動します。しかし、Lindows 日本語版に同梱されているのは、Mozilla をベースとした Lindows Internet Suite という独自のブラウザであって、和ジラではありません。作成にあたって和ジラが参考にされ、その名残がデフォルトのホームページとして残っているだけだと思われます。誤解のないようにお願いします)
Ximian GNOME
Ximian GNOME は、Linux/Unix ユーザのための、完全な使いやすいデスクトップ環境です。Ximian GNOME は、ユーザの生産性を高め、コミュニケーションを促進し、ソフトウェアや情報管理を簡単にする、強固なアプリケーションスイートを提供します。Ximian GNOME には Mozilla ブラウザが含まれており、Red Hat、Linux Mandrake、SuSE、Debian、TurboLinux など、幅広いディストリビューションやプラットフォームをサポートしています。
Debian
Debian は、Linux カーネル (OS のコア) を利用したフリーの OS ですが、基本的な OS ツールの大半は GNU プロジェクトから生まれており、そのために名前が GNU/Linux となっています。Debian GNU/Linux は、純粋な OS よりも多くのものを提供します。Debian GNU/Linux には、マシンにインストールしやすい形式でバンドルされている、あらかじめコンパイルされたソフトを含む 3950 以上のパッケージが付属しています。それらのアプリケーションの 1 つが Mozilla です。
Fabula
Fabula は、ヨーロッパ内で少数言語が使われている地域の小学校における、バイリンガルな学習と読み書き能力のサポートを目的としたプロジェクトです。Fabula は、バイリンガルなマルチメディア童話ブックを作成し、情報をやりとりするための、使いやすい一連のソフトを開発し、教師や生徒が、2 種類の言語で書かれたテキストや、インタラクティブな画像、音声、画面上の言葉リスト、言葉遊びを組み合わせて、リッチで面白い教材を生み出せるようにしています。
Galeon
Galeon は、Gecko (Mozilla のレンダリングエンジン) をベースとした GNOME Web ブラウザです。高速で、手軽なインターフェイスが特長です。
Nautilus
Nautilus シェルは、Linux 上のファイル管理、Web ブラウズ、システム管理を統合した、GNOME 用の画期的なソフト環境です。
Beonex
Beonex は、エンドユーザ - 個人や、小規模から大規模の企業のいずれも - を対象としたインターネットクライアントソフトと、それに付随するサービスです。
Nvu
Nvu は、Mozilla ベースの Linux と Windows 用 HTML エディタです。主要スポンサーである Linspire, Inc. の支援を受けて、Daniel Glazman が開発しています。Nvu は、FTP サイトマネージャや (Mozilla Conmposer では拡張機能として利用可能な) CaScadeS という CSS エディタを統合した、パワフルな Web サイトオーサリングツールです。
DocZilla
DocZilla Browser は、XML/SGML パーサ、DTD パーサ、リンクマネージャ、CALS Table サポートコンポーネントを含む、Mozilla ベースブラウザのためのコンポーネントアドオンです。

さらに、bugzilla.org には、バグ追跡システム Bugzilla を利用している 200 以上の企業、組織、プロジェクトの 独自のリスト があります。

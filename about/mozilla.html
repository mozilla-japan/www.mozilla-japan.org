Mozilla について

Mozilla プロジェクトの ミッション は、インターネットでの選択肢と革新を維持していくことです。

私たちは：

  • 数々の賞を勝ち取った Firefox Web ブラウザおよび Thunderbird 電子メールソフトウェアの 制作者 であり提供者です。
  • ソフトウェア開発ツールの 提供者 です。
  • 開発者やテスタによるオープンソース コミュニティ です。
  • テクノロジー業界の パートナー です。
  • インターネットの標準化や Web 標準の開発に向けたツールを提供する 活動支援団体 です。
  • インターネットテクノロジーやソフトウェア開発についての教育資料を提供する 教育団体 です。

次世代の Web ブラウザと電子メールソフトウェアを開発する、時代の先端をいく革新的な団体です

「Firefox Web ブラウザのセキュリティ、洗練された機能は MicrosoftxA霏の IE を凌ぐ」 ― Walt Mossberg, Wall Street Journal

  • Mozilla Firefox:：私たちの次世代のブラウザは、ブラウジング体験を簡単に、そしてスピーディなものにします。
  • Thunderbird：私たちの次世代スタンドアローン電子メール・ニュースグループクライアントで、Mozilla Firefox のパートナーとして、また単独でも利用できます。 軽く、俊敏で、洗練された迷惑メールフィルタを備えており、Thunderbird は電子メールクライアントの新しい基準を打ち立てます。

ソフトウェア開発ツールの提供者です

私たちが自分たちのプロジェクトを開発し管理するために作り上げたツールは、何百ものオープンソースプロジェクトを後押しています。

  • Bugzilla：Mozilla バグ追跡 Web ツールは、アプリケーションに影響するバグの修正や機能の実装の進捗状況を追跡するために役立ちます。
  • Bonsai： Web インターフェースで、日付ごと、ファイルごと、開発者ごとのチェックインログを確認することができます。
  • Tinderbox： 常時稼動している私たちのビルド・テスト Web ツールで、Mozilla アプリケーションスイートを複数のプラットフォーム上で絶え間なくビルド、テスト、状況報告し、あらゆる時点でのアプリケーションの状態を表示してくれます。

開発者やテスタによるオープンソース・コミュニティです

  • 1998年に Netscape によって立ち上げられた Mozilla プロジェクトは、オープンソース運動にとって決定的な動きでした。私たちは、オープソース開発のリーダーでありパートナーであり続け、テクノロジーに低コストな代替選択肢を提供してきました。そして何百万人というユーザーと共に、私たちは、オープンソースソフトウェア最大のディストリビュータの一員となったのです。
  • 私たちは、ニュースグループ、メーリングリスト、IRC チャンネル、そして Web サイトを通じて交流している大規模な 開発コミュニティ を擁しています。私たちは すべての人を歓迎します。
  • 私たちの開発作業は、皆さんの目の前で行われます。私たちのテクノロジーがどこに向かっているのかを、開発ロードマップ でご覧ください。
  • 私たちはオープンソース開発の原則を信じ、実践しています：素早く、頻繁にリリースを行い、ユーザーの声に耳を傾け、そして 十分な数の目玉があれば、バグは深刻ではない のです。

テクノロジー業界のパートナーです

  • Mozilla テクノロジーを利用し、配布し、開発をサポートしてくれている、たくさんの会社や組織に参加できます。Mozilla のパートナーには、IBM、Sun Microsystems、Hewlett Packard、Red Hat そして Linspire といった各社が参加しています。多数の大学やフランス農業省をはじめとして、世界中の大手企業や大規模組織が Mozilla へと切り替えています。
  • 私たちはさまざまな情報を提供し、Mozilla をベースにした製品を開発するお手伝いをします：
    • 柔軟な オープンソース ライセンス条件。
    • 組み込み開発のための情報。
    • 設定およびカスタマイゼーション情報。
    • クライアント設定キット。
    • XUL Planet - XUL（XML User Interface Language）および Mozilla XPCOM（Cross-Platform Component Object Model）に関する情報サイト。
    • フットプリントの小さなデバイス に Mozilla を展開する新しいプロジェクト。
  • Mozilla とパートナーを組み、組織に Mozilla を展開する準備が整いましたか？　私たちの パートナーページ をご覧ください。

インターネットの標準化やウェブ標準の開発に向けたツールを提供する活動支援団体です

  • Mozilla は インターネット標準を最高の形で実装 します。
  • Venkman：JavaScript で、ブレークポイント、条件付ブレークポイント、ローカル変数監視、変数監視、シングルステップ、エラー停止、プロファイルデータ収集、レポート生成、コード整形（整形表示）などをサポートしています。
  • DOM Inspector：アプリケーションの実行中に、ユーザインターフェースの構造やウィジェットを監視し編集するためのツール。

インターネットテクノロジーやソフトウェア開発についての教育資料を提供する教育団体です

  • 私たちは、ソフトウェア開発者、Web 開発者 そして エンドユーザ のための資料を整えてきました。
  • 私たちは、Mozilla テクノロジーについての理解を深めるためのプロジェクト を開始しました。このプロジェクトは、さまざまな状況に応じて配布したり再利用したりすることができるような教材という形をとっています。

Rhino をどのようにして使っているの？

ここでは、人々が Rhino をどのようにして使っているのか、その一部を紹介します。最初のリストは Netscape 社のマーケティング部によって集められました。それで、Rhino は Netscape 社の Java ベースな JavaScript と呼ばれています。私達は、貴方が Rhino をどのようにして使っているのか、聞ける事が大好きです。ここにメール Norris Boyd をしてください

Attachmate
「 Java 実行環境を含む Netscape JavaScript 1.5 は MacroRecorder を開発するのに完璧な解決法でした。なぜなら、Rhino は私達の開発プロセスを、より早く、より良くしてくれました。そして、私達の顧客は、より能率的で、より信頼できる、スタンダードな製品を得ることができました。」と、Attachmate の製品開発ディレクター、Rob Clark さんは言っています。Attachmate は Netscape の Java ベースな JavaScript 1.5 インタープリターを100% ピュアな Java で、 e-Vantage Viewers と呼ばれる、ウェブ・ホスト型で小型軽量なクライアントに統合しまています。Netscape の Java ベースな JavaScript 1.5 インタープリターは MacroRecorder の特徴として ブラウザー・ベースのユーザーが効果的にメイン・フレームや中型システム上のアプリケーションを効果的に使えるように誘導します。

Bristow Hill Software
「私達は、多くの仕事とスクリプト能力が Bristow Hill サーバー・ページに必要と思っていた。しかし、Java 実行環境を含む Netscape の JavaScript 1.5 が、僅か二、三行のコードの初期設定と、 私達のスタンダード・オブジェクトをエクスポートする為に名前を変えた一行のコード変更で、適合してしまうことを発見し歓喜させられた。また、埋め込まれたスクリプトを引き出し、そのスクリプトを速いスピードで直接 Java クラスにコンパイルできることも、私達を大いに喜ばした。Netscape の JavaScript エンジンは石のようにしっかりした、スタンダードに準拠している。私のたった一つ悔しく思う事は、私達がそれをすぐに使わなかったことだった。」と、Bristow Hill Software の取締役社長である Don Anderson さんは言った。

ICEsoft
ICEsoft は JavaScript のサポートを彼らの Rhino を使った ICE Browser に追加した。

JShell
Rhino は Patrick Beard さんに書かれたオープン・ソース・コマンド・シェル、JShell にスクリプト言語として使われている。

Softcom
Netscape の Java ベースな JavaScript 1.5 と Softcom の Java ベースな RealPlayer プラグイン（RJ）との密接な統合は、Softcom を、メディアやエンターテインメントの為のダイナミックで双方向性なビデオ・アプリケーション、小売りやプロフェッショナルな教育関係の顧客、ウェブ上の完全な双方向性とストリーミング・ビデオでの電子商取引との同期を迅速に生産する事を可能にした。と、Softcom の取締役社長で最高執行責任者の Chris O'Brien さんは言いました。アカデミー賞の授賞式中、オスカー賞の為に作成された、E! Online で、Softcom は Netscape の JavaScript 1.5 を RealPlayer に埋め込む為に RJ を使い、双方向性なチャット、Java ゲーム、ビデオのストリーミングに成功した。

Technology Deployment International
「 Technology Deployment International は 、私達の eBusiness Management System (eBMS) のワーク・フロウ・モジュール（仕事の流れ）が、顧客のビジネス原理を、彼らのアプリケーションのどのようなワーク･ステップにも取り込む事ができるように、 Java ベースな Netscape の JavaScript エンジン を選択しました。」と、Technology Deployment International の eBMS Development で取締役副社長、Kelvin Liu 博士は言っています。「それは、簡単に組み込むこめるし、エンジニア・チームから受け取ったサポートは抜群であるし、JavaScript コードの性能は殆ど Java に等しい。

XYPOINT
XYPOINT は、彼らのサービス WebWirelessNow で使われている Java クラスを 自動的にテストする為に、Rhino を使いました。Abraham Backus さんは、 Rhino を使える事は幸せで、「私はいつでもこの様な JavaScript サポートが欲しかったのだ」と、言いました。

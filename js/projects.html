JavaScript エンジンを使ったプロジェクト

SwigJS
SwigJS は、SWIG への拡張として作られ、 既存の C/C++ API に対して JavaScript のサポートを提供しようとするプログラマを 助けるツールである。SwigJS は、簡単なインターフェースファイルあるいは C/C++ のヘッダを解析し、 JavaScript のスクリプトが API を使うための必要なラッパー関数を作る。 SwingJS の最初のテストは もう一つの gtk ユーザーインターフェースツールキットへの スクリプトバインディング、JSGtk の作成を助ける事となる。 詳しい事は、 SWIG のホームページ を見るか、 Josh Gough にメールを送ってください。

iPlanet Web Servers
iPlanet Netscape Enterprise Server 4.0 は Sun-Netscape Alliance 製の 最新ウェブサーバーである。mozilla.org による JavaScript 1.4 と LiveConnect 3 エンジンを Server Side JavaScript (SSJS) と LiveWire のデータベース接続性を サポートするために使っている。 ウェブサーバーチームと mozilla.org の JavaScript 開発者の密接な強力の製かである。 iPlanet NES 4.0 は、JavaScript のプログラムをコンパイルし格納するために はじめて JavaScript XDR フォーマットを使った製品として知られている。 詳しい事については、Netscape Enterprise Server のドキュメンテーションを読むか、 Viswanath Ramachandran にメールを送ってください。

ICE Browser
ICE Browser 5 は、Java コンポーネントで、最新のウェブブラウジング標準を実装し、 Mozilla との忠実な互換性を提供している。ICE Browser は Rhino をその JavaScript スクリプトエンジンとして用いている。 ICEsoft の Storm アーキテクチャは、あらゆるコンテントハンドラをあらゆるスクリプト言語と 結び付けるフックを提供し、 Scripter インターフェースを Rhino に提供している。これによって ICE Browser を含んでいるあらゆるコンポーネントが JavaScript エンジンへ完全にアクセスできる。素晴らしい副次的効果として、 LiveConnect もコンポーネントやプラグインによって公開されたすべての Java API の JavaScript からの 透過スクリプティングによって得られる。

Web Crossing
Web Crossing は、SpiderMonkey をサーバー側のスクリプティング環境として入れている。 対話式のコンテンツとオンラインコミュニティに合わせて、 完全なウェブサーバー上でのスクリプティング環境を提供している。 このサーバーを実行することで、組み込みオブジェクト指向データベースに格納される 永続的なオブジェクトを作成出来る。 だからサーバー環境を希望通りに拡張出来る。

WebMerger
WebMerger は完全な JavaScript のサポートを含んだサーバーサイドの HTML スクリプトエンジンだ。Socket、Filesystem、ODBC/Database、SMTP、Server、 Session、Application、Request、そして Response を含むオブジェクトが使える。 その環境はトランザクション、スクロール可能なカーソル、Prepared SQL、他にもたくさん のものを完全にサポートしている。Windows 95/98、NT、2000、Linux、 そして FreeBSD のバージョンがある。 WebMerger は特にウェブ開発を目標とする安全な解決方法だ。

K-3D
K-3D は C++ による 3D モデリング・アニメーションシステムだ。 これは、カスタムスクリプティング (ユーザーが、歯車のような複雑な反復形状を作るために スクリプトを作る事のできるもの) を提供するため、および対話式チュートリアルシステム (ユーザーが、再生、マウス・ポインタを移動、メニューを選択、ボタンをクリックなどを行ったときに、 JS スクリプトのプログラムとのやり取りを記録できる) を動かすために、 Spidermonkey JavaScript エンジンを組み込んでいる。 K-3D は GPL 化されていて、クロスプラットフォーム (Linux/Unix & Win32) で、 Pixar Renderman Interface による映画レベルの品質のレンダリングを サポートしている。

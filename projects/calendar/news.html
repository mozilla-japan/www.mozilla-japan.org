2005年2月4日（金）：
Sunbird 開発チームはここに、初の公式リリースを発表いたします。そのリリースとは、 Windows 、 Linux 、 Mac OS X に対応した Sunbird 0.2 であります。様々なプラットフォームに対応する実行ファイルを、ダウンロードページにて取得できます。
2005年1月11日（火）：
Sunbird 開発チームはここに、 Windows 、 Linux 、 Mac OS X に対応する、来るべき Sunbird 0.2 リリースに向けてのリリース候補第二版を発表いたします。様々なプラットフォームに対応する実行ファイルを、ダウンロードページにて取得できます。

RC1 と RC2 間の変更差分一覧はこちらにて参照できます。
2004年12月22日（水）：
Sunbird 開発チームはここに、 Windows 、 Linux 、 Mac OS X に対応する、来るべき Sunbird 0.2 リリースに向けてのリリース候補第一版を発表いたします。様々なプラットフォームに対応する実行ファイルを、 ftp.mozilla.org にて取得できます。

よろしく！
2004年12月22日（水）：
Mozilla 開発陣の一員である Mike Shaver は、 "Lightning project" を公表しました。"Lightning" とは、カレンダーの機能（予定、仕事等）を Thunderbird へより緊密に統合するための拡張機能の作業プロジェクト名です。より詳しい情報は、 wiki.mozilla.org にある、こちらのページを参照してください。

Lightning project の最初のリリースは 2005 年中盤を予定しています。Lightning は現在進められている Thunderbird 向け Calendar 拡張機能とは異なる開発プロジェクトであり、またそれと置き換わるようなものではありません。現在の拡張機能が Thunderbird とのゆるやかな統合のみを目標としているのに対し、Lightning は Thunderbird のメインユーザインターフェイスとユーザとの対話モデルへのできる限り緊密な統合を目標としています。

Lightning project により、 Mozilla Thunderbird はメール機能とカレンダー機能という一連の機能を持つことになり、特に企業向け用途において Outlook との競争力がより大きなものとなります。
2004年11月12日（金）：
Sunbird 開発チームはここに、 Windows 、 Linux 、 Mac OS X に対応する、新たなビルドを発表いたします。これらのビルドはダウンロードページにて取得できます。

これらのビルドは Sunbird 0.2 beta と名付けられました。これはつまり、大きな障害が見つからなければ、このビルドが来るべき Sunbird 0.2 リリースの基礎になるということです。

前回の最終ビルド以降、数多くの変更が加えられています。こちらのページにその一覧があります。
2004年11月7日（日）：

Linux 、 Mac 及び Windows 上で動く Firefox 、 Thunderbird と Mozilla Application Suite 向けのカレンダー拡張機能の新たな XPI が公開されました。ダウンロードページにてお使いの環境に合った XPI をダウンロードできます。

これらの XPI は、数多くのバグの修正を含んでいます。

2004年9月10日（金）：

Sunbird の新たなビルドが公開されました。今回のビルドの特徴は、 Sunbird のユーザインターフェイスに数多くの改良が行われたことです。

  • 標準のテーマ向けの新たなアイコン類
  • ツールバーカスタマイズ機能の改良（小さいアイコンモード、テキスト表示のみのモード、アイコン表示のみのモードがそれぞれ追加されました）
  • そのほかにも、数多くの改良とバグ修正が行われています。
2004年9月4日（土）：
O'REILLY OSDir の人たちが、非常に公平でバランスの取れた視点から、最新の一般向け Sunbird ビルドのレビューを公表しました。数カ所細かい間違いがありますが、ユーザの方々に十分お勧めできるものです。みてね！

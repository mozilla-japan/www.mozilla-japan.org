ライブラリ

Mozilla のコードが用いる多くの様々なプロトコルに関係する標準、 RFC、draft、そして FAQ のすべてを このページに列挙するつもりである。

適切な文書を探し始める上で良い出発点が ここにある:

  • RFC (at ISI)
  • Draft (at IETF)
  • FAQ (at faqs.org)
  • Mail 標準 (at IMC)
  • News 標準 (at IMC)
  • W3C 勧告
  • ネットワーキング標準
  • MIME type のリスト
  • HTML 4.0 リファレンス
  • CSS1 リファレンス
  • Web Standards Project のリソース

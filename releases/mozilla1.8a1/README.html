Mozilla 1.8 Alpha1 新着情報

このドキュメントには、Mozilla 1.8 Alpha1 で実装された新機能や重要なバグの修正、その他新たな問題の一部が載っています。1.8 Alpha1 で行われた修正の詳細なリストは「大まかな変更ログ」をご覧ください。

新機能と修正

ブラウザ

  • ヘルプウィンドウにキーボードショートカットが追加されました。また、Linux 版インストーラでも、キーボードによる操作性が向上しました。
  • ページ内検索機能で、本文だけでなく、フォームのテキストエリアやテキスト入力ボックスに含まれるテキストも検索できるようになりました。
  • 基本的な FTP アップロード機能のユーザインターフェイスが追加されました。
  • Intellimouse Explorer の戻る／進むボタンが Linux でも使えるようになりました。
  • 「他のタブを閉じる」を選択すると、タブを複数開いているときにブラウザウィンドウを閉じようとした場合と同じように、確認ダイアログが表示されるようになりました。

メール

  • 複数 ID サポートのユーザインターフェイスが実装されました。
  • 迷惑メールフィルタの精度がさらに向上しました。
  • 受信した迷惑メールを自動的に既読とするオプションが追加されました。
  • 個人用アドレス帳で、アドレスごとに、メッセージを送信する際に指定した形式 (プレーンテキストまたは HTML) が自動的に記憶されるようになりました。
  • メールウィンドウのメニューが調整され、より使いやすくなりました。また、ウィンドウのサイズを変更しても、フォルダリストの幅が変わらなくなりました。
  • ニュースアカウントでは、メッセージリストに含まれるサイズの表示を (隠し設定を通じて) KB から行数に変更できるようになりました。
  • サーバ全体のニュースフィルタが実装されました。
  • アドレス帳のオートコンプリート (自動補完) 機能が改良されました。
  • メールに複数のファイルが添付されていた場合、それらが正しく表示されなくなる原因となっていた、リストボックスに関する問題が修正されました。また、メッセージを並べ替えたときに、選択されたメッセージが常にリストビュー内に表示されるようになりました。

バックエンド

  • CSS2 の quotes (引用符) プロパティをサポートしました。
  • XUL のツリー部品に大幅な変更が加えられています。詳しくは こちらのメッセージ を参照してください。
  • 保存できる Cookie の数が格段に増え、それでもまだ足りないという人のために (Cookie の最大保存数を指定する) 隠し設定も追加されました。

新たな問題

ここには、前回のマイルストーン以降 既知の問題 のページに追加された事項が載っていますが、バグ自体は前からあった可能性もあります。

一般

  • Java を使用する際によく見られるクラッシュ (Bug 240105) は、Java のバージョンを 1.4.2 にアップグレードすることで解決できます。

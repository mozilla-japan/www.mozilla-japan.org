Webclient 2.0 Alpha 3 リリースノート

著者: Ed Burns <edburns@acm.org>

Ed に全ての変更案を送ってください。直接リリースノートを編集しないでください。

2005年1月19日に最終更新された、これらのリリースノートにはインストールの説明と webclient 2.0 Alpha 3 リリースに関連する既知の問題が記述されています。これらのリリースノートは私たちがフィードバックを受け取った時に更新されるので、随時新しい情報がないか確認してください。

Webclient 2.0 Alpha 3 を動作させるには Mozilla 1.7 がインストールされていなければなりません。

Webclient に参加する方法

  • どんなものであれ、見つけたバグを報告してください。バグが既に報告されているかどうかを調べるのに検索フォームを使ってください。もしそれを見つけられなかったら、Bugzilla という mozilla.org のバグ追跡システムに行き、「Before reporting a bug（バグを報告する前に）」の手順に従ってください。webclient 用の bugzilla における Component は「Java APIs to WebShell」です。
  • netscape.public.mozilla.java と netscape.public.mozilla.embedding ニュースグループでの議論に参加してください。
  • webclient ホームページ上の貢献する方法についての節を読んでください。

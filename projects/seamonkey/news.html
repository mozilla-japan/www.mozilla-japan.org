SeaMonkey プロジェクト > ニュース

2006 年 4 月 13 日

SeaMonkey 1.0.1 が出ました！

SeaMonkey 1.0.1 が 入手できるようになりました。 SeaMonkey の以前のバージョンの全ユーザにアップデートを推奨します。 SeaMonkey 1.0.1 は、クリティカルなバグの修正に加え、複数のセキュリティに関する修正を含んでいます。詳しくは リリースノート をご参照ください。

2006 年 1 月 30 日

SeaMonkey 1.0 リリース

SeaMonkey Council はここに、SeaMonkey 1.0 をアナウンスします。インターネットスイート初のエンドユーザ向けリリースです。このオープンソースアプリケーションは、mozilla.org がホストする Web サイトから 無料でダウンロード でき、先進の Web ブラウザ、強力な電子メールクライアント、WYSIWYG の Web ページ作成機能、そして機能豊富な IRC チャットクライアントが同梱されています。Web 開発者の皆さんに向けては、mozilla.org の DOM インスペクタや JavaScript デバッガツールも含まれています。SeaMoneky 1.0 は、今日利用可能なインターネットソフトウェアパッケージのなかでも、もっとも機能が充実した、パワフルで安全なスイートのひとつです。

SeaMonkey は、先代の Mozilla Application Suite や Netscape Communicator のユーザにお馴染みのルック・アンド・フィール 【訳注：外見と操作感】 を備えていますが、数々の 新機能 を追加し、バックエンドに変更を加えて、セキュリティや安定性、パフォーマンスを増しています。注目の機能は、ドラッグ・アンド・ドロップで並び替えができるタブ、電子メールのフィッシング検出機能、複数アカウントを管理するためのグローバルボックス、Scalable Vector Graphics （SVG） のサポートが挙げられます。

SeaMonkey プロジェクトはコミュニティをベースにしたプロジェクトで、Mozilla Foundation がスイート製品の開発中断を発表した際に、Mozilla のスイートコードベースを中核にして立ち上げられ、mozilla.org でホストされています。この新しいプロジェクトは、このスイートを存続させ、さらにモダンで完成されたインターネットソフトウェアパッケージへと開発することを目的としています。

2005 年 12 月 19 日

SeaMonkey 1.0 Beta

SeaMonkey Council は本日、開発者向けの新しいリリース、SeaMonkey 1.0 Beta を発表します。9 月に発表された Alpha 版と比較して、SeaMonkey 1.0 Beta には、タブのドラッグ・アンド・ドロップといった新機能だけでなく、今月初旬に披露された 新しい SeaMonkey ロゴ を使ってはじめてブランド化されたリリースでもあります。コアコードの大部分は、人気の Firefox 1.5 と共有していますが、SeaMonkey はその先代である Mozilla Application Suite や Netscape Communicator パッケージとよく似たルック・アンド・フィール 【訳注：外見と操作感】 を備えています。

SeaMonkey 1.0 Beta が備えるのは、技術の粋を凝らした Web ブラウザだけではありません。強力な電子メールクライアントや WYSIWYG 形式の Web ページ作成機能、豊富な機能を備える IRC チャットアプリケーションも含まれています。また Web 開発者に向けては、mozilla.org の DOM インスペクタや JavaScript デバッガツールも同梱されています。SeaMonkey 1.0 Beta は、まだテスト段階にあるとはいえ、現在手にすることができる最も強力で安全なインターネットソフトウェアパッケージのひとつです。

このパブリックベータリリースには、2006 年 1 月に予定されている SeaMonkey 1.0 のファイナルリリースで予定されているすべての機能が装備されています。非常に多くのバグ修正を行ったため、先の Alpha リリースと比較すると、さらに安定性が向上し、使いやすくなっているはずです。また、SeaMonkey Council からはユーザの皆さんにお願いがあります：来る 1.0 リリースが機能面で最高のインターネットスイートであるだけでなく、品質やセキュリティ、安定性といった面でも最高のものとなるように、製品をテストして、何か問題点があれば報告してください。

SeaMonkey プロジェクトはコミュニティをベースにしたプロジェクトで、Mozilla Foundation がスイート製品の開発中断を発表した際に、Mozilla のスイートコードベースを中核にして立ち上げられ、mozilla.org でホストされています。この新しいプロジェクトは、かつてのスイートを 「SeaMonkey」 という名前の下に存続させ、さらにモダンで完成されたインターネットソフトウェアパッケージへと開発することを目的としています。

2005 年 12 月 2 日

SeaMonkey のための新しいロゴ

数ヶ月にわたる募集および選定プロセスを経て、SeaMonkey Council はここに、プロジェクトおよびアプリケーションのための新しいロゴを発表します。プロジェクトが初めて ロゴの提供を呼びかけて 以来、197 件を超えるアイデアが Mozilla および SeaMonkey コミュニティのメンバーから寄せられました。これら 197 件のロゴという豊富な選択肢のなかから、Council は幾多にもわたるディスカッションを交わしながらロゴを選定しました。

新しいアートワーク は、中央に 「Sea Monkey」 を思い起こさせる抽象的な意匠が明るいシアン / ブルー系の色で形取られ、暗い青で囲まれた円の上に浮かんでいるように見えます。これは Alex Butin によってデザインされたもので、プロフェッショナルで洗練された印象だけでなく、モダンなコンピュータデスクトップのアイコンとして親和性が高く、また小さなサイズでも簡単に見分けがつきます。さらに、この見た目は mozilla.org ファミリー （Firefox、Thunderbird、Camino、Sunbird） ともピッタリです。また、柔らかな形と色使いから、とても洗練された印象を与えてくれますし、波のイメージから、インターネットスイートアプリケーションのダイナミズムや、新しいプロジェクトからの息吹を感じ取ってもらえると思います。

このアートワークを同梱する初めてのテストリリースは SeaMonkey 1.0 Beta となる予定で、これから数週間以内にリリースの予定です。また SeaMonkey 1.0 のファイナルリリースもそのすぐ後に控えています。開発バージョン、いわゆる 「ナイトリービルド」 には、ここ数日中にこの新しいアートワークが組み込まれる予定です。

2005 年 9 月 15 日

SeaMonkey 1.0 Alpha

SeaMonkey Council はプロジェクト初のリリース、SeaMonkey 1.0 Alpha を公開しました。先にリリースしました Mozilla Application Suite のコードベースを基に開発されました SeaMonkey 1.0 は、 Mozilla suite の最新のバージョンに比較して、多くの新機能、グレードアップ、バグ修正を達成しています。内部的には、中心的コードの多くは、現在の Firefox 1.5 Beta 1 ブラウザと共有されており、外観的には、Mozilla と Netscape の長年のユーザが、好んで慣れ親しんできたルック & フィール を取り入れています。

しかし SeaMonkey 1.0 Alpha は単なる最新の Web ブラウザにとどまりません。アプリケーションは、強力な E メールクライアントだけでなく、WYSIWYG な Web ページコンポーザと、機能豊富な IRC チャットを備えています。 Web 開発者向けには、mozilla.org の DOM インスペクタと JavaScript のデバッガツールも備えています。テスト用限定のリリースとはいえ、SeaMonkey 1.0 Alpha は現在利用可能な最も強力かつ安全なインターネットソフトウェアパッケージとなっています。

SeaMonkey 1.0 Alpha は公式な SeaMonkey のロゴを決めていません。SeaMonkey プロジェクトは、現在も有志達からロゴの提案を公募中です。新ロゴはこれら公募の中から選定され、SeaMonkey 1.0 Beta に取り込まれますが、これは、今年の後半に出荷される SeaMonkey 1.0 の直前のバージョンになります。　

SeaMonkey プロジェクトは、 Mozilla Foundation がパッケージ製品の開発の継続の中止を発表した時に Mozilla のパッケージのコードベースに関連して名乗りを上げた mozilla.org の有志のグループを拠り所としたプロジェクトです。新プロジェクトは、このパッケージ製品を ”SeaMonkey” と命名して継続させるとともに、もっと斬新で完全なインターネットソフトウェアパッケージへ進展させる事に専念しています

2005 年 7 月 28 日

あなたの SeaMonkey ロゴを送ろう!

プロジェクトは有志が創る新しい作品を求めています

現在の SeaMonkey アプリケーションの試験的なナイトリービルドや今回の SeaMonkey 1.0 Alpha のリリースが比較的素朴なロゴを使用していますが、プロジェクトは将来に向けてもっと完成度の高い作品を必要としています。新ロゴが、有志の提案の中から選定されますので、グラフィックアートの専門家であればだれでも支援をお願いしたい次第です。応募基準や応募方法についての情報は、SeaMonkey アートワークページをご参照ください。

2005 年 7 月 2日

SeaMonkey プロジェクトがインターネットパッケージを継承

以前の "Mozilla Application Suite" のコードは独立グループの新しい開発に出会う

Mozilla Foundation が Mozilla としてそれまで長く知られてきたアプリケーションの "移行プラン"を公表した時、この動きには Firefox や Thunderbird という主要な新商品に再度焦点を当てる事だけでなく、新しいパッケージコードの開発を独立グループへ移行する事も含まれていました。新しいプロジェクトの設立と、新しいプロジェクト名 "SeaMonkey" の採用をここに宣言するのは、このグループすなわち SeaMonkey Council です。

SeaMonkey の名称は、有志の提案による名前候補の長いリストの中から、考慮を重ねて、プロジェクトをリードするグループ（通称 "SeaMonkey Council"）によって採用されました。やはり、長く使われてきた内部コード名がプロジェクトに一番ふさわしいとされました。大いに迷った末の決定でしたが、ある種の説得性を備えています。というのは、パッケージは開発者有志の中で、すでに SeaMonkey として知られていましたので。Council は、 Mozilla Foundation の支援を得て、名称を商標化するための手続きを進めています。

Mozilla Foundation は新しい SeaMonkey アプリケーションを Mozilla suite のユーザに対する公式にサポートされたグレードアップ製品と見なしていませんが、プロジェクトの不可欠なインフラという形で、新設のプロジェクトに対して支援を提供し、SeaMonkey の開発者に協力しています。SeaMonkey の開発によって Mozilla のコードベースに戻って、役に立つ多くの改善がもたらされ、しかもそれらのほとんどは、Firefox や Thunderbird を含む製品に共有されています。

グループは、数週間の内に、初回リリースである SeaMonkey 1.0 の Alpha 版を出荷する計画です。 Mozilla 1.7.x や Mozilla 1.8 Alpha や Beta のユーザ達は、使い慣れたアプリケーションの新しい改善版に出会うことになります。Netscape 7.x のユーザもまた、新しい SeaMonkey パッケージに親しみを感じるでしょう。

SeaMonkey は、最新鋭の Web ブラウザと機能豊富なメール、ニューズグループクライアントだけでなく、簡単で、強力な HTML エディタ、 Web 開発ツールや IRC チャットアプリケーションをも一緒に備えています。

プロジェクトは現在、 Web では mozilla.org のアドレス http://www.mozilla-japan.org/projects/seamonkey/ で参照でき、プロジェクトをリードするチーム (SeaMonkey Council) には、E メールで連絡可能です seamonkey-council@mozilla.org。

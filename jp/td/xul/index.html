XUL Tutorial 和訳プロジェクト

このプロジェクトは Mozilla Japan 翻訳部門 のサブプロジェクトであり、XULPlanet で公開されている文書の和訳を公開しています。和訳作業の対象としているのは現在のところ XUL Tutorial のみとなっていますが、将来的には順次他の文書の和訳も用意して公開していく予定です。

現在は XUL Tutorial の和訳もまだ作業途中でありますが、草稿段階のものでよろしければ XULPlanet.jp の XUL Tutorial からたどれますのでご利用ください。和訳作業に協力してくださる方や間違いのご指摘など、何かありましたら Mozilla Japan 翻訳部門フォーラム の XulPlanet-JP フォーラム までよろしくお願いします。

なお、このプロジェクトの草稿をもとにしてアップデートを加えた文書が、MDC (Mozilla Developer Center) の XUL Tutorial - MDC で公開されています。

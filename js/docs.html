JavaScript のドキュメンテーション

以下のリンクは組み込み担当者やスクリプト製作者に関係のありそうなもので、 進行中のクライアントでの JS のアーキテクチャに関する情報に関係する仕様を 含んでいる。
ECMA
ECMA は JavaScript の標準化作業を務めている。 その結果が、ECMA-262 と ISO-16262 である。
  • ECMA-262 (PDF)
  • ECMAv2 の 草案 (PDF)
  • ECMAScript の識別子における Unicode の使用拡大に関する 提案。これは、例えば、識別子に中国語を 使えるようにする事だ。
API のドキュメンテーション
API のドキュメンテーションは PDF 形式 と HTML で利用できる。
File オブジェクト
JavaScript の File オブジェクト に関する 現在作業中の文書。File オブジェクトはファイルと (将来的には) パイプへアクセスできるようにする。 Norris Boyd にコメントを送ってください。
LiveConnect
LiveConnect は Java<->JavaScript 通信技術で、最初 Navigator 3.0 の一部として出てきた。コアエンジンは現在 VM 独立で JNI ベースの LiveConnect システム を含んでいる。LiveConnect の将来のバージョン (``LC3'') についての計画を立てている。LiveConnect の文書 がいくつか集められている。
PerlConnect
PerlConnect は JS と Perl を一緒に結びつける LiveConnect に似た接続道具 (glue) である。 ソースはドキュメンテーションを含んで js/src/perlconnect にある。
Mike Shaver

Mozilla のリリースに関する良くある質問

  1. なぜ Mozilla には様々なバージョンがあるのですか? どれを選べばいいのでしょうか?
  2. Mozilla の新しいバージョンをインストールするには?
  3. Mozilla のパッチはどこでダウンロードできますか?
  4. Mozilla を CD に焼いて友人や同僚に再配布しても構いませんか?
1. なぜ Mozilla には様々なバージョンがあるのですか? どれを選べばいいのでしょうか?

Mozilla は速い開発ペースに従っており、そのリリースは 開発ロードマップ に基づいて作成されています。2004 年現在、Mozilla は四半期ごとのスケジュールでリリースされています。

mozilla.org は、新バージョンをリリースする前に、2 回かそれ以上の中間リリースを公開します。これらの中間リリース、つまりテストリリースは、アルファ版、ベータ版、あるいはリリース候補 (RC) と呼ばれています。これらのビルドはテスト目的に限って公開されるもので、mozilla.org はユーザサポートを提供しません。あなたがテスターあるいは開発者でない限り、ダウンロードするのは最終版だけにしておくべきでしょう。

2. Mozilla の新しいバージョンをインストールするには?

まず最初に、お使いの OS に関するリリースノートの説明に従って Mozilla をアンインストールし、それから新しいバージョンをインストールしてください。絶対に古いバージョンの Mozilla に新しいバージョンを上書きしてはいけません。

プラグインフォルダが Mozilla のアプリケーションフォルダの中にある場合は、再インストールしたあとで復元できるよう、前もって別の場所に保存しておくと良いでしょう。プラグインフォルダはあなたのプロファイルにも保存できます。

プロファイルをデフォルトの場所に保存している場合、Mozilla をアンインストールしてもプロファイルはそのまま残り、ユーザ設定は失われません。ただし、一部のアドオンが使えなくなってしまうため、それらは再インストールする必要があります。

3. Mozilla のパッチはどこでダウンロードできますか?

Mozilla の最終リリースは、安定性、安全性が高められており、通常アップデートの必要はありません。(実際、Mozilla 1.0 は、リリース後 1 年で 2 回のメンテナンスリリースを出すにとどまりました)

mozilla.org はパッチをリリースしていません。新機能が必要な場合は、アドオン をインストールするか、新バージョンが出るのを待ってください。

4. Mozilla を CD に焼いて友人や同僚に再配布しても構いませんか?

はい! Mozilla は自分以外の周りの人たちに自由に配布できます。その際は必ず オープンソースライセンス規約 をお読みください。

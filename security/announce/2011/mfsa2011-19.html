Mozilla Foundation セキュリティアドバイザリ 2011-19

タイトル: 様々なメモリ安全性の問題 (rv:3.0/1.9.2.18)
重要度: 最高
公開日: 2011/06/21
報告者: Mozilla の開発者とコミュニティ
影響を受ける製品: Firefox、Thunderbird

修正済みのバージョン: Firefox 5
  Firefox 3.6.18
  Thunderbird 3.1.11

概要

Firefox やその他の Mozilla ベースの製品に搭載されているブラウザエンジンに含まれていたメモリ安全性に関するいくつかのバグを、Mozilla の開発者が特定、修正しました。これらの問題の一部は、特定状況下でメモリ破壊の形跡が見られ、条件が整えば任意のコード実行に悪用可能と思われるものでした。

参考資料

Firefox 5 と Firefox 3.6.18 で修正されたメモリ安全性の問題が、Bob Clary、Kevin Brosnan、Nils、Gary Kwong、Jesse Ruderman、Christian Biesinger によって報告されました。

  • メモリ安全性の問題 - Firefox 5、Firefox 3.6
  • CVE-2011-2374

Firefox 5 のみで修正されたメモリ安全性の問題が、Bas Schouten、Igor Bukanov、Jesse Ruderman、Bill McCloskey、Olli Pettay、Gary Kwong、Daniel Veditz、Marcia Knous によって報告されました。

  • メモリ安全性の問題 - Firefox 5
  • CVE-2011-2375

Firefox 3.6.18 のみで修正されたメモリ安全性の問題が、Luke Wagner、Gary Kwong によって報告されました。

  • メモリ安全性の問題 - Firefox 3.6
  • CVE-2011-2376

Firefox 3.6 のみに影響していたクラッシュが、Rh0 氏によって報告されました。

  • https://bugzilla.mozilla.org/show_bug.cgi?id=651990
  • CVE-2011-2364

Firefox 3.6 のみに影響していたクラッシュが、secenv 氏によって報告されました。

  • https://bugzilla.mozilla.org/show_bug.cgi?id=655742
  • CVE-2011-2365

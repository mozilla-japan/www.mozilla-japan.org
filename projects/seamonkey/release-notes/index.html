古いマイルストーンリリース

最新の Mozilla 0.x バイナリについては、新しい リリース のページを見てください。

これらは Seamonkey プロジェクトのマイルストーンリリース(Milestone Releases)のためのダウンロードパッケージ およびリリースノートへのリンクです。 すべてのマイルストーンビルドが同時にリリースされるとは限りません。 したがってあなたが捜しているビルドがここにまだ無くても パニックにならないでください。 また後でチェックしてください。 ソースコードがまだリリースされていない場合、現在のマイルストーン・ディレクトリーの README ファイルの中から CVSから得る方法を得て下さい。

Full Circle が有効なビルドでは Netscape へクラッシュを分析するためにクラッシュデータの送信を可能とします。 また netscape.public.mozilla.crash-data ニュースグループへクラッシュ情報をポストします。

もしもあなたがビルドが提供されていないプラットフォームのためにビルドを提供したいならば、私たちの 配布文書を読んで下さい。

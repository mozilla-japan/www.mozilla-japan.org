Rhino 用例

ここでは、どのように JavaScript エンジンを制御して、スクリプト可能なホスト・オブジェクトを実装するかを示します。 すべての例は、mozilla/js/rhino/examples の cvs ツリーにあります。

サンプルスクリプト

unique.js スクリプトは、 ファイルから一意的な (ユニークな) 行をプリントします。

liveConnect.js スクリプトは、 (Java-to-JavaScript 接続性である) LiveConnect のサンプルを示します。

jsdoc.js スクリプトは、 Java での javadoc の類似物です。 それは、正規表現を頻繁に使用しています。

checkParam.js スクリプトは、 Java ドキュメントのコメント欄の @param タグが、Java メソッドの対応するパラメーターと一致するかどうか チェックするのに 有用なツールです。

enum.js スクリプトは、 JavaScript オブジェクトを使い、Java インターフェースを実装するのに JavaAdapter を利用した、よいサンプルです。

NervousText.js スクリプトは、 jsc を使用することで Java のクラスにコンパイルされた JavaScript を使用する、 名高い NervousText applet の JavaScript での実装です。 HTML ページ NervousText.html で走らせることができます。

JavaScript エンジンの制御

RunScript クラス

RunScript.java は、 コマンドラインで与えたスクリプトを実行する、単純なプログラムです。

Control クラス

Control.java は、 単純なスクリプトを実行し、そしてその結果を扱うプログラムです。

JavaScript Shell

Shell.java は、 JavaScript プログラムを実行するためのプログラムです。 それは tools パッケージの中のシェルの簡易版です。 プログラムは、コマンドラインからファイルを指定するか、あるいはシェル起動中に対話形式でタイプします。

PrimitiveWrapFactory

PrimitiveWrapFactory.java は、 WrapFactory の例です。 Java メソッド呼び出し時の、Rhino エンジンの ラッピングの動作を制御するのに、それを使用することができます。 【訳注: コンテキストで、setWrapFactory の引数に指定することで有効になる はず】

マルチスレッドなスクリプトを実行

DynamicScopes.java は、 一つのグローバル・スコープ・オブジェクトを作成して、複数のスレッドを超えて、それを共有するプログラムです。 グローバル・スコープを共有すると、スレッドを超えて情報共有ができたり、 一度の高くつく操作を実行することで、Context.initStandardObjects のコストの清算ができます。

ホスト・オブジェクトの実装

【訳注: ホスト・オブジェクトとは、オブジェクトの内で Nativeオブジェクトではないもの】
まず最初に、チュートリアルを試したことがないのなら、 調べてみてください。

Foo クラス — ScriptableObject を拡張 (extend)

Foo.java は、 簡単な JavaScript ホスト・オブジェクトです。 作用 (アクション) を伴うプロパティー、可変個引数 (可変個数引数) のメソッド、を含んでいます。

Matrix クラス — Scriptable を実装 (implement)

Matrix.java は、 Scriptable インターフェースを実装することによって、簡単な多次元配列を提供します。

File クラス — 上級な例

File.java は、 JavaScript でファイルを読み書きする手段を提供するために、ScriptableObject を拡張 (extend) しています。 ホスト・オブジェクト定義の、より複雑な例です。

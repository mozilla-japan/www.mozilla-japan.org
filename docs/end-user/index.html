エンドユーザ資料

このページでは、Mozilla や Mozilla をベースとしたブラウザのユーザを対象とした資料へのリンクを紹介します。

  • Mozilla リリース・リリースノート - リリースノートには、インストール方法、システム必要条件、新機能、既知の問題のリストが載っています。
  • Mozilla 1.7 スタートページ - FAQ （良くある質問） やトラブルシューティングガイドなど、初めて Mozilla を使う人のための役に立つ情報があります。
  • Mozilla スタートガイド - Microsoft Internet Explorer や Netscape Communicator といった他のプログラムからの移行に関する情報など、初めて Mozilla を使う人のためのインストールガイドです。
  • Gunnar の Unofficial Mozilla Help Page - このサイトには、よくある質問、一般的なトラブルシューティングガイド、そして Mozilla の様々なオプション、設定、機能を解説したチュートリアルやマニュアルがあります。
  • PluginDoc - ヘルパー・アプリケーション（例：Flash や Java）やのインストール方法や、使い方、ダウンロードマネージャの使い方を解説したヘルプ・サイト
  • Mozilla User Guide [PDF] - （ディーキン大学）
  • Introduction to Mozilla - A Manual for First Time Users [PDF]

Mozilla の機能の使い方やカスタマイズ方法:

  • Mozilla カスタムキーワード
  • インターネット・キーワード - 入力したホストが見つからなかった場合、Mozilla はサーバに助けを求めることができます。
  • Mozilla のカスタマイズ - ユーザの好みに合わせてカスタマイズできる、いくつかのあまり文書化されていない Mozilla の機能に関するヒント集です。
  • ドメイン推測 - Mozilla は、「hostname」 が見つからなかった場合、「www.hostname.com」 を試します。
  • 設定可能なセキュリティポリシー - ユーザは、インターネットのサイトごとに異なるセキュリティポリシーをブラウザに設定することができます。
  • コマンドラインオプション - コマンドライン引数のリストと、その機能について。
  • Mozilla の隠し設定 - Mozilla には、設定ファイルを手で編集することによってのみ適用できる設定項目がたくさんあります。ここにはそれらの設定の一部が載っています。
  • Mozilla のプロファイルディレクトリにあるファイル
  • Mozilla キーボードショートカット - Mozilla で、マウスの代わりにキーボードを使ってできる操作の一覧です。
  • ヘルパーアプリケーション - Mozilla のヘルパーアプリケーションがどのように機能するのか、それらを使ってできることを解説します。
  • ユーザスタイルの変更 （Squarefree.com）
  • 設定についての簡単な説明
  • Mozilla の隠し設定 （Pratik Solanki）

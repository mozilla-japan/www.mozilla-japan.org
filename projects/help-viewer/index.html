Mozilla Help Viewer Project

Mozilla ヘルプシステムプロジェクトページへようこそ。このページは Mozilla の基本的なヘルプビューアの開発や、それを Mozilla エンドユーザがヘルプのブラウザとして使うときのことについて説明しています。 そのビューアには簡単なコンテキストセンシティブヘルプを表示するための設備を含んでおり、動的なトピックスを読み込むために RDF を使ったり、HTML 文書を処理するためのナビゲーション機能を備えています。

Firefox ヘルプは 0.9 リリース時から Firefox の一部です。 Thunderbird ヘルプのベータバージョンはインストーレーションページか ら入手できます。 （ノート　それは最近の Thunderbird のバージョンではインストールしないでしょう。）

ヘルプシステムの特徴：　

  • 文章検索内蔵
  • インターネットのヘルプ文章の検索をサポート
  • ツールバーのカスタマイズ
  • Mozilla Seamonkey, Firefox,そして Thunderbird ためのコンテンツ

協力することに興味がおありですか？

もっと協力することについて案内を得るために　How You Can Help 　を見てください。もしもあなたがコードや文章の段階で手伝いたくなくても、ナイトリービルドのヘルプのテストを手伝うことはできます。コーディングでな い側で、私たちは現在、手伝いが必要です：

  • ベータテスターと全体的な QA の仕事
  • 文章を書くこと、 特に Thunderbird と Mozilla Suite 。

Mozilla 1.4 の新機能

Mozilla の新リリースは、より良く、より高速になっています。ユーザインターフェイスは、アクセシビリティを向上させ、OS ネイティブのデザインに合わせるために洗練されました。また、多くの新機能も追加され、インターネットにおけるユーザ経験が高められています。以下のリストは、Mozilla 1.0 以降の主要な機能のアップデートを示したものです。

一般

  • Mozilla 1.4.1 に新しい スペルチェッカー が搭載されました。
  • ツールバー の表示を、テキスト、アイコン、その両方のいずれかに切り替えられるようになりました。
  • Mozilla の実行時に プロファイルを動的に切り替える ことが可能になりました。
  • 異なるコンポーネントに 別々のウィンドウアイコン が設定されました。
  • Windows では、オフラインの状態で Mozilla を起動した場合、自動的にダイヤルアップ接続を開始できるようになりました。
  • Linux 版に フルスクリーンモード が付きました。
  • 隠し設定: アドレスバーに "about:config" と入力して、設定ダイアログからはアクセスできない、高度な設定項目を変更できるようになりました。
  • Linux では、ファイルピッカーのフィルタリングが改良され、新規ディレクトリ作成ボタンが付きました。
  • Mac OS X では、Quartz レンダリングを利用するようになりました。

Mozilla Navigator (Web ブラウザ)

  • DHTML レイアウトのパフォーマンスが改良されました。
  • 新しい ポップアップマネージャ を使って、サイト単位で迷惑なポップアップウィンドウをブロックできるようになりました。
  • 新しい ダウンロードマネージャ で複数のダウンロードを管理しやすくなりました。
  • Find As You Type: ブラウザウィンドウ内に直接検索語を入力して Web ページ内のテキストやリンクを検索できるようになりました。
  • 自動画像リサイズ機能: ブラウザウィンドウに大きな画像を読み込んだ場合、ウィンドウの大きさに合わせて画像がリサイズされます。画像の上でクリックすると、元のフルサイズと自動サイズを切り替えられます。
  • ヘルパーアプリケーションの改良: PDF や Flash などのメディアの処理方法 (例えば、開くか、ディスクに保存するか) を変更できるようになりました。
  • Navigator の起動時や、新しいタブ、新しいウィンドウを開く場合に、どのページを読み込むかを個別に選択できるようになりました。
  • アドレスバーを通じた検索がデフォルトの検索エンジンを利用するようになりました。
  • アドレスバーのオートコンプリート (自動補完) が、より賢く、完成度が向上しました。
  • ブックマークグループをホームページとして設定可能になりました。
  • 選択範囲のソース表示: 選択範囲の上でクリックし、コンテキストメニューからその部分の HTML ソースを表示できるようになりました。
  • ページ情報 パネルの「一般」「メディア」タブの内容が改良され、より多くのページ情報を表示するようになりました。
  • ブックマークの改良: ブックマークにルート階層フォルダが付いたほか、同一の URL を指すブックマークにそれぞれ異なる名前を設定できるようになりました。また、ブックマークマネージャとブックマークサイドバーにサイトアイコンが表示され、ブックマークマネージャには簡易検索バーが追加されました。セパレータ (区切り線) にラベルを設定することも可能になりました。
  • スムーズスクロール が実装されました。(デフォルトでは無効になっています)
  • Windows では、「戻る」「進む」マウスボタン をサポートしました。
  • Microsoft Internet Explorer の XML デフォルト表示に似た XML プリティプリント が利用可能になりました。
  • Mac OS 版で MathML が有効になりました。(Windows と Linux では既に利用可能になっています)
  • 双方向 アラビア語、ヘブライ語 サポートが改良されました。
  • XBM 画像の表示をサポートしました。

Mozilla Messenger (メール & ニュースグループ) とアドレス帳

Mozilla Messenger 1.4 は、高度なジャンクメールフィルタシステム、カスタマイズ可能なメールビュー、メッセージフィルタの改良によって、より便利になりました。以下に Mozilla Messenger の主な機能向上をいくつか挙げます。

  • Messenger に、メールコントロールの履歴をベースとして、迷惑メールを自動的に排除する、賢い ジャンクメールコントロール が実装されました。
  • メールビュー をカスタマイズして、例えば 5 日以内に届いたメールや知人からのメールといったように、ある基準に合ったメッセージを表示できるようになりました。
  • セキュリティとプライバシーの改善によって、メールメッセージ内の 画像やプラグインをブロックする ことが可能になりました。また、HTML メールをプレーンテキストとして表示させるよう設定できます。
  • フィルタ が大幅に改善しました。ニュースグループへのフィルタの適用や、ダウンロード済みメールに対するフィルタの実行も可能になりました。フィルタのログ保存機能も実装されています。
  • メッセージの検索結果ウィンドウからのドラッグ & ドロップが可能になりました。
  • Windows ユーザは Palm とのアドレス帳の同期を利用できるようになりました。
  • Windows では、添付ファイルダイアログを通じて、メールに 複数の添付ファイル を追加できるようになりました。
  • メッセージのヘッダからテキストを選択、コピーできるようになりました。
  • オプションメニューの オリジナルテキストを引用する を使って、メール編集ウィンドウで現在のメッセージを引用できるようになりました。
  • IMAP 割り当てをサポートしました。
  • "プロキシを利用しない" オプションが IP アドレスのワイルドカードをサポートしました。
  • yEnc デコードをサポートしました。
  • ブラウザ設定とは別に、メールのフォント表示をカスタマイズ可能になりました。

Composer

  • Composer に書式設定なしの貼り付けが可能になりました。
  • 挿入メニューから目次を作成できるようになりました。
  • ダイナミックな画像・テーブルリサイズ: 画像を選択したり、テーブルの中にキャレットを置くと、8 方向のリサイズハンドルが表示され、マウスを使って画像やテーブルをリサイズできるようになります。画像の場合、リサイズはリアルタイムで行われ、リサイズの間、画像の半透明の影が目的のサイズで表示されます。また、ツールチップが、目的のサイズと相対的なサイズをピクセル単位でリアルタイムに表示します。

その他のコンポーネント

  • JavaScript デバッガ が主要な開発サイクルを終了しました。9 種類のビューパレットが特徴的で、これらはメインウィンドウ内で配置を変えたり、個別のフローティングウィンドウにドッキングすることもできます。また、JavaScript で直接、ユーザ定義ビューやコマンドの作成が可能になりました。
  • Chatzilla のタブの完成度が向上し、日本語名のチャンネルにログインできるようになりました。

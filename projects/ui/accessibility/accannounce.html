Netscape、Netscape ブラウザの
業界標準アクセシビリティサポートを発表

カリフォルニア州 マウンテンビュー - 2003 年 3 月 13 日 - America Online の関連子会社である Netscape Communications は、本日、Netscape ブラウザソフトウェアの最新版 Netscape 7.02 に、業界標準のアクセシビリティサポートが追加されたと発表しました。これらの新開発は、America Online と Netscape が進めている、すべてのコンシューマーにとってアクセシブルで使いやすいインターネット体験を提供する取り組みの一環です。

Netscape のアクセシビリティ機能強化は、スクリーンリーダー、スクリーン拡大ソフト、音声入力ソフトの開発者など、支援技術ベンダーによる、Netscape ブラウザと互換性のある製品開発を促進し、障害を持ったコンシューマーに対して、ブラウザソフトにおける斬新な代替製品を提供します。

組み込みサポートには、タブブラウズ、My Sidebar、Web ブラウズ、HTML エディタ、メールクライアント、アドレス帳、インスタントメッセージソフトなど、生産性機能のためのアクセシビリティ機能強化が含まれています。その他重要な点は以下の通りです。

  • Microsoft Active Accessibility (MSAA)* と、ドキュメントオブジェクトモデル (DOM) への読み取りアクセスのサポート
  • キーボードサポートと、Netscape Help におけるキーボードショートカット一覧を含む、完全なキーボードドキュメント
  • ユーザが思い通りの大きさにフォントサイズを変更できる、便利なフォント拡大技術
  • クラシックスキン使用時に Windows プログラムと同じフォントや配色を利用できる Microsoft Windows Display Settings のサポート
  • HTML、CSS、DOM、MathML、XML などの W3C 標準のサポート
  • ユーザスタイルシートやウィンドウ外観の設定を含む HTML レイアウトのコントロール
  • コンシューマーがキーボードを使ってテキストを選択できる、新しい「キャレットモード」ブラウズ

これらのアクセシビリティ機能強化は Mozilla 1.2.1 にも含まれ、Netscape Gecko レンダリングエンジンをベースとしたその他の製品でも利用可能です。

Netscape のアクセシビリティ作業は、Mozilla オープンソースプロジェクトの一部です。現在進められている Netscape アクセシビリティプロジェクトの計画や、Netscape 7.02 ブラウザに含まれる新しい業界標準のアクセシビリティサポートについて、より詳しく知りたい場合は、http://www.mozilla.org/projects/ui/accessibility/ をご覧下さい。Netscape ブラウザは http://channels.netscape.com/ns/browsers/ からダウンロードできます。

* MSAA は、単なる「プラグアンドプレイ」ソリューションではありません。支援技術開発者は、Netscape ブラウザとの最適なパフォーマンスを得るために、自社のソフトウェアコードに変更を実装しなければならない可能性もあります。

テクノロジー・エバンジェリズム

参加するには

あなたがもし、少数の人々によって管理されていない、自由で開かれた Web の現状に危機感を抱いているなら、Mozilla テクノロジー・エバンジェリズムに参加 して、Mozilla その他の Gecko をベースとしたブラウザなど、標準に準拠したブラウザに対する制限を Web からなくすために協力してください。

問題のあるサイトを見つけたら

Web サイトが Mozilla やその仲間のブラウザをサポートするかどうかを決める、唯一の最も重要な要素は、Mozilla に対するサポートの欠如が収益にどう響くか、ということです。Web サイトに 苦情を言うこと は、この世に存在する ブラウザが 1 種類だけではない という事実を Web サイトに気付かせる有効な手段となり得るのです。

ボランティア募集中

サイトを分析し、Mozilla をサポートするための改良方法を提案できる、Web 開発や品質保証の経験をお持ちの方の協力を求めています。今すぐ Mozilla テクノロジー・エバンジェリズムに参加 してください!

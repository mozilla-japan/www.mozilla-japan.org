SeaMonkey 1.0 のインストールガイド

以前のバージョンとの互換性問題が発生しないように、SeaMonkey の インストーラはインストール先のディレクトリが空でない場合、中身を 削除してよいか問い合わせてきます。インストール先のディレクトリに プロファイルや個人的なファイルがある場合は、 "NO" と答えて ください。 "NO" と答えないと、ファイルは削除されます。既定の場所に プロファイルを置いてあるユーザには影響しません。

プロファイルのバックアップについてのヘルプは、プロファイルに関して よくある質問を参照してください。

Linux

システム要件

  • 以下のライブラリ・バージョン（または互換性のあるもの）が GTK1 用のビルド では必要とされます： glibc 2.2.5、 XFree86 3.3.x、 GTK 1.2.x、 Glib 1.2.x 。 Red Hat Linux 7.0 と Debian 3.0 そして SuSE 7.3 （またはそれ以降） のインストールは上手く動作するはずです。推奨されるビルド（GTK2）は glibc 2.3.2、 XFree86 4.3.x、 GTK 2.x、 glib 2.x、 libstdc++-3.3 を必要とします。
  • Intel pentium クラスのプロセッサ（233 MHz もしくはそれ以上を推奨）
  • 64 MB の RAM
  • 100 MB の空きディスク容量（キャッシュ・サイズを削減すれば 必要なスペースは著しく少なくなります）
  • 注記： linux のディストリビューションによっては既定の libstdc++ と互換性のあるバージョンを収めていません。このパッケージをインストールしなければ なりませんが、通常 compat-libstdc++、 compat-libstdc++-33 、 libstdc++5 などと呼ばれています。

インストールの説明

注記： 以前のバージョンの SeaMonkey をアンインストールしておいて から SeaMonkey 1.0 をインストールする方法を推奨します。 SeaMonkey をアンインストールしても、ブックマーク、履歴、 Cookie、その他 プロファイル・ディレクトリに収められている情報は削除されません。

注記： SeaMonkey を少なくとも一度は実行して 自動登録を完了させ、 compreg.dat ファイルをインストールディレクトリに書き込ませておくと 良いでしょう。他の（製品をインストールした人間とは別の）ユーザがプログラムを最初に実行すると、 コア・ダンプが発生してしまいます。（bug 49345）

注記：ここで示す説明には、 bash シェルを使用しています。 bash を使用していない場合には、 bash シェルを使用するか、必要に応じてコマンドを 変更してください。

SeaMonkey インストーラを使ったインストール

SeaMonkey インストーラをダウンロードして SeaMonkey をインストールする場合は、 次に述べる段階を踏んでください：

  1. seamonkey1.0 という名前のディレクトリを 作成します（mkdir seamonkey1.0）。次に、そのディレクトリに移動します。 （cd seamonkey1.0） 。
  2. リンクをクリックしてインストーラのファイル （seamonkey-1.0.en-US.linux-i686.installer.tar.gz と呼ばれています） をダウンロードします。
  3. SeaMonkey のディレクトリに移動して（ cd seamonkey1.0 ）、 以下のコマンドに従って アーカイブを解凍します：
    gunzip -dc sea*.tar.gz | tar -xvf -
    （このようにしてインストーラを SeaMonkey という名前の seamonkey-installer のサブディレクトリに置きます。）
  4. seamonkey-installer ディレクトリに移動して（cd seamonkey-installer）、 ./seamonkey-installer というコマンドを打ってインストーラを実行します。
  5. インストール・ウィザードの指示に従って SeaMonkey をインストールします。
  6. SeaMonkey を起動するには、ディレクトリをインストールした場所に移動して、 ./seamonkey というコマンドで実行します。

.tar.gz アーカイブを使った手作業のインストール

SeaMonkey をインストールするのに .tar.gz アーカイブ・ファイルを使うなら：

  1. seamonkey1.0 という名前のディレクトリを 作成します（mkdir seamonkey1.0）。次に、そのディレクトリに移動します。 （cd seamonkey1.0）
  2. リンクをクリックしてインストール機能のついていない seamonkey*.tar.gz ファイルを seamonkey1.0 ディレクトリにダウンロードします。
  3. SeaMonkey のディレクトリに移動して（cd seamonkey1.0）、 以下のコマンドに従って アーカイブを解凍します：
    gunzip -dc sea*.tar.gz | tar -xvf - （このようにして "seamonkey" サブディレクトリを seamonkey1.0 ディレクトリの下に作成します。）
  4. seamonkey ディレクトリに移動します（cd seamonkey）。
  5. ./seamonkey 動作スクリプトを走らせて、 SeaMonkey を実行します。

マルチユーザの Linux インストール

SeaMonkey を Unix 上で複数ユーザにインストールするには、通常通りインストールします。 次に、以下のスクリプトを SeaMonkey ディレクトリに作成して、実行可能に設定します。 （chmod u+x <scriptname>）そしてそれを mozilla ディレクトリに 書き込み特権を持つユーザ（例えば root）として実行します。このプログラムは su[do] root では実行しないようにしてください。（bug 52653） 

#!/bin/sh
dist_bin=`dirname $0`
MOZILLA_FIVE_HOME=$dist_bin
LD_LIBRARY_PATH=$dist_bin
export MOZILLA_FIVE_HOME LD_LIBRARY_PATH
$dist_bin/regxpcom
$dist_bin/regchrome
touch $dist_bin/chrome/user-skins.rdf $dist_bin/chrome/user-locales.rdf

そうすると、 アクセスを許可されているユーザなら誰でも SeaMonkey のインストーラを実行することができます。

SeaMonkey を GNOME のパネルに追加するには

  1. GNOME Main Menu ボタンをクリックして、 Panel メニューを開きます。それから Panel に sub-menu を追加して Launcher を選択します。
  2. パネルにある SeaMonkey のアイコンを右クリックします。そして以下のコマンドを入力します。

    ディレクトリ名 /seamonkey

    ディレクトリ名は SeaMonkey をダウンロードしたディレクトリの名前です。 例を挙げると、 SeaMonkey がおすすめする既定のディレクトリは /usr/local/seamonkey です。
  3. アイコンの名前を打ち込んで、お望みであればコメントに打ち込みます。
  4. アイコンのボタンをクリックして以下のようにアイコンの場所を打ち込みます：

    ディレクトリ名 /icons/mozicon50.xpm

    ディレクトリ名は SeaMonkey をインストールしたディレクトリです。 例えば、既定のディレクトリは /usr/local/seamonkey/icons/mozicon50.xpm です。

RPM ビルドのインストール

この方法はまだ SeaMonkey には適用できないかもしれません。

Red Hat の RPM ビルドは既定の Red Hat 7.x と Red Hat 8.x のインストールで 動作するために作成されています。これらの RPM を SuSE 上で使用するには gdk-pixbuf パッケージが必要となります。（bug 85218）

アンインストール

  • SeaMonkey をインストールした seamonkey1.0 ディレクトリを削除してください。
  • ~/.mozilla ディレクトリを削除します。その中に プロファイル情報が含まれているのです。

Macintosh

システム要件

  • Mac OS X 10.2 以降
  • PowerPC プロセッサ（266 MHz 以上を推奨）
  • 64 MB RAM （256 MB を推奨）
  • 100 MB の空きディスク容量（キャッシュ・サイズを削減すれば 必要なスペースは著しく少なくなります）

インストールの説明 (Mac OS X)

SeaMonkey のディスクイメージをダウンロードして SeaMonkey をインストールするには、 以下の手順を踏んでください：

  1. seamonkey-1.0.en-US.mac.dmg リンクをクリックして自分のマシンに ダウンロードします。既定では、ファイルはデスクトップにダウンロードされます。
  2. .dmg.gz ファイルをダブルクリックして、 SeaMonkey アイコンをハードディスクにドラッグします。
  3. Applications フォルダにディスクイメージのコピーをとっておくのをおすすめします。
  4. さて、ディスクイメージを取り出します。
  5. お望みなら、 SeaMonkey をドックにドラッグしていつでも使えるようにすることもできます。

アンインストールの説明

  1. /Users/ <ユーザー名> /Library/Mozilla にある アプリケーション・ファイルを削除します。 （しかし Profiles サブフォルダは残しておきます）。
  2. /Users/ 自分のユーザ名 /Library/Preferences から "Mozilla Registry" ファイルを削除します。
  3. インストールした場所（つまり /Applications）から SeaMonkey を削除します。

Windows

システム要件

  • Windows 95*、 98、 98SE、 Windows Me、 Windows NT 3.51、 Windows NT 4.0、 Windows 2000、 Windows XP あるいは Windows 2003
       * DCOM for Windows 95 のついているもの
  • Intel pentium クラスのプロセッサ（233 MHz 以上を推奨）
  • 64 MB RAM
  • 100 MB の空きディスク容量（キャッシュ・サイズを削減すれば 必要なスペースは著しく少なくなります）

特定の ATI ビデオ・ドライバによっては SeaMonkey と互換性がなくて クラッシュすることもあります。繰り返しクラッシュする場合は、 アップグレードして 最新の ATI ドライバにしてみてください。 （bug 101055）

インストールの説明

注記：セットアップを実行する前に全てのプログラムを 終了させるよう強く推奨します。また、ウイルス発見ソフトも一時的に無効としておいた方が 良いでしょう。

SeaMonkey インストーラを使ったインストール

SeaMonkey インストーラをダウンロードして SeaMonkey をインストールする場合は、 これらの手順を踏んでください：

  1. seamonkey-1.0.en-US.win32.installer.exe リンクをクリックしてファイルを自分の マシンにダウンロードします。
  2. ファイルをダウンロードしたところに行って、自分のマシンにある SeaMonkey プログラムのアイコンをダブルクリックして、セットアッププログラムを開始します。
  3. セットアッププログラム内で、画面に表示される指示に従います。 インストール直後に、プログラムは自動起動します。

.zip ファイルを使った手作業によるインストール

SeaMonkey をインストールするのに .zip ファイルをダウンロードして手作業で行うには 以下の手順に従ってください：

  1. seamonkey-1.0.en-US.win32.zip のリンクまたは the seamonkey-win32.zip のリンクをクリックして、 .zip ファイルを自分のマシンにダウンロードしてください。
  2. ファイルをダウンロードしたところに行って、圧縮されたファイルをダブルクリック します。注記：この段階では WinZip の最新版が手元にインストールされていて、 使い方を知っているものとします。そうでない場合には、 www.winzip.com から WinZip を取得してその使い方を覚えてください。
  3. .zip ファイルを C:\Program Files\SeaMonkey1.0 などのディレクトリに解凍します。
  4. SeaMonkey を起動するには、 SeaMonkey を解凍したディレクトリに行って SeaMonkey.exe アイコンをダブルクリックします。

アンインストールの説明

  • SeaMonkey installer を使った場合は、単純に 設定 / コントロールパネル / プログラム の追加と削除 に行って SeaMonkey 1.0 を選択してください。
  • zip ビルドを使った場合は単純に SeaMonkey をインストールした the SeaMonkey1.0 ディレクトリとプロファイル・ディレクトリを削除してください。

プロファイルの場所

典型的なプロファイルの場所は以下の通りです：

Windows

（ログインなしの） Windows 95、 Windows 98 または Windows ME 上では、 プロファイルのディレクトリは
    C:\Windows\Application Data\Mozilla\Profiles\[ プロファイル名 ]\[ 無作為な文字列 ].slt\ となるのが典型的です。

ログインありの Windows 95、 Windows 98 または Windows ME 上では、プロファイル・ディレクトリは
    C:\Windows\Profiles\[ ログイン名 ]\Application Data\Mozilla\Profiles\[ プロファイル名 ]\[ 無作為な文字列 ].slt\ となるのが典型的です。

Windows 2000 または Windows XP 上では、プロファイル・ディレクトリは
    C:\Documents and Settings\[ ログイン名 ]\Application Data\Mozilla\Profiles\[ プロファイル名 ]\[ 無作為な文字列 ].slt\ となるのが典型的です。

注記： Windows 2000 または Windows XP 上では、 Document and Settings ディレクトリが二重に隠されています。そのディレクトリにアクセスするには、 Explorer でシステムディレクトリと隠しファイルを見られるようにしてください。

Mac OS X

プロファイルの場所は、典型的には：
    ~/Library/Mozilla/Profiles/[ プロファイル名 ]/[ 無作為の文字列 ].slt/

Linux

プロファイルの場所は、典型的には：
    ~/.mozilla/[ プロファイル名 ]/[ 無作為な文字列 ].slt/

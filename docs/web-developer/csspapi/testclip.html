クロスブラウザ DHTML 位置指定 API テスト：切り抜き範囲の取得と設定

テストが成功すれば、true が出力されます。失敗すれば false が出力されます。

height 及び width 属性が要素のそれと同じに設定され、背景色プロパティも同一の 1x1 table 中のテキストです。こうすることで、テキストの後ろだけではなく要素領域全体に背景色が確実に描画されます。ここに2つの要素があることに注目してください。緑は後ろ、赤は手前です。赤い要素の clip は上端から 25 ピクセル、右端から 50 ピクセル、下端から 75 ピクセル、左端から 100 ピクセルとなっており、貫通して下の緑の要素が見えることに注目してください。

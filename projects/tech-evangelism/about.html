テクノロジー・エバンジェリズムについて

Mozilla とは

Mozilla は、標準に準拠した Web ブラウザ、メール・ニュースクライアント、IRC クライアント、さらに、クロスプラットフォームなインターネットベースのアプリケーション開発に使える ツールキット を統合した アプリケーションスイート です。Mozilla のツールキットは、Mozilla ブラウザのみならず、Netscape 6、Netscape 7、CompuServe 7、Firefox、Galeon、Camino の基礎となっており、これらすべてに Mozilla の Gecko レンダリングエンジン が組み込まれています。

テクノロジー・エバンジェリズムとは

テクノロジー・エバンジェリズムとは、ごく簡単に言えば、特定のテクノロジーを推進することです。しかし、このような単純な視点で見てしまうと、テクノロジー・エバンジェリズムの目的や重要性が分からなくなってしまいます。

Mozilla を成功に導くには、「為せば成る」といったことわざに頼るわけにはいきません。その代わり、可能な限りあらゆる方法で Mozilla を積極的に推進しなくてはなりません。Mozilla の推進をやめるということ、それは あってほしくない運命 を受け入れることを意味します。

私たちの使命

Mozilla テクノロジー・エバンジェリストとしての私たちの使命は、Mozilla を、世界で いちばんの インターネット・クライアント・テクノロジーとして、積極的に推進することです。私たちは、mozilla.org コミュニティ における活動を通じて、実施例の開発やドキュメントの作成、一般教育の実施、Web サイトの制作者に直接連絡を取って啓蒙するといった自分たちの使命を果たしています。

私たちの使命は容易なものではありません。私たちは、コンピュータの OS から開発ツール、Web ブラウザまで、すべての市場を一企業が独占する世界に直面しています。自分たちの強みを利用し、チームとして共同作業をすることによってのみ、競争に打ち勝つ望みを持つことができるのです。私たちの強みは、プロジェクトの公開性、私たちのテクノロジーの基礎となっている標準、Windows、Macintosh から Linux、Solaris まで、あらゆる OS をサポートできる能力、そして Mozilla が優れたブラウザである という事実です。

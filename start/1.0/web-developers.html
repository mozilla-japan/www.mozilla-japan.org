Mozilla は Web 開発者にとって、スタンダードに準拠した サイトを制作するための大いに最高のプラットフォームです。 Mozilla の画面のいちばん上に組み込まれたツールが、Web サイト、特に複雑でダイナミックなサイトの開発・デバッグを、他の環境とは比べものにならないほど簡単にします。

Web標準普及プロジェクト(もじら組) IEやNN4.x等、過去のブラウザで表示できていたのにMozillaで表示が崩れてしまう、 そんなWebページ作成時のよくある間違いをTipsとしてまとめあげた日本オリジナルのサイトです。 ページの作成時にMozillaのバグかな？と思ったとき、これらの記事で間違ったページを作っていないかを確認することができます。

Venkman - JavaScript デバッガ Venkman は Mozilla をベースとしたブラウザ向けの JavaScript デバッグ環境です。 あなたの OS で実行している C++ コードをデバッグするのとまさに同じ方法で、Web ページの中で実行している JavaScript のコードをデバッグできます。 ブレークポイントを抽出し、スタック検査と変数／オブジェクト検査を呼び出し、任意の JavaScript コードを実行させます。
ホーム | インストール | チュートリアル
DOM Inspector DOM Inspector は、あらゆる Web ドキュメントのライブ DOM の検査・編集に使用できる独自のツールで、ブラウザがページを見るのと同じようにページを見ることができます。 DOM の階層は 2 分割されたウィンドウを使ってナビゲートされ、内部のドキュメントとすべてのノードを様々な異なった視点で表示できます。
ホーム | チュートリアル
JavaScript コンソール 私たちの JavaScript コンソールは、ソースファイルへのハイパーリンクに沿って、ページ内のスクリプトで見つかったあらゆるエラーの厳密な詳細を表示します。 また、任意の JavaScript を入力して実行したり、厳密な Javascript の警告や XSLT の <xsl:message> アウトプットを調べることもできます。

Mozilla のフルインストールを選んだのなら、上の 3 つのツールは［ツール］メニューの［Web 開発］から使うことができます。

Web スタンダード リファレンス Sidebar HTML 4.01、XHTML、DOM 2 を含め、CSS から XSLT まで、あなたが思いつくことのできるほとんどすべての Web テクノロジーに関するスタンダードへのリファレンスが含まれる Sidebar です。 これらの小さな Sidebar が、すぐ手に入るところにその仕様を保存し、あらゆる要素の構文やステータスをすばやく調べることができます。

Mozilla Source Generator スクリプトが実行された後のページのソースを調べたいと思ったことはありませんか？ ぜひ Mozilla Source Generator をチェックしてください。このジェネレータは、現在のページの DOM からフォーマット済み HTML を生成します。

Mozilla と関わっていくには

  • 見つけたバグは Bugzilla Helperを使って報告してください。
    (訳者注: 日本語で報告可能な Bugzilla-ja や、その Bugzilla-ja向けの Bugzilla-ja Heplerがあります。)
  • 既知の問題に関する詳細な内容は、 最新の 修正されたバグ のリストを見てください。
  • それぞれのマイルストーンの機能の状態や注意事項は、milestone page を見てください。
  • Bug Day に参加してください。 もしもあなたが Mozillaのテストを支援したいのだけどもどこからスタートするべきか判らなければ、火曜日の午後3時（太平洋沿岸標準時）に irc.mozilla.org の #mozillazine という IRC チャンネルに参加してください。手順と始める手助けをしてくれる親切な人たちに会う事ができます。
  • BugAThon へ参加して下さい。 あなたがスクリプトを書くことができれば、テストケースを簡単にすることを支援できます。
もっと深く関わりたい場合は、お勧めのスタートポイント（suggested starting points）をチェックして下さい。

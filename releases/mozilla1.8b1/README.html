Mozilla 1.8 Beta1 新着情報

この資料では、Mozilla 1.8 Beta1 に実装された新機能、重要なバグ修正、そして新たな問題をいくつか紹介します。1.8 Beta1 に施された修正のより包括的なリストは、大まかな変更ログ (英語) をご覧ください。

新機能と修正

ブラウザ

  • Mac OS 版ディスクイメージが埋め込み圧縮形式になりました。
  • 各種ブックマークアイテムが中クリックで展開可能になりました。
  • Windows 用の日本語 IME に関する多くの作業が行われました。
  • 背面のタブによってステータスバーの文字列が変更されなくなりました (Bug 104532 の一部)。
  • スクロールバー上のスライダをドラッグした際のスクロールが、逆向きにドラッグしたときに再開されるようになりました (Bug 282195)。
  • [iann] ヘルプビューアの検索が (ブックマークや about:config のように) 自動的に実行されるようになりました (Bug 281563)。
  • [bsmedberg] Windows 版インストーラを使用した場合、-edit 起動オプションが使えるようになりました (Bug 276943)。
  • マウスによる縦横交互・水平スクロールのアクションを指定できるようになりました (Bug 274179)。
  • パワーユーザは、新しいウィンドウをタブで開く設定を自己責任で行えるようになりました (Bug 231984)。
  • [CTho] Windows では、Backspace キーを押したときの動作を変更して、ページを戻る代わりに上方向にスクロールするか、何もしないよう設定することが可能になりました (Bug 219203)。
  • ヘルプビューア内の外部リンクが既存のウィンドウで開かれるようになりました (Bug 226911)。
  • [Mano] ページやテキスト欄の方向を切り替えられるようになりました (Bidi) (Bug 279416)。
  • フォント設定のセリフとサンセリフが言語ごとになりました (Bug 95227)。
  • [bz] JavaScript コンソールで、最後から最初への順番で、長いメッセージが正しく表示されるようになりました (Bug 126620)。
  • 画像をコピーする際、URL とともにピクセルデータもコピーされるようになりました (一部のアプリケーションでは、画像の選択に「形式を選択して貼り付け」を使用する必要があります) (Bug 135300)。

メール

  • [bienvenu] ニュースメッセージが、一時的なネットワーク障害で読み込まれなかった場合、再読み込みできるようになりました (Bug 107776)。
  • [bienvenu] TLS をサポートしている受信サーバへ安全に接続できるようになりました (Bug 60377)。
  • [shaver] WPAD プロキシの 1 形式が使えるようになりました。手動プロキシの詳細設定は独自のダイアログに移されました。DNS サーバとして SOCKS プロキシを使えるようになりました。
  • メッセージが既読にされるまでの時間を簡単に設定できるようになりました (Bug 231428)。
  • [bienvenu] 下書きメッセージの自動保存が可能になりました (Bug 16360/279034)。
  • フラグ付きメッセージを選択できるようになりました (Bug 72823)。
  • [kairo] ニュースグループを読み終わったときにそれらを既読にできる隠し設定が追加されました (bug 99048).

Composer

  • [glazou] HTML 編集時に、Enter キーを使って、簡単に新しい段落を書き始められるようになりました (Bug 92686)。

ChatZilla

  • メールクライアントをインストールしなかった場合でも、ChatZilla のメニューアイテムが正しい位置に表示されるようになりました (Bug 206803)。

バックエンド

  • E4X (http://www.ecma-international.org/publications/files/ECMA-ST/ECMA-357.pdf) が、DOM バインディングマジックを除いて、1.8 Beta2 で実装されます。
  • Bug 280711 の修正によって、Windows で長いブロックの表示が改善されました (Windows 95、98、ME を除く)。
  • Bug 212034 の修正によって、長いテーブルの印刷結果が向上し、より一貫した表示が得られるようになりました (クロスプラットフォーム)。
  • [jshin] Mac 向けに作成された Web ページを認識できるようになりました (Bug 237712)。

新たな問題

ここには、前回のマイルストーン以降 既知の問題 のページに追加された事項が載っていますが、バグ自体は前からあった可能性もあります。

一般

  • メールアドレスの自動補完が正しく動作しない場合があります (Bug 272002 によるリグレッション)。
  • 既定のメールクライアントの設定が正しく動作しない場合があります。

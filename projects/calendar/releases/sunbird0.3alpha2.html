Sunbird 0.3 alpha2 リリースノート

2006 年 5 月 11 日、, the Sunbird/Calendar プロジェクトは Sunbird 0.3 alpha2 をリリースしました。長期にわたるバックエンドコードの書き換え作業が始まって以来、2 番目の正式リリースです。Sunbird 0.3 alpha2 は Sunbird 0.3 に向けてのロードマップにおいて 2 番目のマイルストーンになります。このリリースノートでは、このバージョンにおいてユーザが注意すべきこと、期待すべきでないことを取り上げます。

Sunbird 0.3 alpha2 は Sunbird のダウンロードページ からダウンロードできます。

0.3 alpha2 リリースの目標

Sunbird 0.3 alpha2 は製品品質を備えたリリースではありません（制限事項と既知のバグ を参照してください） 。その代わりに、新しいバックエンドコードの公開やテストのためのリリースという位置付けです。それには以下のようなものが含まれます：

  • イベントタイトルのインライン編集
  • 日毎表示および週毎表示でのドラッグ・アンド・ドロップ編集の復活

Sunbird 0.3 alpha2 では、新しい表示形式の披露とテストに加えて、Sunbird 0.3 alpha1 に見つかった多数のバグを修正することも目的にしています。重要なものとしては以下が挙げられます：

  • 繰り返しイベントの個別要素を変更できる機能
  • インポート・エクスポートでのエラーを少なくすること
  • 適切なエラーレポート
  • アラーム機能の改善
  • 新しく、より直感的に理解しやすい 「新規イベント」 ダイアログ
  • Mac とのより高い親密性
  • その他の多数ある重要なバグ修正

制限事項と既知のバグ

Sunbird 0.3 alpha2 の使用前に、このリリースに存在する制限事項に必ず目を通してください：

  • Mozilla のコアコードに構造的な変更が行われたため、Sunbird 0.3 alpha2 は Windows 95 や Windows 98、Windows ME では動作しません。
  • 他のカレンダープログラムで作成したカレンダーファイルと共に Sunbird を使用した場合、データロスが発生する可能性があります。
  • CalDAV サポートは引き続き非常に限られています。
  • 変更直後 （5 秒以内） に Sunbird をシャットダウンすると、その変更が記録されない場合があります。
  • Sunbird 0.3 alpha2 では、1970 年以前および 2031 年以降の日付では利用できません。
  • スリープ機能 （snooze functionality） は現在装備されていません。
  • 繰り返しイベントのためのアラームが正常に作動しない場合があります。
  • Sunbird 0.3 alpha1 のプロファイルは自動的に Sunbird 0.3 alpha2 のものへとアップグレードされますが、内部の SQLite データベースがアップデートされるため Sunbird 0.3 alpha1 プロファイルとの互換性がなくなります。
  • Sunbird 0.3 alpha2 は、Sunbird 0.2 からのカレンダーファイルを自動的には移行しません。お使いの プロファイルフォルダ から手動でインポートし、リモートカレンダーのファイルを再購読する必要があります。

また、Lightning の 意表をつく挙動 が Sunbird 0.3 alpha2 にもあてはまります。

Sunbird 0.3 alpha2 からのリグレッション

開発が進められていますが、一部の問題は Sunbird 0.3 alpha1 リリース時よりも悪化しています：

  • カレンダー内に重なるイベントがあった場合、同じ日に登録されている他の相反していないイベントも幅が狭く表示されます。
  • 位置・詳細情報はもう日毎表示および週毎表示に表示されなくなりました。
  • 終日イベントは、星アイコンが表示されるのではなく、単純に時間設定が表示されません。
  • 各表示設定では、複数のイベントを選択できません。unifinder では引き続き複数のイベントを選択できます。
  • 同様に、［Edit］ メニューから ［Select All］ をしても動作しません。
  • Sunbird 0.3 alpha2 では、起動時に「Failed to load XPCOM component」 というエラーが JavaScript コンソールに多数報告されますが、機能は一切失われていません。

次のステップ

現在の Sunbird 開発ロードマップでは、このリリースが Sunbird 0.3 に向けた最後のアルファリリースとされています。次に計画されている Sunbird リリースは、最終的な製品品質を備えたリリースを作成するための開発努力にむけて、このリリースで見つかったバグを修正することのみを目標とした Sunbird 0.3 beta です。Sunbird 0.3 beta には新機能を直接盛り込むことは計画されていませんが、並行して進められている Lightning の開発状況によっては、Sunbird の次期リリースにも新機能が盛り込まれる可能性があります。

これらの機能の実装具合を追いかけたいユーザはぜひナイトリービルドをダウンロードしてください。いつもどおり、テストや役に立つバグ報告を大いに歓迎します。

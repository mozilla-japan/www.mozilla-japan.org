SSL プロトコルのバージョン

セキュリティーソケットレイヤー （Secure Sockets Layer： SSL） プロトコルは、Web サイトとブラウザソフトウェアとの両者間でやり取りされる、暗号化された情報の相互認証を管理する際の手順を定義しています。トランスポートレイヤーセキュリティー （Transport Layer Security： TLS） プロトコルは、SSL に基づいた IETF スタンダードで、 TLS 1.0 は SSL 3.1 であると理解されています。

通常、次の三つのチェックボックスを選択し、古いタイプと新しいタイプの Web サーバがブラウザと交信できるようにしてください：

  • SSL version 2 を有効にする： 古いタイプの Web サーバとブラウザが交信できるようにします。
  • SSL version 3 を有効にする： 新しいタイプの Web サーバとブラウザが交信できるようにします。
  • TLS を有効にする： TLS をサポートする Web サーバがその機能を利用できるようにします。

個々の暗号化を有効／向こうにするには、［暗号の編集］ボタンをクリックしてください：

  • 暗号の編集： 何をしたらいいのか確実に分かっている場合か、システム管理者と共に変更を確認しない限り、個々の暗号化を編集しようとしないでください。詳細情報は、［暗号の編集］をご覧ください。

TLS についての大切な注意： サーバの中には SSL が正しく実装されていないものもあり、TLS をサポートするクライアントソフトウェア （例えばブラウザ） と SSL ハンドシェークを確立できないものがあります。このようなサーバは "TLS intolerant" として知られています。

SSL 設定パネルで "TLS を利用可能にする" オプションを選択した場合、サーバと安全性の高い接続を確立しようとする際に TLS プロトコルを使用しようとします。サーバが TLS intolerant でこの接続に失敗した場合は SSL 3.0 を使用します。

現在、この挙動に関して一つだけ重要な例外があります： プロキシを通じて SSL 接続をしている場合、ブラウザは全く TLS を使用しようとはしません。その代わり SSL 3.0 を使用します。

 

SSL の警告

表示しているウェブサイトが暗号化した接続を利用しているかどうかは、簡単に判ります。接続が暗号化されている場合は、 Navigator ウィンドウの右下隅の南京錠のアイコンが閉じた状態になります。接続が暗号化されていない場合は、南京錠は開いた状態です。

追加の警告が必要な場合は、SSL 設定パネルの警告チェックボタンを選択してください。人によっては、こうした警告を鬱陶しく感じるようです。

これらの Navigator の警告を利用可能にしたい場合は、対応するチェックボックスを選択します：

  • 暗号化をサポートするページに入るとき： 暗号化をサポートするサイトに入るとき常に警告を受け取りたい場合に、この項目を選択します。
  • 低い暗号化の程度を使用しているページを表示するとき： 低い暗号化をサポートするページを表示するとき常に警告を受け取りたい場合に、この項目を選択します。 （低い暗号化とは 40 ビットキーを使用した、最も程度の低い暗号化を使用するものです。）
  • 暗号化をサポートするサイトから離れるとき： 暗号化をサポートするページから離れて暗号化をサポートしないページを表示するときに、 常に警告を受け取りたい場合、この項目を選択します。
  • 暗号化されていないページから暗号化されていないページにデータを送信するとき： 暗号化されていない接続でデータを送るとき、常に警告を受け取りたい場合に、この項目を選択してください。暗号化されていない情報をインターネット上で送る場合、第三者によって簡単に読み取られてしまいます。
  • 暗号化/非暗号化が混在したページを表示するとき： 暗号化されていない情報を含むページを示する時には常にこの警告を表示したい場合に、このオプションを選択してください。

簡単な言葉の定義は、 認証、 暗号化、 または証明書をご覧ください。

暗号の編集

このセクションは、［暗号の編集］ダイアログボックスの使い方を解説します。このダイアログボックスが表示されていない場合は、次のステップに従ってください：

  1. ［編集］ メニューを開き ［設定］ を選択します。
  2. ［プライバシーとセキュリティー］ カテゴリーで、 ［SSL］ をクリックします。（サブカテゴリーが表示されていない場合は、［プライバシーとセキュリティー］をダブルクリックして一覧を展開してください。）
  3. ［暗号の編集］ボタンをクリックします。

［暗号の編集］ダイアログボックスでは、暗号や暗号化アルゴリズムを選択したり、選択を解除したりすることができます。設定を変更する前に、システム管理者に問い合わせてください。

それぞれのチェックボックスは暗号スイート、または暗号アルゴリズムのセットに対応しています。

暗号や暗号化についての詳細情報は、次のオンラインドキュメントをご覧ください：

  • 公開鍵暗号入門
  • SSL 入門
  • NSS 3.4 で利用可能な暗号技術

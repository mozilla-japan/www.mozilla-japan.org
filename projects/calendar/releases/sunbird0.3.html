Mozilla Sunbird™ 0.3 の新機能

Mozilla Sunbird 0.3 は、独立して動作するカレンダーアプリケーションの最新リリースです。すべてのユーザの皆さんにこの最新バージョンへのアップグレードを推奨します。

  • 新しい保存機構：何千何百という多数のイベントが記録されているカレンダーをより機敏に扱えるようになりました.
  • より直感的なインターフェース：必要な事項を見つけやすく、必要のない事項は表示を減らすようになりました。
  • 再設計された設定項目：必要にあわせて簡単に Sunbird のカスタマイズできます。
  • アドオンのサポート：addons.mozilla.org で提供されている拡張機能やテーマ、言語パックなどで Sunbird の機能を拡張できるようになりました。
  • 新しい Windows インストーラ：Windows ユーザでも Sunbird のインストールをこれまで以上に簡単にできます。
  • 印刷機能の向上：カレンダーをさまざまに印刷できます。Mac ユーザも印刷ができるようになりました！
  • 24 時間表示：数時間だけでなく、一日全体を表示できるようになりました。
  • アラームの信頼性： イベントやタスクを確実にお知らせします。
  • ローカライゼーションサポートの改善：Sunbird での言語パックのインストール、有効化、無効化が簡単にできるようになりました。
  • 大幅に改善された信頼性：データを失う可能性が大幅に減少しました。
  • 多数のパフォーマンス面の向上

リリース日：2006 年 10 月 11 日

ダウンロードとインストール

必要システム構成

インストールの前に、お使いのコンピュータが 必要システム構成 を満たしているか確認してください。Sunbird 0.3 より、Mac OS X ユーザの皆さんは Mac OS X 10.2 以降、Windows ユーザの皆さんは Windows 2000 以降をお使いになる必要があります。

Sunbird 0.3 のダウンロード

mozilla.org では Sunbird を Windows、Linux、そして Mac OS X 用に、さまざまな言語で提供しています。Sunbird 0.3 は こちらからダウンロード できます。

提供されていない、他のシステム・言語向けのビルドについては、このページの最後にある「提供されたビルド」の項目をご覧ください。

Sunbird のインストール

Sunbird をダウンロードしたら、次の説明に従ってインストールしてください：

Windows

Sunbird はインストールの前に削除してください。スタート メニューから コントロールパネル を選びます。コントロールパネルが表示されたら、アプリケーションの追加と削除 をダブルクリックします。一覧から 「Mozilla Sunbird」 を選択して、削除 ボタンを押してアンインストールします。

そしてインストーラをダブルクリックして、インストールを開始します。

Mac OS X

Sunbird 0.3.dmg ファイルをダウンロードしたら、Sunbird Disk Image をダブルクリックして展開します。そして Sunbird アプリケーションをハードディスクにドラッグ・アンド・ドロップします。Dock に表示したい場合はアイコンを Dock にドラッグします。

Linux/GTK2

Sunbird をインストールしたい場所で tarball を展開します：

tar -xjvf sunbird-0.3.tar.bz2

これでそのディレクトリ下に sunbird というサブディレクトリが作成されます。

その他のプラットフォーム

圧縮アーカイブを展開して sunbird を実行します。

Sunbird のアンインストール

次の説明に従って Sunbird をアンインストールします：

Windows

スタート メニューから コントロールパネル を選びます。コントロールパネルが表示されたら、アプリケーションの追加と削除 をダブルクリックします。一覧から 「Mozilla Sunbird (0.3)」 を選択して、削除 ボタンを押してアンインストールします。

Mac OS X

Sunbird アプリケーションをゴミ箱にドラッグします。

その他

sunbird フォルダを削除します。

これらのガイドでは、将来再び Sunbird をインストールする場合を考慮して、プロファイルは削除されません。プロファイルフォルダを削除したい場合は、「プロファイルフォルダ」 セクションにあるガイドにあるフォルダを削除してください。ただし、プロファイルフォルダを削除すると、すべてのカレンダーや保存されたパスワード、設定などの情報が削除され、再び Sunbird を使用する際にはまっさらなプロファイルが作成されることに注意してください。

 

その他の情報

プロファイルフォルダ

Sunbird はユーザデータをまとめて次の場所に保存します：

Windows 2000、XP Documents and Settings\<ユーザ名>\Application Data\Mozilla\Sunbird
Mac OS X ~/Library/Application Support/Sunbird
Linux and Unix ~/.mozilla/sunbird

 

既知の問題

Sunbird 0.3 で確認されている問題の一部を掲載します。このリリースに存在するその他の既知の問題は、こちらのページ に掲載されています。新しいバグを報告する前に、必ず両方の一覧を確認してください。

すべてのシステム

  • Sunbird 0.3 では、Sunbird 0.2 または Mozilla Calendar 拡張機能からのカレンダーファイル （.ICS） を自動的には移行しません。代わりに、プロファイルフォルダから手動でインポートし、リモートカレンダーのファイルを再購読する必要があります。こうすることで確実にデータを変換できます。
  • Sunbird 0.3 は起動時にタイムゾーンを設定しようと試みます。ただし、正確に設定できない場合もありますので、データ入力前に設定 / オプションパネルで正しいタイムゾーンを設定してください。
  • Sunbird 0.3 のタイムゾーンは、合衆国議会が 2005 年に採択した日光節約時間 （サマータイム） をまだサポートしていません。
  • コンピュータにある Mozilla 以外のカレンダークライアントからインポートしたイベントやタスクが、間違ったタイムゾーンに表示されるケースが多々あります。リモートで購読しているカレンダーではこうしたエラーは確認されていません。（bug 314339）
  • リモートサーバに保存されているカレンダーを複数の人が編集すると、直前の変更が上書きされてしまう場合があります。（bug 329570）
  • コピー・アンド・ペーストや電子メール、またはインポートで他のカレンダークライアントからイベントを追加した場合、イベントの例外設定が処理されません。（bug 354578）
  • Sunbird 0.3 は Sunbird 0.2 が作成した .ICS ファイルを読み込めません。（bug 352795）

Windows

  • Mozilla のコアコードに構造的な変更が行われたため、Sunbird 0.3 は Windows 95 や Windows 98、Windows ME では動作しません。これはバグではなく、カレンダーチームの担当範囲外の事です。（bug 330276）

Mac OS X

  • Sunbird 0.3 はユニバーサルバイナリでありません。現時点では PowerPC ネイティブなアプリケーションとしてビルドされています。しかし、Apple の Rosetta 環境下で快適に動作します。

Linux および Unix システム

  • Linux では印刷時にクラッシュすることがあります。（bug 351944）
  • X でのドラッグ・アンド・ドロップの非同期性と Mozilla の同期モデルとの違いにより、Linux デスクトップでドラッグ・アンド・ドロップをおこなうと、特にリモートの X セッションをまたぐ際にフリースすることがあります。（bug 348009）

トラブルシューティング

  • 設計が悪い、あるいは互換性のない拡張機能を使うと、クラッシュしたり、カレンダーの読み込みが遅くなるなど、Sunbird に問題を引き起こす可能性があります。Sunbird の一部機能が使えない、Sunbird が起動しない、ウィンドウの表示がおかしい、パフォーマンスが低下した、などの問題に遭遇した場合、拡張機能やテーマが原因となっている可能性があります。そのようなときは、Sunbird をセーフモードで再起動してください。Windows では、スタートメニュー内に作成された "Safe Mode" ショートカットを使うか、sunbird.exe -safe-mode で起動してください。Linux では、./sunbird -safe-mode として起動してくださ。Mac OS X では、ターミナルを開いて次のコマンドを実行してください： /Applications/Sunbird.app/Contents/MacOS/sunbird-bin -safe-mode Sunbird がセーフモードで起動すると、すべてのアドオンが無効になります。問題の原因となったアドオンを無効化してから、通常通り起動してください。

 

よくある質問

  1. どのような協力ができますか？

    計画されている Sunbird の次期リリースは Sunbird 0.5 です。

    このリリースに向けた進捗状況を追跡したいユーザの皆さんは、ぜひナイトリービルドをダウンロードしてください。いつもどおり、テストや役に立つバグ報告を大いに歓迎します。バグの提出方法やカレンダーのテストへの参加方法について知りたい場合は、こちらをご覧ください。

  2. Sunbird のソースコードはどこにありますか？

    Sunbird のソースコードは tarball として ダウンロードできます。最新の開発コードは cvs を通じて入手可能です。Sunbird 固有のソースは "mozilla/calendar"、"mozilla/toolkit"、"mozilla/chrome" にあります。詳しくは ビルド方法 に従ってください。
